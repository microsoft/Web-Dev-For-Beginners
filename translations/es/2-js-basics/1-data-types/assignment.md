# Práctica de Tipos de Datos: Carrito de Compras de Comercio Electrónico

## Instrucciones

Imagina que estás construyendo un sistema moderno de carrito de compras para comercio electrónico. Esta tarea te ayudará a entender cómo diferentes tipos de datos en JavaScript trabajan juntos para crear aplicaciones del mundo real.

### Tu Tarea

Crea un análisis exhaustivo de cómo usarías los tipos de datos de JavaScript en una aplicación de carrito de compras. Para cada uno de los siete tipos de datos primitivos y objetos, necesitas:

1. **Identificar** el tipo de dato y su propósito  
2. **Explicar** por qué este tipo de dato es la mejor opción para características específicas del carrito de compras  
3. **Proporcionar** ejemplos de código realistas que muestren el tipo de dato en acción  
4. **Describir** cómo este tipo de dato interactúa con otras partes del carrito de compras  

### Tipos de Datos Requeridos a Cubrir

**Tipos de Datos Primitivos:**  
- **String**: Nombres de productos, descripciones, información de usuario  
- **Number**: Precios, cantidades, cálculos de impuestos  
- **Boolean**: Disponibilidad de ítems, preferencias de usuario, estado del carrito  
- **Null**: Valores intencionadamente vacíos (como códigos de descuento faltantes)  
- **Undefined**: Valores no inicializados o datos faltantes  
- **Symbol**: Identificadores únicos (uso avanzado)  
- **BigInt**: Cálculos financieros grandes (uso avanzado)  

**Tipos de Referencia:**  
- **Object**: Detalles del producto, perfiles de usuario, contenido del carrito  
- **Array**: Lista de productos, historial de órdenes, categorías

# வரிசைகள் மற்றும் 반복 작업 ஒதுக்கீடு

## அறிவுறுத்தல்கள்

வரிசைகள் மற்றும் 반복 உபயோகப்படுத்துவதற்கான பயிற்சிகளை நிறைவு செய்ய கீழ்க்கண்ட பயிற்சிகளை முடிக்கவும். ஒவ்வொரு பயிற்சியும் பாடத்திலிருந்து கருத்துக்களை அடிப்படையாக கொண்டு பல வகையான 반복 வகைகளையும் வரிசை முறைகளையும் பயன்படுத்த ஊக்குவிக்கிறது.

### பயிற்சி 1: எண் முறை உருவாக்கி
1-20 இடையே உள்ள ஒவ்வொரு மூன்றாவது எண்ணையும் பட்டியலிட்டு அதை கான்சோலில் அச்சிடும் ஒரு நிரலை உருவாக்கவும்.

**தேவைகள்:**
- தேர்ந்தெடுத்த உயர்வு கொண்ட `for` 반복 ஒன்றைப் பயன்படுத்தவும்
- எண்களை பயனருக்கு உணர்வுப்பூர்வமாகக் காட்டவும்
- உங்கள் மூலக்கூற்றை விளக்கும் விளக்கக் குறிப்புகள் சேர்க்கவும்

**எதிர்பார்க்கப்பட்ட வெளி:**
```
3, 6, 9, 12, 15, 18
```

> **குறிப்பு:** உங்கள் for 반복ில் iteration-expression-ஐ மாற்றி எண்களை தவிர்க்கவும்.

### பயிற்சி 2: வரிசை பகுப்பாய்வு
குறைந்தபட்சம் 8 வேறுபட்ட எண்கள் கொண்ட வரிசையை உருவாக்கி அந்த தரவை பகுப்பாய்வு செய்யும் செயல்பாடுகளை எழுதவும்.

**தேவைகள்:**
- `numbers` எனும் 8 மதிப்புகள் கொண்ட வரிசையை உருவாக்கவும்
- மிக பெரிய எண்ணை திருப்பும் `findMaximum()` என்ற செயல்பாட்டை எழுதவும்
- மிக சிறிய எண்ணை திருப்பும் `findMinimum()` என்ற செயல்பாட்டை எழுதவும்  
- அனைத்து எண்களின் கூட்டையே திருப்பும் `calculateSum()` என்ற செயல்பாட்டை எழுதவும்
- ஒவ்வொரு செயல்பாட்டையும் சோதனை செய்து முடிவுகளை காட்சி செய்யவும்

**போனஸ் சவால்:** வரிசையில் இரண்டாவது மிக பெரிய எண்ணை கண்டுபிடிக்கும் செயல்பாட்டை உருவாக்கவும்.

### பயிற்சி 3: ஸ்டிரிங் வரிசை செயலாக்கம்
உங்கள் பிடித்த திரைப்படங்கள்/புத்தகங்கள்/பாடல்கள் கொண்ட ஒரு வரிசையை உருவாக்கி பல 반복 வகைகளை பயிற்சி செய்யவும்.

**தேவைகள்:**
- குறைந்தது 5 ஸ்டிரிங் மதிப்புகள் கொண்ட வரிசையை உருவாக்கவும்
- ஆவண எண்களுடன் உருப்படிகளை காட்டு பழமையான `for` 반복ைப் பயன்படுத்தவும் (1. உருப்படி பெயர்)
- பெரிய எழுத்துக்களால் உருப்படிகளை காட்டு `for...of` 반복ைப் பயன்படுத்தவும்
- மொத்த எழுத்துக்களை எண்ணவும் மற்றும் காட்டு `forEach()` முறையை பயன்படுத்தவும்

**உதாரண வெளி:**
```
Traditional for loop:
1. The Matrix
2. Inception
3. Interstellar

For...of loop (uppercase):
THE MATRIX
INCEPTION
INTERSTELLAR

Character count:
Total characters across all titles: 42
```

### பயிற்சி 4: தரவு வடிகட்டி (மேம்பட்டது)
மாணவர்கள் என்ற பொருள்களை உள்ளடக்கிய ஒரு வரிசையை செயலாக்கும் நிரலை உருவாக்கவும்.

**தேவைகள்:**
- அட்டவணையில் குறைந்தபட்சம் 5 மாணவர் பொருள்கள்; `name`, `age`, `grade` என்ற பண்புகளை கொண்டவை
- 18 அல்லது அதற்கும் மேற்பட்ட மாணவர்களை கண்டறிய 반복ங்களைக் கையாளவும்
- அனைத்து மாணவர்களின் சராசரி மதிப்பெண்களை கணக்கிடவும்
- 85 மேற்பட்ட மதிப்பெண்கள் கொண்ட மாணவர்கள் மட்டுமே உள்ள புதிய வரிசையை உருவாக்கவும்

**உதாரண அமைப்பு:**
```javascript
const students = [
  { name: "Alice", age: 17, grade: 92 },
  { name: "Bob", age: 18, grade: 84 },
  // மேலும் மாணவர்களைச் சேர்க்கவும்...
];
```

## உங்கள் கோ드를 சோதனை செய்யவும்

உங்கள் நிரல்களை சோதிக்க:
1. ஒவ்வொரு பயிற்சியையும் உலாவியின் கான்சோலில் இயக்கவும்
2. வெளிகள் எதிர்பார்த்த முடிவுகளுக்கு ஒப்பிடுக
3. வேறுபட்ட தரவு தொகுப்புகளுடன் சோதனை செய்யவும்
4. உங்கள் கோடு எடியால் உள்ள சார்ந்த நிலைகளை (பூஜ்ஜிய வரிசைகள், ஒரே உறுப்பினர்) கையாள்வதைச் சரிபார்க்கவும்

## சமர்ப்பிப்பு வழிகாட்டிகள்

சமர்ப்பிப்பில் கீழ்க்கண்டவை சேர்க்கவும்:
- ஒவ்வொரு பயிற்சிக்கும் நன்கு விளக்கமான JavaScript கோடு
- உங்கள் நிரல்கள் இயங்கும் வலைப்பிரதி அல்லது உரை வெளி
- ஒவ்வொரு பணிக்குமான நீங்கள் தேர்ந்தெடுத்த 반복 வகை மற்றும் காரணம் குறித்த சுருக்க விளக்கம்

| -------- | -------------------- | ------------------- | --------------------------- |
| **செயல்பாடு** | அனைத்து பயிற்சிகளும் சரியாக முடிக்கப்பட்டது, போனஸ் சவால்களுடன் | தேவையான அனைத்து பயிற்சிகளும் சரியாக இயங்கும் | சில பயிற்சிகள் அநெறியவோ அல்லது பிழையோ கொண்டவை |
| **கோடு தரம்** | சுறுசுறுப்பும் நல்ல கட்டமைப்பும் கொண்ட, விளக்கமான மாறிலிகள் | கோடு வேலை செய்கிறது, ஆனால் சுத்தமாக்கம் தேவை | கோடு குழப்பமானதும் புரிதல் கஷ்டமானதும் |
| **குறிப்புகள்** | நுணுக்கமான விளக்கக் குறிப்புகள் | அடிப்படைக் குறிப்புகள் உள்ளன | குறிப்பு அரிதாக அல்லது இல்லை |
| **반복 பயன்பாடு** | வெவ்வேறு 반복 வகைகளின் புரிதலைத் தெளிவுப்படுத்துகிறது | 반복 சரியாக பயன்படுகிறது, ஆனால் வகை மட்டுமே | தவறான அல்லது செயல்திறன் குறைந்த 반복 பயன்பாடு |
| **சோதனை** | பல சூழல்களுடன் முழுமையான சோதனைக் குறிப்புகள் | அடிப்படை சோதனை | குறைந்த சோதனைப் பதிவு |

## பிரதிபலிப்பு கேள்விகள்

பயிற்சிகளை முடித்து பிறகு:
1. எது அதிகமாக இயல்பாக தோன்றியது மற்றும் ஏன்?
2. வரிசைகளுடன் பணியாற்றும்போது நீங்கள் எதிர்கொண்ட சவால்கள் என்ன?
3. இந்த திறன்கள் எப்படி நிஜ உலக வலை மேம்பாட்டுத் திட்டங்களில் பயன்படும்?
4. செயல்திறனை மேம்படுத்த உங்கள் கோட்டை எப்படி வேறுபடுத்தப்போகிறீர்கள்?

# 충돌 탐색

## 지침

충돌 감지 지식을 적용하여 다양한 유형의 객체 상호작용을 보여주는 맞춤형 미니게임을 만드세요. 이 과제는 창의적인 구현과 실험을 통해 충돌 메커니즘을 이해하는 데 도움을 줄 것입니다.

### 프로젝트 요구 사항

**다음 기능을 가진 작은 인터랙티브 게임을 만드세요:**
- 키보드 또는 마우스 입력으로 제어할 수 있는 **여러 개의 움직이는 객체**
- 수업에서 배운 직사각형 교차 원리를 사용하는 **충돌 감지 시스템**
- 충돌 시 발생하는 **시각적 피드백**(객체 파괴, 색상 변경, 효과 등)
- 충돌이 의미 있고 흥미로워지도록 하는 **게임 규칙**

### 창의적 제안

**다음 시나리오 중 하나를 구현해 보세요:**
- **소행성 필드**: 위험한 우주 부유물을 피해 우주선을 조종하기
- **범퍼카**: 물리 기반 충돌 경기장 만들기
- **운석 방어**: 지구를 다가오는 우주 암석으로부터 보호하기
- **수집 게임**: 장애물을 피하며 아이템 모으기
- **영역 점령**: 공간을 차지하려 경쟁하는 객체들

### 기술적 구현

**솔루션은 다음을 보여줘야 합니다:**
- 직사각형 기반 충돌 감지의 적절한 사용
- 사용자 입력을 위한 이벤트 기반 프로그래밍
- 객체 생명 주기 관리(생성 및 파괴)
- 적절한 클래스 구조를 갖춘 깔끔한 코드 조직

### 추가 도전 과제

**게임에 다음 기능을 추가해 보세요:**
- 충돌 시 발생하는 **입자 효과**
- 다양한 충돌 유형에 맞는 **사운드 효과**
- 충돌 결과를 바탕으로 한 **점수 시스템**
- 서로 다른 동작을 가진 **복수 충돌 유형**
- 시간이 지날수록 증가하는 **점진적 난이도**

## 평가 기준

| 기준 | 우수 | 적절 | 개선 필요 |
|----------|-----------|----------|-------------------|
| **충돌 감지** | 여러 객체 유형과 정교한 상호작용 규칙을 갖춘 정확한 직사각형 기반 충돌 감지 구현 | 간단한 객체 상호작용과 함께 기본적인 충돌 감지가 올바르게 작동 | 충돌 감지에 문제가 있거나 일관되게 작동하지 않음 |
| **코드 품질** | 적절한 클래스 구조, 의미 있는 변수 이름, 적절한 주석을 갖춘 깔끔하고 잘 조직된 코드 | 코드는 동작하지만 더 잘 구성하거나 문서화할 수 있음 | 코드가 이해하기 어렵거나 구조가 좋지 않음 |
| **사용자 상호작용** | 부드러운 게임 플레이, 명확한 시각 피드백, 몰입감 있는 메커니즘을 갖춘 반응형 컨트롤 | 기본 컨트롤이 작동하며 적절한 피드백 제공 | 컨트롤이 반응하지 않거나 혼란스러움 |
| **창의성** | 독창적인 컨셉, 고유한 기능, 시각적 완성도, 혁신적인 충돌 동작 | 일부 창의적인 요소를 포함한 표준 구현 | 기본 기능만 있으며 창의성 부족 |

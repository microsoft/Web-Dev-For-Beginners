# 배열과 반복문 과제

## 지침

배열과 반복문 작업을 연습하기 위해 다음 연습문제를 완료하세요. 각 연습문제는 수업의 개념을 기반으로 하며 다양한 반복문 유형과 배열 메서드를 적용하도록 권장합니다.

### 연습문제 1: 숫자 패턴 생성기
1부터 20까지의 숫자 중에서 3번째 숫자마다 나열하여 콘솔에 출력하는 프로그램을 작성하세요.

**요구사항:**
- 사용자 정의 증감이 있는 `for` 반복문 사용
- 사용자 친화적인 형식으로 숫자 표시
- 논리를 설명하는 상세한 주석 추가

**기대 출력:**
```
3, 6, 9, 12, 15, 18
```

> **팁:** for 반복문에서 반복 식을 수정하여 숫자를 건너뛰세요.

### 연습문제 2: 배열 분석
최소 8개의 서로 다른 숫자로 이루어진 배열을 만들고 데이터를 분석하는 함수를 작성하세요.

**요구사항:**
- 적어도 8개의 값을 가진 `numbers` 배열 생성
- 가장 큰 숫자를 반환하는 `findMaximum()` 함수 작성
- 가장 작은 숫자를 반환하는 `findMinimum()` 함수 작성  
- 모든 숫자의 합계를 반환하는 `calculateSum()` 함수 작성
- 각 함수를 테스트하고 결과를 출력

**추가 도전:** 배열에서 두 번째로 큰 숫자를 찾는 함수 작성.

### 연습문제 3: 문자열 배열 처리
좋아하는 영화/책/노래 배열을 만들고 다양한 반복문 유형을 연습하세요.

**요구사항:**
- 최소 5개의 문자열 값을 가진 배열 생성
- 전통적 `for` 반복문을 사용해 번호와 함께 아이템 표시 (1. 아이템 이름)
- `for...of` 반복문을 사용해 대문자로 아이템 표시
- `forEach()` 메서드를 사용해 총 문자의 개수 세고 출력

**예시 출력:**
```
Traditional for loop:
1. The Matrix
2. Inception
3. Interstellar

For...of loop (uppercase):
THE MATRIX
INCEPTION
INTERSTELLAR

Character count:
Total characters across all titles: 42
```

### 연습문제 4: 데이터 필터링 (고급)
학생을 나타내는 객체 배열을 처리하는 프로그램을 만드세요.

**요구사항:**
- `name`, `age`, `grade` 속성을 가진 최소 5명의 학생 객체 배열 생성
- 18세 이상 학생을 찾기 위해 반복문 사용
- 모든 학생의 평균 성적 계산
- 성적이 85점 이상인 학생만 포함하는 새 배열 생성

**예시 구조:**
```javascript
const students = [
  { name: "Alice", age: 17, grade: 92 },
  { name: "Bob", age: 18, grade: 84 },
  // 학생을 더 추가하세요...
];
```

# 새로운 키보드 게임 만들기

## 지침

타이핑 게임을 통해 이벤트 기반 프로그래밍의 기본을 마스터했으니, 이제 창의력을 발휘할 시간입니다! 이벤트 처리, DOM 조작, 사용자 상호작용 패턴에 대한 이해를 보여주는 키보드 기반 게임을 설계하고 구축하세요.

키보드 이벤트를 사용하여 특정 작업을 수행하는 작은 게임을 만드세요. 이 게임은 다른 유형의 타자 게임, 키 입력 시 화면에 픽셀을 그리는 아트 애플리케이션, 화살표 키로 조작하는 간단한 아케이드 스타일 게임, 또는 당신이 상상할 수 있는 그 어떤 창의적인 개념도 좋습니다. 창의력을 발휘하여 서로 다른 키가 어떻게 다양한 동작을 유발하는지 생각해보세요!

**당신의 게임은 다음을 포함해야 합니다:**

| 요구 사항 | 설명 | 목적 |
|-------------|-------------|---------|
| **이벤트 리스너** | 최소 3가지 다른 키보드 이벤트에 반응 | 이벤트 처리 이해도 증명 |
| **시각적 피드백** | 사용자 입력에 즉각적인 시각적 반응 제공 | DOM 조작 숙련도 보여주기 |
| **게임 로직** | 점수, 레벨 또는 진행 메커니즘 포함 | 애플리케이션 상태 구현 연습 |
| **사용자 인터페이스** | 명확한 지시사항과 직관적인 조작법 | 사용자 경험 디자인 능력 개발 |

**고려할 창의적 프로젝트 아이디어:**
- **리듬 게임**: 음악이나 시각적 신호에 맞춰 키를 누름
- **픽셀 아트 크리에이터**: 다른 키가 다른 색상이나 패턴 그리기
- **단어 만들기**: 글자들을 특정 순서로 입력하여 단어 생성
- **스네이크 게임**: 화살표 키로 뱀을 조작해 아이템을 수집
- **뮤직 신디사이저**: 다른 키가 다른 음표나 소리 연주
- **스피드 타이핑 변형**: 특정 카테고리별 타이핑 (프로그래밍 용어, 외국어 등)
- **키보드 드러머**: 키를 드럼 소리와 매핑해 비트 생성

**구현 지침:**
- **간단한 개념**으로 시작해 점차 복잡성 증가
- **부드럽고 반응 빠른 컨트롤**에 집중하여 자연스러운 조작감 구현
- **게임 상태와 플레이어 진행 상황에 대한 명확한 시각적 표시 포함**
- **다양한 사용자로 게임을 테스트하여 직관적인 플레이 보장**
- **이벤트 처리 전략을 설명하는 주석으로 코드 문서화**

## 평가 기준

| 기준 | 뛰어남 | 적절함 | 개선 필요 |
| -------- | --------- | -------- | ----------------- |
| **기능성** | 여러 기능이 완비되고 부드러운 게임플레이를 갖춘 완성도 높은 게임 | 키보드 이벤트 처리를 보여주는 기본 기능의 작동하는 게임 | 제한된 기능이나 심각한 버그가 있는 최소 구현 |
| **코드 품질** | 최적화된 이벤트 처리와 함께 잘 정리되고 주석 처리된 코드, 모범 사례 준수 | 적절한 이벤트 리스너와 DOM 조작 사용, 깔끔하고 읽기 쉬운 코드 | 기초적인 코드 구조, 조직 문제나 비효율적 구현 포함 |
| **사용자 경험** | 직관적인 조작, 명확한 피드백, 전문적인 느낌의 흥미로운 게임플레이 | 적절한 사용자 안내와 반응형 컨트롤의 기능적 인터페이스 | 불명확한 지시사항이나 반응성 부족의 기본적인 인터페이스 |
| **창의성** | 키보드 이벤트의 독창적 사용과 창의적 문제 해결이 돋보이는 독특한 개념 | 일반적인 게임 패턴의 흥미로운 변형과 우수한 이벤트 처리 활용 | 최소한의 창의적 요소를 가진 기본 개념의 단순 구현 |

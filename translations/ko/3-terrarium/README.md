## Terrarium 배포하기

**Azure Static Web Apps**를 사용하여 Terrarium을 웹에 배포하거나 공개할 수 있습니다.

1. 이 저장소를 포크하세요

2. 이 버튼을 누르세요 👇  

[![Deploy to Azure](https://aka.ms/deploytoazurebutton)](https://portal.azure.com/#create/Microsoft.StaticApp)

3. 설정 마법사를 따라 앱을 만드세요.  
   - **App root**를 `/solution` 또는 코드베이스의 루트로 설정하세요.  
   - 이 앱에는 API가 없으므로 API 구성은 건너뛰어도 됩니다.  
   - Azure Static Web Apps가 앱을 빌드하고 배포할 수 있도록 `.github` 폴더가 자동으로 생성됩니다.

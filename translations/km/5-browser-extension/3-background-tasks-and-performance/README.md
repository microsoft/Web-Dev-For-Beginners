бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ•бһ»бһҹбһҳбҹүбһ»бһҒбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ - бһҠбһјбһ…бһҮбһ¶бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ–бһҺбҹҢбһ‘бҹ…бһҸбһ¶бһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбһ¶бһ”бһјбһ“бҹ” бһ“бҹҒбҹҮбһ”бһ„бҹ’бһңбһ·бһӣбһҒбҹ’бһүбһ»бҹҶбһІбҹ’бһҷбһӮбһ·бһҸбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣ NASA бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӣбҹҶбһўбһ„бһӮбҹ’бһҡбһ”бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ“бҹ…бһӣбһҫбһҷбһ¶бһ“бһўбһ¶бһҖбһ¶бһҹ Apolloбҹ” бһ–бһҪбһҖбһӮбҹҒбһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһҹбһҳбҹ’бһ—бһ¶бһҡбҹҲбһҠбһјбһ…бһ”бҹүбҹҮбһҹбҹ’бһҳбһҫбһ¬бһҳбҹүбҹ„бһҳбҹүбһ¶бһҡбһ”бһҹбҹӢбһҶбҹ’бһ“бһ¶бҹҶбһ„бһ”бһ¶бһ“бһ‘бҹҒ бһ–бҹ’бһҡбҹ„бҹҮбһҮбһёбһңбһ·бһҸбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ” бһҒбһҺбҹҲбһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҳбһ·бһ“бһҠбҹӮбһӣбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ‘бҹҒ бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҠбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸ - бһҖбһјбһҠбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһӣбҹ’бһўбһҮбһ¶бһ„бҹ”

## бһҸбһҫбһҸбҹ’бһҡбһјбһңбһҳбһҫбһӣбһўбҹ’бһңбһёбһ–бҹҒбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ profiler

бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ profiler бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ - бһҮбҹҶбһ“бһ¶бһүбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһӮбһәбһҖбһ¶бһҡбһҷбһӣбҹӢбһҗбһ¶бһҸбһ¶бһҡбһ¶бһ„бһ–бһҺбҹҢбһ…бһҳбҹ’бһҡбһ»бҹҮбһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһ“бһ·бһҷбһ¶бһҷбһўбҹ’бһңбһёбҹ” бһҖбһ»бҹҶбһ”бһ¶бһҡбһҳбҹ’бһ— бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ…бҹҒбҹҮбһ”бһҖбһўбһ¶бһ“бһңбһ¶бҹ” бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһ”бһ¶бһ“бһҡбҹҖбһ“бһӮбҹҒбһ…бһ…бҹҒбһүбһ–бһёбһҹбһүбҹ’бһүбһ¶бһ–бҹ’бһҡбһҳбһ¶бһ“бһҳбһ»бһ“бһ–бҹҒбһӣбһңбһ¶бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһ”бһүбҹ’бһ бһ¶бһ’бҹҶбҹ”

бһҷбһҫбһ„бһ“бһ№бһ„бһ“бһ·бһҷбһ¶бһҷбһ–бһёбһҳбҹҒбһҠбҹғбһ”бһүбҹ’бһ бһ¶бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ“бҹ…бһӣбһҫбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ бһҠбҹӮбһӣбһҮбһ¶бһ‘бһҳбҹ’бһӣбһ¶бһ”бҹӢбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһңбҹҒбһ”бҹ” бһҠбһјбһ…бһҮбһ¶ Marie Curie бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҖбһ¶бҹҶбһҡбһҹбҹ’бһҳбһёбһҷбҹүбһ¶бһ„бһ”бҹ’бһҡбһ»бһ„бһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“бһҖбҹ’бһ“бһ»бһ„бһ”бһ“бҹ’бһ‘бһ”бҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢбһ“бһ¶бһ„ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҸбһ¶бһҳбһҠбһ¶бһ“бһ”бҹҶбһ–бһ¶бһ“бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶бһ”бһүбҹ’бһ бһ¶бһҖбҹҶбһ–бһ»бһ„бһҖбһҫбһҸбҹ” бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ”бһүбҹ’бһ бһ¶бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҳбһ»бһ“бһӮбһәбһҮбһ¶бһҮбҹҶбһ“бһҪбһҷбһ’бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

**бһ‘бҹҶбһ бҹҶбһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·**бҹ– бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ”бһ¶бһ“бһ’бҹ’бһӣбһ¶бһ”бҹӢбһҖбҹ’бһӣбһ¶бһҷбһ„бһ¶бһ”бҹӢбһ’бҹ’бһ„бһ“бҹӢбһҮбһ¶бһ„бһҳбһ»бһ“бһҡбһҷбҹҲбһ–бҹҒбһӣбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҶбҹ’бһ“бһ¶бҹҶ бһ бһҫбһҷбһ‘бҹҶбһ„бһ“бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһ…бҹ’бһҡбһҫбһ“бһӮбһәбһҳбһҖбһ–бһёбһҡбһјбһ”бһ—бһ¶бһ–бҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбҹҖбһ”бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһңбһҸбҹ’бһҗбһ»бһ”бһ“бҹ’бһҗбҹӮбһҳбһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһ”бһјбһ”бһҢбһёбһҮбһёбһҗбһӣбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ”

- **бһ”бһ„бҹ’бһҡбһҪбһҳбһ”бһ„бҹ’бһ бһ¶бһ”бҹӢ** бһҡбһјбһ”бһ—бһ¶бһ–бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ! бһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ‘бҹҶбһ“бһҫбһ”бһҠбһјбһ…бһҮбһ¶ WebP бһўбһ¶бһ…бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бҹҶбһҜбһҖбһҹбһ¶бһҡбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„

**бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ…бһҳбҹ’бһ”бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбҹҶбһҡбһ»бһү DOMбҹ–**

- **бһҷбһҖбһ…бҹҒбһү** бһ”бһ‘бһ”бһүбҹ’бһҮбһ¶ CSS бһҳбһ·бһ“бһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹ’бһ‘бһ¶бһҷбһҮбһ¶бһҹбһ¶бҹҶбһ„бҹ—бһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
- **бһҡбҹҖбһ”бһ…бҹҶ** CSS бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһҸбҹӮбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бҹҒбһӣбһ‘бҹҶбһ–бҹҗбһҡбһ“бһёбһҳбһҪбһҷбҹ—
- **бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’** HTML бһҮбһ¶бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡ

**JavaScript**бҹ– бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһӮбһҪбһҡбһҸбҹӮбһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“ scripts бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҡбһ¶бҹҶбһ„бһҒбҹ’бһ‘бһ”бҹӢ (В« render-blocking В») бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ•бҹ’бһ‘бһ»бһҖбһҳбһ»бһ“бһ–бҹҒбһӣ DOM бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ“бһ·бһ„бһӮбһјбһҡбһўбҹ„бһҷбһҳбһ¶бһ“бһ“бҹ…бһӣбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбҹ” бһҹбһјбһҳбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһ”бҹ’бһҡбһҫ `defer` бһҮбһ¶бһҳбһҪбһҷбһҸбһҪбһўбһҖбҹ’бһҹбһҡ inline scripts (бһҠбһјбһ…бһҮбһ¶ module Terrarium)бҹ”

**вҖӢбһ“бһ·бһҡбһ“бҹ’бһҸбһҡбһ—бһ¶бһ–бһҖбһјбһҠ JavaScript бһ”бҹӮбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбһҳбҹҗбһҷбһҗбҹ’бһҳбһёбҹ–**
- **бһ”бҹ’бһҡбһҫ** бһўбһҸбҹ’бһҗбһ”бһ‘ `defer` бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖ scripts бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё DOM бһ…бҹ’бһҡбһ¶бһҹбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ
- **бһўбһ“бһ»бһңбһҸбҹ’бһҸ** бһҖбһ¶бһҡбһ…бҹӮбһҖбһҖбһјбһҠ (code splitting) бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖ JavaScript бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ
- **бһўбһ“бһ»бһңбһҸбҹ’бһҸ** бһӣбҹ’бһ”бһҝбһ“бһҷбһәбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһ·бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢ
- **бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷ** бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ бһ“бһ·бһ„бһҹбҹҠбһ»бһҳбһҒбҹ’бһӣбһёбһҖбҹ’бһҡбҹӮбһ“бһ’бҹ’бһңбһҫбһ”бһҺбҹ’бһҸбһ¶бһүбһҠбһӣбҹӢбһҸбһ·бһ…бһ”бҹҶбһ•бһ»бһҸ

вң… бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҒбҹ’бһӣбҹҮбҹ—бһҮбһ¶бһҳбһҪбһҷ [бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ”бҹ’бһҡбһЎбһ„бһӣбҹ’бһ”бһҝбһ“бһҸбҹҶбһҺ](https://www.webpagetest.org/) бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ“бһ–бһёбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ‘бһјбһ‘бҹ…бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбҹ”

### рҹ”„ **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһҡбһјбһ”бһ„бҹ’бһҡбҹҖбһ“**
**бһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–**бҹ– бһҳбһ»бһ“бһ–бҹҒбһӣбһҹбһ„бҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳ бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһўбһ¶бһ…бҹ–
- вң… бһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһ•бҹ’бһӣбһјбһңбһҖбҹҶбһҺбһҸбҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһё HTML бһ‘бҹ… pixels
- вң… бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ”бһүбҹ’бһ бһ¶бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”
- вң… бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ‘бҹҶбһ–бҹҗбһҡ
- вң… бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ‘бҹҶбһ бҹҶбһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ· бһ“бһ·бһ„бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү DOM бһ“бһ¶бҹҶбһІбҹ’бһҷбһӣбҹ’бһ”бһҝбһ“бһҡбҹҶбһӣбһҹбҹӢ

**бһҸбҹҒбһҹбҹ’бһҠбһҳбһ»бһҒбһҜбһ„бһҶбһ¶бһ”бҹӢбҹ—**бҹ– бһҸбһҫбһҳбһ¶бһ“бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„бһ–бҹҒбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“ JavaScript бһҠбҹӮбһӣбһҡбһ¶бҹҶбһ„бһҒбҹ’бһ‘бһ”бҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү?
*бһ…бһҳбҹ’бһӣбһҫбһҷбҹ– бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ‘бһ¶бһүбһҷбһҖ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһҪбһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҳбһ»бһ“бһңбһ¶бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ…бҹ’бһҡбһ¶бһҹ HTML бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹҶбһ–бҹҗбһҡ*

**бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ–**
- **бһ–бһ“бҹ’бһ’бһ“бһ¶бһ‘бһё бҹЎбҹ бҹ  бһҳбһёбһӣбһ·бһңбһ·бһ“бһ¶бһ‘бһё**бҹ– бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһӮбҹ’бһҳбһ¶бһ“бһ…бһ·бһҸбҹ’бһҸбһӣбҹҶбһ бһјбһҡ
- **бһ–бһ“бҹ’бһ’бһ“бһ¶бһ‘бһё бҹЎ бһңбһ·бһ“бһ¶бһ‘бһё**бҹ– бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҒбҹ’бһңбҹҮбһҖбһҳбҹ’бһ–бһҹбҹӢ
- **бҹЈ бһңбһ·бһ“бһ¶бһ‘бһёбһЎбһҫбһ„бһӣбһҫ**бҹ– бҹӨбҹ % бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ…бһ¶бһҖбһ…бҹҒбһүбһ–бһёбһ‘бҹҶбһ–бҹҗбһҡ
- **бһ”бһҺбҹ’бһҠбһ¶бһүбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘**бҹ– бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҮбһ¶бһ„бһҳбһ»бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ·бһҸбһ·бһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘

бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһӮбҹҶбһ“бһ·бһҸбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ•бҹ’бһүбһҫбһҡбһІбҹ’бһҷбһңбһ¶ бһҳбһҖбһҳбһҫбһӣбһҡбһҝбһ„бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ–бһёбһҡбһҷбҹҲбһ–бҹҒбһӣбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

```javascript
	const co2Scale = [0, 150, 600, 750, 800];
	// Corresponding colors from green (clean) to dark brown (high carbon)
	const colors = ['#2AA364', '#F5EB4D', '#9E4229', '#381D02', '#381D02'];

	// Find the closest scale value to our input
	const closestNum = co2Scale.sort((a, b) => {
		return Math.abs(a - value) - Math.abs(b - value);
	})[0];
	
	console.log(`${value} is closest to ${closestNum}`);
	
	// Find the index for color mapping
	const num = (element) => element > closestNum;
	const scaleIndex = co2Scale.findIndex(num);

	const closestColor = colors[scaleIndex];
	console.log(scaleIndex, closestColor);

	// Send color update message to background script
	chrome.runtime.sendMessage({ action: 'updateIcon', value: { color: closestColor } });
}
```

**бһ…бҹӮбһҖбһңбһ¶бһІбҹ’бһҷбһҷбһӣбҹӢбһӣбҹ’бһўбһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡбһӣбҹ’бһўбһ“бҹҒбҹҮбҹ–**
- **бһҖбҹҶбһҺбһҸбҹӢ** бһўбһ¶бһҡбҹҒбһ–бһёбһҡбһҠбһ»бҹҶ - бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҡбһ·бһҸ CO2 бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһҺбҹҢ (бһ”бҹғбһҸбһ„ = бһҹбҹ’бһўбһ¶бһҸ, бһҹбҹ’бһңбһ¶бһҷ = бһ”бҹ’бһҡбҹҒбһ„!)
- **бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ** бһҖбһ¶бһҡбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһҹбҹҖбһҡбһёбһҠбһјбһ…бһҮбһ¶бһҸбһҳбҹ’бһӣбҹғ CO2 бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбҹҖбһ”бһўбһ¶бһҡбҹҒбһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„
- **бһ‘бһ‘бһҪбһӣбһҷбһҖ** бһ–бһҺбҹҢбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ findIndex()
- **бһ•бҹ’бһүбһҫ** бһҹбһ¶бһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢ script бһ•бҹ’бһ‘бҹғбһҖбҹ’бһҡбҹ„бһҷбһҡбһ”бһҹбҹӢ Chrome бһҮбһ¶бһҳбһҪбһҷбһ–бһҺбҹҢбһҠбҹӮбһӣбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ
- **бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ** template literals (бһҹбһүбҹ’бһүбһ¶бһҖбҹ’бһ“бһ»бһ„бһҸбһҪбһҹбһүбҹ’бһүбһ¶бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ–бһёбһӣбһҫ) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбҹҖбһ”бһ…бҹҶбһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡбһ”бһ¶бһ“бһҹбҹ’бһўбһ¶бһҸ
- **бһҡбһҖбҹ’бһҹбһ¶** бһҡбҹҖбһ”бһ…бҹҶбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹ„бһҷ const

`chrome.runtime` [API](https://developer.chrome.com/extensions/runtime) бһӮбһәбһҠбһјбһ…бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҹбһ¶бһҹбһ“бҹҚбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ - бһңбһ¶бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһ“бҹ…бһ–бһёбһҖбҹ’бһҡбҹ„бһҷ бһ“бһ·бһ„бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ–

> "бһ”бҹ’бһҡбһҫ chrome.runtime API бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ¶бһүбһҷбһҖбһ‘бҹҶбһ–бҹҗбһҡбһ•бҹ’бһ‘бҹғбһҖбҹ’бһҡбҹ„бһҷ бһҷбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ–бһё manifest бһ бһҫбһҷбһҹбҹ’бһҸбһ¶бһ”бҹӢбһ“бһ·бһ„бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбһӣбҹӢбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҖбҹ’бһ“бһ»бһ„бһҮбһёбһңбһ…бһӣбһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ¬бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ API бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҳбҹ’бһӣбҹӮбһ„бһ•бҹ’бһӣбһјбһңбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶ URLs бһ‘бҹ…бһҮбһ¶ URLs бһ–бҹҒбһүбһӣбҹҒбһүбһ”бһ¶бһ“бһ•бһ„бҹ”"

**бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶ Chrome Runtime API бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ:**
- **бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷ** бһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бһ·бһҷбһ¶бһҷбһӮбҹ’бһ“бһ¶
- **бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„** бһҖбһ¶бһҡбһ„бһ¶бһҡбһ•бҹ’бһ‘бҹғбһҖбҹ’бһҡбҹ„бһҷбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’ UI
- **бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„** бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҮбһёбһңбһ…бһӣбһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳ
- **бһ’бҹ’бһңбһҫбһІбҹ’бһҷ** бһҖбһ¶бһҡбһ•бҹ’бһүбһҫбһҹбһ¶бһҡбһҡбһңбһ¶бһ„ script аёҮбһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҺбһ¶бһҹбҹӢ

вң… бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ»бһҖбһҡбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Edge бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһўбҹ„бһҷбһўбҹ’бһ“бһҖбһ—бҹ’бһүбһ¶бһҖбҹӢбһ•бҹ’бһўбһҫбһӣбһҗбһ¶бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫ chrome APIбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ Edge бһҮбҹҶбһ“бһ¶бһ“бҹӢбһҗбҹ’бһҳбһёбһҡбһҸбҹӢбһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ Chromium бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ”бһ¶бһ“бҹ”

```mermaid

    group browser(logos:chrome)[бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ]
    
    service popup(logos:html5)[бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бһ„бҹ’бһ бһ¶бһү Popup UI] in browser
    service content(logos:javascript)[бһҖбһјбһҠбһҒбҹ’бһҳбҹӮбһҡ Content Script] in browser
    service background(database)[бһҖбһјбһҠбһ•бҹ’бһ‘бҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ Background Script] in browser
```

> **бһҮбҹҶбһҡбһҫбһҹбһ–бһ·бһҹбҹҒбһҹ**бҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹ’бһҡбһҫ profiler бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ»бһҖбһҡбһҖ бһҹбһјбһҳбһ”бһҫбһҖ dev tools бһ–бһёбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ‘бҹ…бһҳбһҖ бһҳбһ»бһ“бһ–бҹҒбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһҠбһјбһ…бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбҹ” бһңбһ¶бһҮбһҪбһҷбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”

### бһҖбҹҶбһҺбһҸбҹӢбһ–бһҺбҹҢбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ

бһҳбһ»бһ“бһ–бҹҒбһӣбһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһўбҹ’бһ“бһҖбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢбһҳбһ·бһ“бһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҳбһҫбһӣбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһ‘бһ‘бҹҒ бһ¬бһҳбһ¶бһ“бһҳбһ»бһҒбһҳбһ¶бһҸбҹӢбһҒбһјбһ…бһҒбһ¶бһҸбһЎбһҫбһҷбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷбһ–бһҺбҹҢбһ”бҹғбһҸбһ„ бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҠбһ№бһ„бһҗбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ–бһёбһ–бҹҒбһӣбһҒбһ¶бһ„бһҖбһ¶бһ“бҹӢбһ–бһҪбһҖбһӮбҹҒбһҠбҹҶбһЎбһҫбһ„бһңбһ¶бҹ”

бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `init()` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҹбһјбһҳбһҖбҹҶбһҺбһҸбҹӢбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһ–бһҺбҹҢбһ”бҹғбһҸбһ„бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбҹ–

```javascript
chrome.runtime.sendMessage({
	action: 'updateIcon',
	value: {
		color: 'green',
	},
});
```

**бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ“бҹҒбҹҮбһ”бһ¶бһ“бһ’бҹ’бһңбһҫ:**
- **бһҖбҹҶбһҺбһҸбҹӢ** бһ–бһҺбҹҢбһ”бҹғбһҸбһ„бһ’бһҳбҹ’бһҳбһҸбһ¶бһҮбһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ
- **бһ•бҹ’бһҠбһӣбҹӢ** бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҳбһҸбһ·бһ—бҹ’бһ“бҹӮбһҖбһ—бҹ’бһӣбһ¶бһҳбһ–бҹҒбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ•бҹ’бһ‘бһ»бһҖ
- **бһ”бһ„бҹ’бһҖбһҫбһҸ** бһӣбҹҶбһ“бһ¶бҹҶбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷ script бһ•бҹ’бһ‘бҹғбһҖбҹ’бһҡбҹ„бһҷ
- **бһ’бһ¶бһ“бһ¶** бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһғбһҫбһүбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһ»бһ“бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

### бһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ бҹ…

бһҘбһЎбһјбһңбһ–бһёбһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бһ–бһёбһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ–бҹҒбһӣбһҠбҹӮбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ CO2 бһҗбҹ’бһҳбһёбһҳбһҖ бһўбҹ’бһ“бһҖбһҸбҹҶбһҺбһ¶бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбҹ’бһ“бһ»бһ„бһ–бһҺбҹҢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ” бһңбһ¶бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбҹҖбһӮбҹ’бһңбһёбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһўбҹҒбһЎбһ·бһ…бһҸбҹ’бһҡбһјбһ“бһ·бһ… - бһҖбҹ’бһҡбҹ„бһҷбһҳбһҖбһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҸбҹӮбһҳбһҪбһҷбҹ”

бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бҹҒбҹҮбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹғ CO2 бһ–бһё APIбҹ–

```javascript
// бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ CO2 бһ–бһё API

calculateColor(CO2);
```

**бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ“бҹҒбҹҮбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбҹ–**
- **бһ—бҹ’бһҮбһ¶бһ”бҹӢ** бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ API бһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһүбҹ’бһүбһ¶бһ—бҹ’бһ“бҹӮбһҖ
- **бһ”бһүбҹ’бһ…бһјбһӣбһ–бһ“бҹ’бһӣбһҝбһ“** бһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ–бҹҒбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбһҳбһҖ
- **бһ’бһ¶бһ“бһ¶** бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҳбһҸбһ·бһ”бһ„бҹ’бһ бһ¶бһүбһ—бҹ’бһ“бҹӮбһҖбһ–бҹҒбһӣбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһҖбһ¶бһ”бһјбһ“бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“
- **бһҡбһҖбҹ’бһҹбһ¶** бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһ…бҹҒбһүбһ–бһёбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү

бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ /dist/background.js бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһҸбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ бҹ…бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ•бҹ’бһ‘бҹғбһҖбҹ’бһҡбҹ„бһҷбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ–

```javascript
// бһҹбҹ’бһҠбһ¶бһ”бҹӢбһҹбһ¶бһҡбһ–бһёбһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡ
chrome.runtime.onMessage.addListener(function (msg, sender, sendResponse) {
	if (msg.action === 'updateIcon') {
		chrome.action.setIcon({ imageData: drawIcon(msg.value) });
	}
});

// бһӮбһјбһҡбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһҢбһёбһҺбһ¶бһҳбһ·бһ…бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Canvas API
// бһҒбҹ’бһ…бһёбһ–бһёбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ energy lollipop - бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһӣбҹ’бһў!
function drawIcon(value) {
	// бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹҶбһҺбһ¶бһҸбҹӢбһҖбһ„бҹӢбһҖбҹ’бһҡбҹ…бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡ
	const canvas = new OffscreenCanvas(200, 200);
	const context = canvas.getContext('2d');

	// бһӮбһјбһҡбһ–бһҺбҹҗбһ•бҹ’бһҖбһ„бҹӢбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ—бһ¶бһ–бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҖбһ¶бһ”бһјбһ“
	context.beginPath();
	context.fillStyle = value.color;
	context.arc(100, 100, 50, 0, 2 * Math.PI);
	context.fill();

	// бһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһңбһ·бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһјбһ”бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбһҖбһҳбһҫбһӣбһңбҹҒбһ”
	return context.getImageData(50, 50, 100, 100);
}
```

**бһ“бҹҒбҹҮбһҮбһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣ script бһ•бҹ’бһ‘бҹғбһҖбҹ’бһҡбҹ„бһҷбһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбҹ–**
- **бһҹбҹ’бһҸбһ¶бһ”бҹӢ** бһҹбһ¶бһҡбһ–бһё script бһҳбҹҒбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (бһҠбһјбһ…бһҮбһ¶бһҳбһ“бҹ’бһҡбҹ’бһҸбһёбһ‘бһ‘бһҪбһӣбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘)
- **бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ** бһҹбҹҶбһҺбһҫ 'updateIcon' бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҠбһјбһҡбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ»бһҖбһҡбһҖ
- **бһ”бһ„бҹ’бһҖбһҫбһҸ** бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһҗбҹ’бһҳбһёбҹ—бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Canvas API
- **бһӮбһјбһҡ** бһңбһ„бҹӢбһ–бһҺбҹҢбһҹбһ¶бһҳбһүбҹ’бһүбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһҳбҹ’бһҡбһ·бһҸбһҖбһ¶бһ”бһјбһ“бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“
- **бһўбһ¶бһ”бҹӢбһҠбҹҒбһҸ** бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһ“бҹ…бһӣбһҫбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
- **бһ”бҹ’бһҡбһҫ** OffscreenCanvas бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһӣбҹ’бһў (бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҡбһ¶бҹҶбһ„бһҹбҹ’бһ‘бһ¶бһҖбҹӢ UI)

вң… бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһё Canvas API бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [бһҳбҹҒбһҡбҹҖбһ“бһ бҹ’бһӮбҹҒбһҳбһўбһ¶бһҖбһ¶бһҹ](../../6-space-game/2-drawing-to-canvas/README.md)бҹ”

```mermaid
sequenceDiagram
    participant CS as бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡ
    participant BG as бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһ•бҹ’бһ‘бҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ
    participant Canvas as бһҖбҹ’бһҡбһЎбһ¶бһ•бҹ’бһ‘бҹғбһҖбҹ’бһҡбҹ…бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ
    participant Browser as бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһҡбһ»бһҖбһҡбһҖ
    
    CS->>BG: sendMessage({action: 'updateIcon', color})
    BG->>Canvas: new OffscreenCanvas(200, 200)
```

### рҹ”„ **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһҡбһјбһ”бһ„бҹ’бһҡбҹҖбһ“**
**бһҷбһӣбҹӢбһҠбһ№бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҹбһҡбһ»бһ” бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳ**бҹ– бһҹбһјбһҳбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ–бһёбһҮбҹҶбһ“бһ¶бһүбһ”бҹҶбһ–бҹҒбһүбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбҹ–
- вң… бһҸбһҫбһҖбһ¶бһҡбһ•бҹ’бһүбһҫбһҹбһ¶бһҡ (message passing) бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҖбҹ’бһ“бһ»бһ„бһ…бһ“бҹ’бһӣбҹ„бҹҮбһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһ•бҹ’бһҹбҹҒбһ„бҹ—?
- вң… бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһҷбһҫбһ„бһ”бҹ’бһҡбһҫ OffscreenCanvas бһҮбҹҶбһ“бһҪбһҹ Canvas бһ‘бһјбһ‘бҹ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҸбҹ’бһҡбһ¶бһ‘бһ·бһ…бһҗбһ¶бһҳбһ–бһӣ?
- вң… бһҸбһҫ API бһҡбһ”бһҹбҹӢ Chrome Runtime бһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһўбҹ’бһңбһёбһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһҹбҹҶбһҺбһ„бҹӢбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ?
- вң… бһҸбһҫ algorithm бһӮбһҺбһ“бһ¶бһ–бһҺбҹҢбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹ…бһ—бһ¶бһ–бһғбһҫбһүбһ”бҹӮбһ”бһҺбһ¶?

**бһҖбһ¶бһӣбҹҲбһ‘бҹҒбһҹбҹҲбһ“бҹғбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ**: бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҘбһЎбһјбһңбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбҹ–
- **бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһҹбһ¶бһҡбһҷбҹүбһ¶бһ„бһ”бҹ’бһҡбһҹбһҫбһҡ**: бһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷ
- **бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһҪбһӣбһ•бҹ’бһ“бҹӮбһҖбһӮбҹҶбһ“бһјбһҡ**: OffscreenCanvas бһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһҡбһўбһ¶бһҖбҹӢ UI
- **бһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ**: бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ
- **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶ**: бһҖбһ¶бһҡбһҹбһҳбҹ’бһўбһ¶бһҸ бһ“бһ·бһ„бһҠбһ№бһҖбһҮбһүбҹ’бһҮбһјбһ“бһ’бһ“бһ’бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң

**бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–**
- **бһ”бһ„бҹ’бһҖбһҫбһҸ** бһҜбһҖбһҹбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҮбһ¶бһҳбһҪбһҷ `npm run build`
- **бһ”бһүбҹ’бһ…бһјбһӣбһҳбҹ’бһҠбһ„бһ‘бҹҖбһҸ** бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ (бһҖбһ»бҹҶбһ—бҹ’бһӣбҹҒбһ…бһҮбҹҶбһ бһ¶бһ“бһ“бҹҒбҹҮ)
- **бһ”бһҫбһҖ** бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһҹбһ„бҹ’бһҖбҹҒбһҸбһғбһҫбһүбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ–бһҺбҹҢбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„
- **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ** бһңбһ¶бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһ…бҹҶбһ–бҹ„бҹҮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбһ¶бһ”бһјбһ“бһ–бһ·бһҸбҹ—бһ–бһёбһ‘бһјбһ‘бһ¶бҹҶбһ„бһ–бһ·бһ—бһ–бһӣбҹ„бһҖ

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҠбһ№бһ„бһ—бҹ’бһӣбһ¶бһҳбҹ—бһҗбһ¶бһҸбһҫбһ–бҹҒбһӣбһ“бҹҒбҹҮбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ„бһҖбһ”бһүбҹ’бһ…бһҖбҹӢбһ¬бһҸбһҫбһҸбһҳбҹ’бһҡбһјбһңбһҡбһ„бҹӢбһ…бһ¶бҹҶбһҗбһ¶бһҳбһ–бһӣбһ”бһҡбһ·бһҹбһ»бһ‘бҹ’бһ’бҹ” бһўбҹ’бһ“бһҖбһ‘бһҫбһ”бһҸбҹӮбһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһ бһҫбһҷбһ”бһ¶бһ“бһҡбҹҖбһ“бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

## бһҮбһҳбҹ’бһҡбһ»бҹҮ GitHub Copilot Agent рҹҡҖ

бһ”бҹ’бһҡбһҫбһҳбһјбһҠбһ”бҹ’бһҡбһ¶бһҮбҹ’бһүбһ¶бһ’бһ·бһ”бһҸбһёбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

**бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶:** бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ“бҹғбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһҠбҹ„бһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ“бҹғбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”

**бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶:** бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһҠбҹӮбһӣбһңбһ¶бһҹбҹӢбһ“бһ·бһ„бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҹбҹҠбһҫбһ”бһўбһ„бҹ’бһҖбҹҒбһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ CO2 бһ–бһё API бһҖбҹҶбһҺбһҸбҹӢбһ–бһҺбҹҢ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһ»бһҒбһ„бһ¶бһҡбһ”бҹ’бһҡбһ¶бһ”бҹӢбһҲбҹ’бһҳбҹ„бҹҮ `performanceTracker` бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫ Performance API бһҠбһҫбһҳбҹ’бһ”бһёбһңбһ¶бһҹбҹӢбһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Console бһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҮбһ¶бһҳбһҪбһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ“бһ·бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһҡбһҷбҹҲбһ–бҹҒбһӣбҹ”

бһҹбһјбһҳбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһё [mode agent](https://code.visualstudio.com/blogs/2025/02/24/introducing-copilot-agent-mode) бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбҹ”

## рҹҡҖ бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…

бһ“бҹҒбҹҮбһҮбһ¶бһ”бҹҒбһҹбһҖбһҖбһҳбҹ’бһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһӮбһҪбһҡбһұбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбҹ– бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһҮбһ¶ open source бһ“бһ·бһ„бһҳбһ¶бһ“бһңбһҸбҹ’бһҸбһҳбһ¶бһ“бһҡбһҷбҹҲбһ–бҹҒбһӣбһңбҹӮбһ„ (бһӮбһ·бһҸбһ•бҹ’бһ‘бҹғбһҠбһјбһ…бһҮбһ¶ Wikipedia, GitHub, бһ¬ Stack Overflow) бһ бһҫбһҷбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“ commit бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶бҹ” бһҸбһҫбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһғбһҫбһүбһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһ–бһҪбһҖбһӮбҹҒбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–? бһҸбһҫбһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮбһҠбҹӮбһӣбһҖбһҫбһҸбһЎбһҫбһ„бһҮбһ¶бһҗбҹ’бһҳбһё?

**бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–**
- **бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ** бһҹбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“ commit бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ¶бһҖбҹ’бһҷбһҠбһјбһ…бһҮбһ¶ "optimize," "performance," бһ¬ "faster"
- **бһҹбһ„бҹ’бһҖбҹҒбһҸ** бһҹбҹҶбһҺбһ»бҹҶбһ”бҹӮбһ”бһ”бһ‘ - бһҸбһҫбһ–бһҪбһҖбһӮбҹҒбһҸбҹӮбһ„бһҸбҹӮбһҮбһҪбһҹбһҮбһ»бһӣбһ”бһүбҹ’бһ бһ¶бһ”бҹӮбһ”бһҺбһ¶?
- **бһҖбҹҶбһҺбһҸбҹӢ** бһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖвҖӢбһ”бһүбҹ’бһ бһ¶бһҡбһўбһ¶бһ”бҹӢбһЎбһјбһңбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ–бһ·бһ”бһ¶бһҖбһҡбһ бҹҗбһҹ
- **бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖ** бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҡбһҖбһғбһҫбһү - бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһўбһ¶бһ…бһҡбҹҖбһ“бһ–бһёбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ–бһ·бһҸ

## бһ”бҹ’бһҡбһӣбһ„бһҖбҹ’бһҡбҹ„бһҷбһҳбҹҒбһҡбҹҖбһ“

[бһ”бҹ’бһҡбһӣбһ„бһҖбҹ’бһҡбҹ„бһҷбһҳбҹҒбһҡбҹҖбһ“](https://ff-quizzes.netlify.app/web/quiz/28)

## бһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһЎбһҫбһ„бһңбһ·бһү бһ“бһ·бһ„бһҡбҹҖбһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“

бһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһҹбһ¶бһҡбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ [performance newsletter](https://perf.email/)

бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһңбһ¶бһҹбҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ¶бһүбһҠбҹ„бһҷбһҳбһҫбһӣбһҸбһ¶бһҳвҖӢбһҡбһҷбҹҲвҖӢбһ—бһ¶бһӮ Performance бһҖбҹ’бһ“бһ»бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶бҹ” бһҸбһҫбһўбҹ’бһ“бһҖбһғбһҫбһүбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ’бҹҶбҹ—бһҺбһ¶бһҳбһҪбһҷбһ‘бҹҒ?

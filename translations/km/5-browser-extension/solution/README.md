# ផ្នែកបន្ថែមកម្មវិធីរុករក Carbon Trigger: កូដបានបញ្ចប់

ការប្រើប្រាស់ API C02 Signal របស់ tmrow ដើម្បីតាមដានការប្រើប្រាស់ថាមពលអគ្គីសនី សាងសង់ផ្នែកបន្ថែមកម្មវិធីរុករក ដើម្បីឲ្យអ្នកមានការរំលឹកដោយផ្ទាល់នៅក្នុងកម្មវិធីរុករករបស់អ្នកអំពីការប្រើប្រាស់ថាមពលអគ្គីសនីរបស់តំបន់របស់អ្នក។ ការប្រើប្រាស់ផ្នែកបន្ថែមនេះជាសមាសធាតុបណ្តោះអាសន្ននឹងជួយឲ្យអ្នកធ្វើការសម្រេចចិត្តលើសកម្មភាពរបស់អ្នកដោយផ្អែកលើព័ត៌មាននេះ។

![screenshot ផ្នែកបន្ថែម](../../../../translated_images/km/extension-screenshot.0e7f5bfa110e92e3.webp)

## ការចាប់ផ្តើម

អ្នកត្រូវតែមាន [npm](https://npmjs.com) ដំឡើងរួចហើយ។ ទាញយកច្បាប់បញ្ជាទាំងនេះទៅក្នុងថតនៅលើកុំព្យូទ័ររបស់អ្នក។

ដំឡើងបណ្ណាល័យទាំងអស់ដែលត្រូវការ៖

```
npm install
```

សង់ផ្នែកបន្ថែមពី webpack

```
npm run build
```

ដើម្បីដំឡើងលើ Edge, ប្រើម៉ឺនុយ 'បីចំណុច' នៅកំពូលមុំស្តាំនៃកម្មវិធីរុករកដើម្បីស្វែងរកផ្ទាំង Extensions។ ពីទីនោះ ប្ដូរទៅជ្រើសរើស 'Load Unpacked' ដើម្បីផ្ទុកផ្នែកបន្ថែមថ្មីមួយ។ បើកថត 'dist' នៅប្រអប់បញ្ចូល ហើយផ្នែកបន្ថែមនោះនឹងត្រូវបានផ្ទុក។ ដើម្បីប្រើវា អ្នកត្រូវការកុំព្យូទ័រលេខ API សម្រាប់ CO2 Signal's API ([ទទួលបាននៅទីនេះតាមអ៊ីមែល](https://www.co2signal.com/) - បញ្ចូលអ៊ីមែលរបស់អ្នកនៅក្នុងប្រអប់នៅលើទំព័រនេះ) និង [កូដសម្រាប់តំបន់របស់អ្នក](http://api.electricitymap.org/v3/zones) ដែលស្របតាម [ផែនទីអគ្គិសនី](https://www.electricitymap.org/map) (នៅ Boston ជាឧទាហរណ៍ ខ្ញុំប្រើ 'US-NEISO')។

![ការដំឡើង](../../../../translated_images/km/install-on-edge.78634f02842c4828.webp)

ពេលកុំព្យូទ័រលេខ API និងតំបន់ត្រូវបានបញ្ចូលចូលទៅក្នុងចំណុចផ្ទាំងផ្នែកបន្ថែម ចំណុចពណ៌នៅប៊ារ ផ្នែកបន្ថែមកម្មវិធីរុករក គួរតែផ្លាស់ប្តូរដើម្បីបញ្ចេញពីការប្រើប្រាស់ថាមពលរបស់តំបន់របស់អ្នក ហើយផ្តល់ការជូនដំណឹងពីសកម្មភាពដែលប្រើថាមពលរស់រវើកយ៉ាងហោចណាស់ដែលអ្នកអាចអនុវត្តបាន។ គំនិតខាងក្រោយនៃប្រព័ន្ធ 'ចំណុច' នេះ ត្រូវបានផ្តល់ឲ្យខ្ញុំដោយផ្នែកបន្ថែម [Energy Lollipop](https://energylollipop.com/) សម្រាប់ការបញ្ចេញមិនមានក្លិនកៅស៊ូនៅ California។

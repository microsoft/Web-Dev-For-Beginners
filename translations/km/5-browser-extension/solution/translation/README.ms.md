бһ”бҹ’бһҡбһҫ API бһҹбһүбҹ’бһүбһ¶ C02 tmrow бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбһғбһҫбһүбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһё бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ бһјбһҸбһҠбһӣбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһ–бҹ’бһҡбһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһёбһ’бҹ’бһ„бһ“бҹӢбһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹҒбҹҮбһҠбҹ„бһҷбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ“бһ№бһ„бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһ–бһёбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ“бҹҒбҹҮбҹ”

## бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһ‘бһёбһ“бҹҒбҹҮ

бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҠбҹҶбһЎбһҫбһ„ [npm] (https://npmjs.com)бҹ” бһ‘бһ¶бһүбһҷбһҖбһ…бһҳбҹ’бһӣбһ„бһҖбһјбһҠбһ“бҹҒбҹҮбһ‘бҹ…бһҗбһҸбһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһҠбҹҶбһЎбһҫбһ„бһҖбһүбҹ’бһ…бһ”бҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ–

```
npm install
```

бһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бһё webpack

```
npm run build
```

бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһЎбһҫбһ„бһ“бҹ…бһӣбһҫ Edge бһ”бҹ’бһҡбһҫбһҳбҹүбһәбһ“бһ»бһҷ 'бһ”бһёбһ…бҹҶбһҺбһ»бһ…' бһ“бҹ…бһҖбҹ’бһ”бҹӮбһӣбһҒбһ¶бһ„бһҹбҹ’бһҠбһ¶бҹҶбһӣбһҫбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„ Sambunganбҹ” бһ–бһёбһ‘бһёбһ“бҹ„бҹҮ бһҮбҹ’бһҡбһҫбһҹ 'Load Unpacked' бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҗбҹ’бһҳбһёбҹ” бһ”бһҫбһҖбһҗбһҸ 'dist' бһ“бҹ…бһӣбһҫбһ–бһ¶бһҖбҹ’бһҷбһҹбҹ’бһ“бһҫбһҡ бһ бһҫбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһ‘бһ»бһҖбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһңбһ¶ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ key API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ API бһҹбһүбҹ’бһүбһ¶ CO2 ([бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҳбһҪбһҷбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһҸбһ¶бһҳбһҡбһҷбҹҲбһўбҹҠбһёбһҳбҹӮбһӣ](https://www.co2signal.com/) - бһ”бһүбҹ’бһ…бһјбһӣбһўбҹҠбһёбһҳбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһўбһ”бҹӢбһ“бҹ…бһ‘бҹҶбһ–бҹҗбһҡбһ“бҹҒбҹҮ) бһ“бһ·бһ„ [бһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ](http://api.electricitymap.org/v3/zones) бһҠбҹӮбһӣбһҹбһ¶бһҖбһҹбһҳбһ“бһ№бһ„ [бһ•бҹӮбһ“бһ‘бһёбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһё](https://www.electricitymap.org/map) (бһ“бҹ… Boston бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҒбҹ’бһүбһ»бҹҶбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫ 'US- NEISO')бҹ”

![sedang memuat turun](../../../../../translated_images/km/install-on-edge.78634f02842c4828.webp)

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё key API бһ“бһ·бһ„бһҸбҹҶбһ”бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбһ»бһ…бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳ бһҒбҹ’бһ‘бһ„бҹӢбһ–бһҺбҹҢбһ“бҹ…бһҡбһ”бһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ»бһҖбһҡбһҖбһ“бһ№бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҗбһ¶бһҳбһ–бһӣбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһҹбһүбҹ’бһүбһ¶ бһўбҹҶбһ–бһёбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бһҳбҹ’бһ„бһ“бҹӢбһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ” бһӮбҹҶбһ“бһ·бһҸбһ“бҹ…бһ–бһёбһҖбҹ’бһҡбҹ„бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ 'dot' бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһІбҹ’бһҷбһҒбҹ’бһүбһ»бҹҶбһҠбҹ„бһҷ [бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ»бһҖбһҡбһҖ Lollipop Tenaga](https://energylollipop.com/) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷ Californiaбҹ”

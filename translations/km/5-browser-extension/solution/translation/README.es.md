# бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ Carbon Triggerбҹ– бһҖбһјбһҠбһ–бҹҒбһүбһӣбҹҒбһү

бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ API бһҹбһүбҹ’бһүбһ¶ C02 бһҡбһ”бһҹбҹӢ tmrow бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһё бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҖбһ¶бһҡбһҮбһјбһ“бһ…бһ¶бҹҶбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһёбһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё ad hoc бһ“бҹҒбҹҮбһ“бһ№бһ„бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһўбҹҶбһ–бһёбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ“бҹҒбҹҮбҹ”

## бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбһ¶бһ“ [npm](https://npmjs.com) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бҹ” бһ‘бһ¶бһүбһҷбһҖбһ…бһҳбҹ’бһӣбһ„бһҖбһјбһҠбһ“бҹҒбҹҮбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһҳбһҪбһҷбһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһҠбҹҶбһЎбһҫбһ„бһҖбһүбҹ’бһ…бһ”бҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ–

```
npm install
```

бһҹбһ¶бһ„бһҹбһ„бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ–бһё webpackбҹ–

```
npm run build
```

бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһЎбһҫбһ„бһ“бҹ…бһӣбһҫ Edge бһҹбһјбһҳбһ”бҹ’бһҡбһҫбһҳбҹүбһәбһ“бһ»бһҷ 'бһ”бһёбһ…бҹҶбһҺбһ»бһ…' бһ“бҹ…бһҳаёёаёЎбһҒбһ¶бһ„бһҹбҹ’бһҠбһ¶бҹҶбһӣбһҫбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ” бһ–бһёбһ‘бһёбһ“бҹ„бҹҮбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ 'бһ•бҹ’бһ‘бһ»бһҖбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһүбҹ’бһ…бһ”бҹӢ' бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҗбҹ’бһҳбһёбһҳбһҪбһҷбҹ” бһ”бһҫбһҖбһҗбһҸ 'dist' бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһ“бһҫ бһ бһҫбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһ‘бһ»бһҖбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһңбһ¶ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбҹ„ API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ API бһҹбһүбҹ’бһүбһ¶ CO2 ([бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҳбһҪбһҷбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһҸбһ¶бһҳбһўбҹҠбһёбһҳбҹӮбһӣ](https://www.co2signal.com/) - бһ”бһүбҹ’бһ…бһјбһӣбһўбҹҠбһёбһҳбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһўбһ”бҹӢбһӣбһҫбһ‘бҹҶбһ–бҹҗбһҡбһ“бҹҒбҹҮ) бһ“бһ·бһ„ [бһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ](http://api.electricitymap.org/v3/zones) бһҠбҹӮбһӣбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһ“бһ№бһ„ [бһ•бҹӮбһ“бһ‘бһёбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһё](https://www.electricitymap.org/map) (бһ“бҹ… Boston бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҒбҹ’бһүбһ»бҹҶбһ”бҹ’бһҡбһҫ 'US-NEISO')бҹ”

![installing](../../../../../translated_images/km/install-on-edge.8bd0ee3ca7dcda1c.webp)

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹ„ API бһ“бһ·бһ„бһҸбҹҶбһ”бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһҠбҹ„бһҸбһ–бһҺбҹҢбһ“бҹ…бһҒбҹ’бһ“бһ„бһҡбһ”бһҹбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ»бһҖбһҡбһҖбһӮбһҪбһҡбһҸбҹӮбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҗбһ¶бһҳбһ–бһӣбһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһҹбһүбҹ’бһүбһ¶бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһўбҹҶбһ–бһёбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҗбһ¶бһҳбһ–бһӣбһҒбҹ’бһ–бһҹбҹӢбһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбҹ” бһӮбҹҶбһ“бһ·бһҸбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ "бһ–бһ·бһ“бҹ’бһ‘бһ»" бһ“бҹҒбҹҮбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ–бһё [бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Energy Lollipop](https://energylollipop.com/) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһ•бҹ’бһҹбҹӮбһ„бһ“бҹ… Californiaбҹ”

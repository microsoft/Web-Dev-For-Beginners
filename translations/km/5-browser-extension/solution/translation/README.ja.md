# ការពង្រីកកម្មវិធីរុករក Carbon Trigger៖ កូដដែលបានបញ្ចប់

ដើម្បីតាមដានការប្រើប្រាស់ថាមពលដោយប្រើប្រាស់ tmrow C02 Signal API ដើម្បីបង្ហាញជារូបភាពជាប្រព័ន្ធរំលឹកលើកម្មវិធីរុករកថាតើការប្រើប្រាស់ថាមពលនៅតំបន់របស់អ្នកមានច្រើនប៉ុនណា យើងនឹងបង្កើតការពង្រីកកម្មវិធីរុករកនេះ។ ការប្រើប្រាស់កម្មវិធីនេះដោយអត្រាតាមតម្រូវអាចជួយឱ្យអ្នកបោះពុម្ពដំណើរការរបស់អ្នកដោយផ្អែកលើព័ត៌មាននេះ។

## បឋមបណ្តាល

ត្រូវតែមានការបញ្ចូល [npm](https://npmjs.com)។ សូមទាញយកចម្លងកូដនេះទៅក្នុងថតនៅលើកុំព្យូទ័ររបស់អ្នក។

ដំឡើងកញ្ចប់ទាំងអស់ដែលចាំបាច់។

```
npm install
```

សន្និធិបង្កើតការពង្រីកកម្មវិធីពី webpack។

```
npm run build
```

ដើម្បីដំឡើងលើ Edge សូមស្វែងរកផ្ទាំង «ការពង្រីកកម្មវិធី» ពីម៉ឺនុយ «ត្រីចុច» នៅផ្នែកខាងលើខាងស្ដាំនៃកម្មវិធីរុករក។ ពីទីនោះជ្រើសរើស «Load Unpacked» ហើយបើកថត «dist» ក្នុងផ្ទាំងផ្ដល់សិទិ្ថ។ កម្មវិធីពង្រីកនឹងត្រូវបានផ្ទុកឡើង។ ដើម្បីប្រើ អ្នកត្រូវការកូនី API សំរាប់ CO2 Signal API ([ទទួលបានមួយនៅទីនេះតាមអ៊ីមែល](https://www.co2signal.com/) - បញ្ចូលអ៊ីមែលរបស់អ្នកក្នុងប្រអប់នៅលើទំព័រនេះ) និង [Electricity Map](https://www.electricitymap.org/map) សម្រាប់ [កូដតំបន់របស់អ្នក](http://api.electricitymap.org/v3/zones) (នៅ Boston ដូចជា «US-NEISO» ត្រូវបានប្រើ)។

![installing](../../../../../translated_images/km/install-on-edge.78634f02842c4828.webp)

បន្ទាប់ពីបញ្ចូលកូនី API និងតំបន់ទៅក្នុងផ្ទាំងផ្ទាល់ខ្លួននៃការពង្រីក អណ្ដូងពណ៌នៅលើរបារកម្មវិធីរុករកនឹងផ្លាស់ប្តូរ ដើម្បីបង្ហាញការប្រើប្រាស់ថាមពលនៅតំបន់របស់អ្នក និងលើកដាក់អ្នកអំពីប្រភេទសកម្មភាពដែលអ្នកគួរធ្វើដោយផ្អែកលើថាមពលដែលត្រូវការពីតំបន់នេះ។ គំនិតស៊ុម «អណ្ដូង» នេះ ត្រូវបានជំពាក់ពីការពង្រីក [Energy Lollipop](https://energylollipop.com/) សម្រាប់កម្រិតការបញ្ចេញកាបូននៅរដ្ឋ California។

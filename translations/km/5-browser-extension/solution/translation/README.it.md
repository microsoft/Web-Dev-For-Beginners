# Estensione del browser Carbon Trigger: бһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ API Signal C02 бһҡбһ”бһҹбҹӢ tmrow бһҠбһҫбһҳбҹ’бһ”бһёбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһё бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбҹ’бһҒбһҮбһ¶бһҸбһ· бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ…бһҳбһ¶бһ“бһҖбһ¶бһҡбһҡбҹҶбһӯбһҖбһҮбһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбҹ’бһҒбһҮбһ¶бһҸбһ·бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бһӮбһәбһүбһ№бһҖбһүбһ¶бһ”бҹӢбһ”бҹүбһ»бһҺбҹ’бһҺбһ¶бҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹҒбҹҮбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ“бһ№бһ„бһҮбһҪбһҷбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ”

## бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

бһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„ [npm](https://npmjs.com)бҹ” бһ‘бһ¶бһүбһҷбһҖбһ…бҹ’бһ”бһ¶бһ”бҹӢбһҳбһҪбһҷбһ“бҹғбһҖбһјбһҠбһ“бҹҒбҹҮбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһҳбһҪбһҷбһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”

бһҸбҹҶбһЎбһҫбһ„бһӮбҹ’бһҡбһ”бҹӢбһҖбһүбҹ’бһ…бһ”бҹӢбһҠбҹӮбһӣбһ‘бһ¶бһҳбһ‘бһ¶бһҡбҹ–

```
npm install
```

бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бһё webpack

```
npm run build
```

бһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹҶбһЎбһҫбһ„бһӣбһҫ Edge бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһ”бҹ’бһҡбһҫбһҳбҹүбһәбһ“бһ»бһҷ "бһ…бҹҶбһҺбһ»бһ…бһ”бһё" бһ“бҹ…бһҮбҹ’бһҡбһ»бһ„бһҒбһ¶бһ„бһӣбһҫбһҹбҹ’бһҠбһ¶бҹҶбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбҹ’бһҒбһҮбһ¶бһҸбһ· бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ”бһҫбһҖ бһҹбһјбһҳбһ”бһҫбһҖбһҡбһ”бҹҖбһ”бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ (бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҶбҹ’бһңбҹҒбһ„)бҹ” бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ "бһ•бҹ’бһ‘бһ»бһҖбһҳбһ·бһ“бһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһ”бҹӢ" бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҗбҹ’бһҳбһёбһҳбһҪбһҷбҹ” бһ”бһҫбһҖбһҗбһҸ "dist" бһ“бҹ… prompt бһ бһҫбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһңбһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбҹүбһјбһҹбһҪбһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ API CO2 Signal (бһўбһ¶бһ…[бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҮбһ¶бһҳбҹүбҹӮбһӣбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ](https://www.co2signal.com/) - бһ–бҹ„бһӣбһўбҹҠбһёбһҳбҹӮбһӣбһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһўбһ”бҹӢбһ“бҹ…бһӣбһҫбһ‘бҹҶбһ–бҹҗбһҡбһ“бҹҒбҹҮ) бһ“бһ·бһ„ [бһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“](http://api.electricitymap.org/v3/zones) бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ“бһ№бһ„ [бһ•бҹӮбһ“бһ‘бһёбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһё](https://www.electricitymap.org/map) (бһ“бҹ… Boston бһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ "US-NEISO")бҹ”

![installazione](../../../../../translated_images/km/install-on-edge.78634f02842c4828.webp)

бһ–бҹҒбһӣбһҠбҹӮбһӣбһҖбһјбһ“бһҹбҹ„ API бһ“бһ·бһ„бһҸбҹҶбһ”бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹҒбҹҮ бһҖбһ“бҹ’бһӣбҹӮбһ„бһ–бһҺбҹҢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ”бһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ»бһҖбҹ’бһҒбһҮбһ¶бһҸбһ·бһӮбһҪбһҡбһҸбҹӮбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҗбһ¶бһҳбһ–бһӣбһҡбһ”бһҹбҹӢбһҸбҹҶбһ”бһ“бҹӢ бһ бһҫбһҷбһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһ“бһјбһңбһ“бһ·бһҡбһ“бҹ’бһҸбһҡбһ—бһ¶бһ–бһ…бҹҶбһ“бһ»бһ…бһ‘бҹ…бһӣбһҫбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҗбһ¶бһҳбһ–бһӣбһҒбҹ’бһ–бһҹбҹӢ бһҠбҹӮбһӣбһӮбһҪбһҡбһҸбҹӮбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ” бһӮбҹҶбһ“бһ·бһҸбһ“бҹ…а№Җаёҡаё·а№үаёӯаёҮаё«аёҘаёұаёҮбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ "бһ…бҹҶбһҺбһ»бһ…" бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһҠбҹ„бһҷ [бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳ Energy Lollipop](https://energylollipop.com/) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһҳбһ¶бһҸбҹ’бһҡбһҠбҹ’бһӢбһ¶бһ“бһ“бҹ… Californiaбҹ”

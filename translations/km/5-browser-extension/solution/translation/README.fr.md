# бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ Carbon Trigger: бһҖбһјбһҠбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ

бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ API C02 Signal бһ“бҹғ tmrow бһҠбһҫбһҳбҹ’бһ”бһёбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһё бһҹбһјбһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһ¶бһ“бһҖбһ¶бһҡбһҡбҹҶбһӣбһ№бһҖбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһёбһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ–бһ·бһҹбҹҒбһҹбһ“бҹҒбҹҮбһ“бһ№бһ„бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһӣбһҫбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ”

## бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбһ¶бһ“ [npm](https://npmjs.com) бһҸбҹҶбһЎбһҫбһ„бһҡбһҪбһ…бҹ” бһ‘бһ¶бһүбһҷбһҖбһ…бҹ’бһ”бһ¶бһ”бҹӢбһ“бҹғбһҖбһјбһҠбһ“бҹҒбҹҮбһҳбһҖбһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһҳбһҪбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһҸбҹҶбһЎбһҫбһ„ packages бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ–

```
npm install
```

Build the extension from webpack

```
npm run build
```

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҶбһЎбһҫбһ„бһӣбһҫ Edge бһҹбһјбһҳбһ”бҹ’бһҡбһҫбһҳбҹүбһәбһ“бһ»бһҷ 'бһ”бһҪбһ“бһ…бҹҶбһҺбһ»бһ…' бһ“бҹ…бһҖбҹҶбһ–бһјбһӣбһҒбһ¶бһ„бһҹбҹ’бһҸбһ¶бҹҶбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„ Extensionsбҹ” бһ–бһёбһ‘бһёбһ“бҹ„бҹҮ бһ”бҹ’бһҡбһҫбһҮбһҳбҹ’бһҡбһҫбһҹ 'Load unpacked extension' бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҗбҹ’бһҳбһёбҹ” бһ”бһҫбһҖбһҗбһҸ 'dist' бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚ бһ бһҫбһҷбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһңбһ¶ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҒбҹ’бһҹбҹӮ API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ API CO2 Signal ([бһ‘бһ‘бһҪбһӣбһҷбһҖбһҸбһ¶бһҳбһўбҹҠбһёбһҳбҹӮбһӣбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ](https://www.co2signal.com/) - бһ”бһүбҹ’бһ…бһјбһӣбһўбҹҠбһёбһҳбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһўбһ”бҹӢбһӣбһҫбһ‘бҹҶбһ–бҹҗбһҡбһ“бҹҒбҹҮ) бһ“бһ·бһ„ [бһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ](http://api.electricitymap.org/v3/zones) бһҠбҹӮбһӣбһ•бҹ’бһӮбһјбһҡбһ•бҹ’бһӮбһ„бһ“бһ№бһ„ [бһ•бҹӮбһ“бһ‘бһёбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһё](https://www.electricitymap.org/map) (бһ“бҹ…бһ”бһјбһҹбҹ’бһҸбһ»бһ„ бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҒбҹ’бһүбһ»бҹҶбһ”бҹ’бһҡбһҫ 'US- NEISO')бҹ”

![installation](../../../../../translated_images/km/install-on-edge.78634f02842c4828.webp)

бһ–бҹҒбһӣбһҠбҹӮбһӣбһҒбҹ’бһҹбҹӮ API бһ“бһ·бһ„бһҸбҹҶбһ”бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһ“бҹғбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖ бһҒбҹ’бһ‘бһ„бҹӢбһ–бһҺбҹҢбһ“бҹ…бһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһҡбһ”бһҹбҹӢбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһӮбһҪбһҡбһҸбҹӮбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҗбһ¶бһҳбһ–бһӣбһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ•бҹ’бһҸбһӣбҹӢбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ“бһјбһңбһҹбһүбҹ’бһүбһ¶бһҸбҹ’бһҡбһ„бҹӢбһӣбһҫбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҗбһ¶бһҳбһ–бһӣбһҒбҹ’бһ–бһҹбҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһ’бҹ’бһңбһҫбҹ” бһӮбҹҶбһ“бһ·бһҸбһ“бҹ…бһ–бһёбһҖбҹ’бһҡбҹ„бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ 'бһ…бҹҶбһҺбһ»бһ…' бһ“бҹҒбҹҮ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһІбҹ’бһҷбһҒбҹ’бһүбһ»бҹҶбһҠбҹ„бһҷбһ“бһјбһң [бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖ Energy Lollipop](https://energylollipop.com/) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ¶бҹҶбһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһ§бһҹбҹ’бһҳбҹҗбһ“бһ“бҹ…бһҖбһ¶бһӣбһёбһ бҹ’бһңбҹҗбһҡбһүбһ¶бҹ”

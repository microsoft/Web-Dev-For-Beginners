# бһҖбһ¶бһ”бһјбһ“бһҸбҹ’бһҡбһ·бһ бҹ’бһӮбһәбһҡ бһ”бҹ’бһҡбһјбһҹбҹ’бһҹбһәбһҡбһҡбһёбһ„ бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ– бһҖбһјбһҠбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ

бһ”бҹ’бһҡбһҫ API CO2 Signal бһҡбһ”бһҹбҹӢ tmrow бһҠбһҫбһҳбҹ’бһ”бһёбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ—бҹ’бһӣбһҫбһ„ бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбҹ„бһҹбҹҠбһәбһҡбһҳбһҪбһҷ бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҠбһӣбҹӢбһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҠбһӣбҹӢбһўбҹ’бһ“бһҖбһҗбһ¶ бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҗбһ¶бһҳбһ–бһӣбһ—бҹ’бһӣбһҫбһ„бһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ—бһ¶бһ–бһ’бҹ’бһ„бһ“бҹӢбһ’бҹ’бһ„бһҡбһ”бҹүбһ»бһҺбҹ’бһ“бһ¶бҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹҒбҹҮ бһ“бһ№бһ„бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһӣбһҫбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ“бҹҒбҹҮбҹ”

## бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҠбҹҶбһЎбһҫбһ„ [npm](https://npmjs.com)бҹ” бһ”бһүбҹ’бһ…бһјбһӣбһҖбһјбһҠбһ“бҹҒбҹҮбһ…бһҳбҹ’бһӣбһ„бһҳбһҪбһҷбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”

бһҠбҹҶбһЎбһҫбһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбҹ–

```
npm install
```

бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбһҮбһ¶бһҳбһҪбһҷ webpack

```
npm run build
```

бһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹҶбһЎбһҫбһ„бһӣбһҫ Edge бһ”бҹ’бһҡбһҫбһҳбһ“бһ»бһңбһҸбҹ’бһҸ 'бһ”бһёбһ…бһ»бһ…' бһҖбҹ’бһ“бһ»бһ„бһҳбһәбһ“бһ»бһҷбһҒбһ¶бһ„бһӣбһҫбһҹбҹ’бһҸбһ¶бҹҶбһ“бҹғбһ”бҹ’бһҡбҹ„бһҹбҹҠбһәбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ” бһ–бһёбһ‘бһёбһ“бҹ„бҹҮбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ 'load unpacked' бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҗбҹ’бһҳбһёбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ бһ”бһҫбһҖбһҗбһҸ 'dist' бһ бһҫбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһ‘бһ»бһҖбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҒбҹ’бһ“Eu API бһҹбҹҶбһҡбһ¶бһ”бҹӢ CO2 Signal ([бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҒбҹ’бһ“Eu API бһҸбһ¶бһҳбһўбҹҠбһёбһҳбҹӮбһӣбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ](https://www.co2snal.com/) - бһ”бһүбҹ’бһ…бһјбһӣбһўбҹҠбһёбһҳбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһўбһ”бҹӢбһӣбһҫбһ‘бҹҶбһ–бҹҗбһҡбһ“бҹҒбҹҮ) бһ“бһ·бһ„ [бһҖбһјбһҠбһҹбҹҶбһҡбһ¶бһ”бҹӢбһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ](http://api.electricitymap.org/v3/zones) бһ“бҹғ [ElectricityMap](https://www.electricitymap.org/map) (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ“бҹ…бһ”бҹүбһјбһҹбҹ’бһ‘бһ»бһ“ бһҒбҹ’бһүбһ»бҹҶбһ”бҹ’бһҡбһҫ 'US-NEISO').

![бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„](../../../../../translated_images/km/install-on-edge.78634f02842c4828.webp)

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһјбһӣбһҒбҹ’бһ“Eu API бһ“бһ·бһ„бһҸбҹҶбһ”бһ“бҹӢбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбһ»бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳ бһңбһ¶бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһ”бҹ’бһҡбҹ„бһҹбҹҠбһәбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹҶбһҺбһ»бһ…бһ–бһҺбҹҢ бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҗбһ¶бһҳбһ–бһӣбһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ•бҹ’бһҠбһӣбҹӢбһҹбһүбҹ’бһүбһ¶бһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶ бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҺбһ¶бһҠбҹӮбһӣбһ…бҹ’бһҡбһҫбһ“бһҗбһ¶бһҳбһ–бһӣбһӮбһҪбһҡбһўбҹ„бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ…бҹҶбһ–бҹ„бҹҮбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһӮбҹҶбһ“бһ·бһҸбһ“бҹ…бһ–бһёбһҖбҹ’бһҡбҹ„бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ 'бһ…бҹҶбһҺбһ»бһ…' бһ“бҹҒбҹҮ бһ”бҹ’бһҡбһҫбһҳбһҖбһ–бһё [Energy Lollipop Extension](https://energylollipop.com/) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһ§бһҹбҹ’бһҳбҹҗбһ“бһҖбҹ’бһ“бһ»бһ„бһҡбһҠбҹ’бһӢвҖӢбһҖбһ¶бһӣбһёбһ бҹ’бһңбҹҗбһҡбһүбһ¶бҹ”

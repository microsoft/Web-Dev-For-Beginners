# ការបង្កើតកម្មវិធីបន្ថែមក្នុងកម្មវិធីរុករក

ការបង្កើតកម្មវិធីបន្ថែមក្នុងកម្មវិធីរុករកគឺជាវិធីកំសាន្តនិងគួរឲ្យចាប់អារម្មណ៍ក្នុងការគិតអំពីប្រសិទ្ធភាពនៃកម្មវិធីរបស់អ្នក ខណៈពេលបង្កើតទ្រព្យសម្បត្តិវែបប្រភេទផ្សេងៗ។ មូឌុលនេះរួមបញ្ចូលមេរៀនអំពីរបៀបដែលកម្មវិធីរុករកដំណើរការ និងរបៀបដាក់ពាណិជ្ជកម្មកម្មវិធីបន្ថែមក្នុងកម្មវិធីរុករក របៀបបង្កើតបែបបទ អ៊ិនធើហ្វេសជាមួយ API និងប្រើការផ្ទុកក្នុងតំបន់ រួមទាំងរបៀបវាស់ប្រសិទ្ធភាពនៃគេហទំព័ររបស់អ្នក ហើយធ្វើឲ្យវាប្រសើរឡើង។

អ្នកនឹងបង្កើតកម្មវិធីបន្ថែមក្នុងកម្មវិធីរុករកមួយដែលដំណើរការលើ Edge, Chrome និង Firefox។ កម្មវិធីបន្ថែមនេះ ដែលគឺដូចជាគេហទំព័រតូចមួយ ដែលត្រូវបានប្ដូរតាមភារកិច្ចជាក់លាក់មួយ ដើម្បីត្រួតពិនិត្យ [C02 Signal API](https://www.co2signal.com) សម្រាប់ការប្រើប្រាស់អគ្គិសនីនិងភាពតឹងតែងកាបូននៃតំបន់ណាមួយ ហើយផ្តល់មតិអំពីទង់ដៅកាបូនរបស់តំបន់នោះ។

កម្មវិធីបន្ថែមនេះអាចត្រូវបានហៅប្រើដោយអ្នកប្រើប្រាស់នៅពេលដែលសមាសភាព API និងកូដតំបន់ត្រូវបានបញ្ចូលក្នុងបែបបទ ដើម្បីកំណត់ការប្រើប្រាស់អគ្គិសនីក្នុងតំបន់នោះ ហើយផ្តល់ទិន្នន័យដែលអាចមានឥទ្ធិពលលើសេវាកម្មអគ្គិសនីរបស់អ្នកប្រើ។ ឧទាហរណ៍ វាអាចមានប្រយោជន៍ក្នុងការពន្យារពេលដំណើរការប្រព័ន្ធស្ងួតគ្រឿងសម្អាង (សកម្មភាពដែលមានកាបូនខ្ពស់) នៅពេលដែលមានការប្រើប្រាស់អគ្គិសនីខ្ពស់ក្នុងតំបន់របស់អ្នក។

## យោងតាម

គំនិតសម្រាប់កម្មវិធីបន្ថែមកាបូនវែបនេះត្រូវបានផ្តល់ដោយ Asim Hussain​ ជាអ្នកដឹកនាំក្រុម Green Cloud Advocacy នៅ Microsoft និងជាអ្នកសរសេរលើ [Green Principles](https://principles.green/)។ វាត្រូវបានបង្កើតដំបូងជាផ្នែកមួយនៃគម្រោង [គេហទំព័រ](https://github.com/jlooper/green)។

រចនាសម្ព័ន្ធនៃកម្មវិធីបន្ថែមក្នុងកម្មវិធីរុករកត្រូវបានមានឥទ្ធិពលដោយកម្មវិធីបន្ថែម COVID របស់ [Adebola Adeniran](https://github.com/onedebos/covtension)។

គំនិតនៅពីក្រោយប្រព័ន្ធតំណរគឺត្រូវបានណែនាំដោយរចនាសម្ព័ន្ធរូបតំណាងនៃកម្មវិធីបន្ថែមរុករក [Energy Lollipop](https://energylollipop.com/) សម្រាប់ការបញ្ចេញហៀបនៅ California។

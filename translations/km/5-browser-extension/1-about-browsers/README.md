## бһҹбҹҶбһҺбһҪбһҡбһҮбҹҶбһ“бһ¶бһүбһҳбһ»бһ“бһҳбҹҒбһҡбҹҖбһ“

### бһӮбҹҶбһ“бһ·бһҸбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ

бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһңбҹӮбһ”бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбһјбһ…бҹ—бһҠбҹӮбһӣбһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҡбһ»бһҖбһҡбһҖбһңбҹӮбһ”бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҠбһјбһ…бһҠбҹӮбһӣбһ…бһҖбҹ’бһҒбһ»бһңбһ·бһҹбҹҗбһҷбһҠбһҫбһҳбһҡбһ”бһҹбҹӢ Tim Berners-Lee бһ–бһёбһңбҹӮбһ”бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бһ„бҹ’бһҡбһёбһҖбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһңбҹӮбһ”бһІбҹ’бһҷбһӣбһҫбһҹбһҖбһ¶бһҡбһҳбһҫбһӣбһҜбһҖбһҹбһ¶бһҡбһҠбһјбһ…бһ’бһҳбҹ’бһҳбһҸбһ¶бҹ” бһ…бһ¶бһ”бҹӢбһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ–бһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҠбҹӮбһӣбһҡбһҖбҹ’бһҹбһ¶бһ”бһҡбһ·бһҷбһ¶бһ”бһүбҹ’бһ бһ¶бһ”бҹ’бһҡбһ¶бһҖбҹӢбһӮбһҺбһ“бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһІбҹ’бһҷбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ‘бҹ…бһҠбһӣбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹ’бһҡбһҫбһҹбһ–бһҺбҹҢбһҠбҹӮбһӣбһҮбһҪбһҷбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҡбһ…бһ“бһ¶бһҡбһҖбһ–бһҺбҹҢбһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮ бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҡбһ»бһҖбһҡбһҖбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғбҹ”

бһҳбһ»бһ“бһ–бҹҒбһӣбһҷбһҫбһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҳбһҖбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҡбһ”бҹҖбһ”бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһңбҹӮбһ”бҹ” бһҠбһјбһ…бһҠбҹӮбһӣ Alexander Graham Bell бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҖбһ¶бһҡбһ•бҹ’бһүбһҫбһҹбҹҶбһЎбҹҒбһ„бһҳбһ»бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘ бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһңбҹӮбһ”бһ“бһ№бһ„бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһІбҹ’бһҷбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“бһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

### бһҡбһ”бҹҖбһ”бһ‘бҹҶбһӣбҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҳбһ¶бһҸбһ·бһҖбһ¶бһңбҹӮбһ”

- **бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ**бһҳбһ¶бһҸбһ·бһҖбһ¶бһ‘бҹҶбһ–бҹҗбһҡбһңбҹӮбһ”бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ–бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ

- **бһ”бһ„бҹ’бһ бһ¶бһү**бһҳбһ¶бһҸбһ·бһҖбһ¶бһҡбһ”бһҹбҹӢбһ‘бҹҶбһ–бҹҗбһҡбһңбҹӮбһ”бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһӮбһ·бһҸбһғбһҫбһү

### рҹ”„ **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҡбҹҖбһ“**  
**бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҹбҹҶбһҺбһ»бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ**бҹ– бһҳбһ»бһ“бһ–бҹҒбһӣбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһўбһ¶бһ…бҹ–  
- вң… бһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбҹҶбһҺбһҫбһңбҹӮбһ” бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҳбһ¶бһҸбһ·бһҖбһ¶  
- вң… бһҖбҹҶбһҺбһҸбҹӢбһ’бһ¶бһҸбһ»бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғбһҹбҹҶбһҺбһ»бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ  
- вң… бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ  
- вң… бһҹбҹ’бһӮбһ¶бһӣбҹӢбһҳбҹүбһјбһҢбҹӮбһӣбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҠбҹӮбһӣбһҖбһ¶бһҡбһ–бһ¶бһҡбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ

**бһҖбһ¶бһҡбһ”бҹ’бһҡбһӣбһ„бһҒбҹ’бһӣбһёбһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„**бҹ– бһҸбһҫбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҸбһ¶бһҳбһҠбһ¶бһ“бһ•бҹ’бһӣбһјбһңбһ–бһёбһҖбһ¶бһҡбһңбһ¶бһҷ URL бһ‘бҹ…бһҖбһ¶бһҡбһҳбһҫбһӣбһ‘бҹҶбһ–бҹҗбһҡбһңбҹӮбһ”бһ”бһ¶бһ“бһ‘бҹҒ?

4. **бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү** бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹҶбһ–бҹҗбһҡбһңбҹӮбһ”бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ  
5. **бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ** бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһҳбһ¶бһҸбһ·бһҖбһ¶бһ“бҹ…бһҮбҹҶбһ бһ¶бһ“бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“

- **[бһҖбһјбһҠбһҸбҹҶбһ”бһ“бҹӢ](http://api.electricitymap.org/v3/zones)**бҹ– бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбһјбһҠбһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбһ¶бһҳбһҡбһҷбҹҲ [Electricity Map](https://www.electricitymap.org/map) (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ Boston бһ”бҹ’бһҡбһҫ 'US-NEISO')

**бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ:**  
- **[Node.js бһ“бһ·бһ„ NPM](https://www.npmjs.com)**бҹ– бһ§бһ”бһҖбһҡбһҺбҹҚбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһүбҹ’бһ…бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһЎбһҫбһ„бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ—бһ¶бһ–бһӮбһҳбҹ’бһҡбҹ„бһ„  
- **[бһҖбһјбһҠбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ](../../../../5-browser-extension/start)**бҹ– бһ‘бһ¶бһүбһҷбһҖбһҗбһҸ `start` бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ

вң… **бһҹбһјбһҳбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳ**бҹ– бһ”бҹ’бһҡuckerбһӣбһҫбһҮбҹҶбһ“бһ¶бһүбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһүбҹ’бһ…бһ”бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҳбһҪбһҷ [бһҳбҹҒбһҡбҹҖбһ“бһҹбһ·бһҖбҹ’бһҹбһ¶] бһҠбҹӮбһӣбһ”бҹҶбһ–бҹҒбһүбһҠбҹ„бһҷбһҳбҹүбһјбһҢбһ»бһӣбһҡбҹҖбһ“бһ“бҹҒбҹҮ (https://docs.microsoft.com/learn/modules/create-nodejs-project-dependencies/?WT.mc_id=academic-77807-sagibbon)

### бһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбһҳбҹ’бһҡбҹ„бһ„

бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбһҳбҹ’бһҡбҹ„бһ„бһҮбһҪбһҷбһҡбҹҖбһ”бһ…бҹҶбһҖбһ¶бһҡбһ„бһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ” бһҠбһјбһ…бһҠбҹӮбһӣбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Alexandria бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҷбһҖбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһ”бһ¶бһ“бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ бһӮбҹҒбһҖбһјбһҠбһҠбҹӮвҖӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӣбҹ’бһўбһӣбһҫбһҖбһҹбҹ’бһ‘бһҪбһҷбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ”бһ¶бһ“бһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһҳбһ»бһ“бҹ–

```
project-root/
в”ңв”Җв”Җ dist/                    # Built extension files
в”Ӯ   в”ңв”Җв”Җ manifest.json        # Extension configuration
в”Ӯ   в”ңв”Җв”Җ index.html           # User interface markup
в”Ӯ   в”ңв”Җв”Җ background.js        # Background script functionality
в”Ӯ   в””в”Җв”Җ main.js              # Compiled JavaScript bundle
в”ңв”Җв”Җ src/                     # Source development files
в”Ӯ   в””в”Җв”Җ index.js             # Your main JavaScript code
в”ңв”Җв”Җ package.json             # Project dependencies and scripts
в””в”Җв”Җ webpack.config.js        # Build configuration
```

**бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҜбһҖбһҹбһ¶бһҡбһ–бһёбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ’бҹ’бһңбһҫбҹ–**  
- **`manifest.json`**бҹ– **бһҖбҹҶбһҺбһҸбҹӢ** бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҳбһ¶бһҸбһ·бһҖбһ¶ бһ•бһ¶бһҖбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһ…бҹҶбһҺбһ»бһ…бһ…бһјбһӣ  
- **`index.html`**бҹ– **бһ”бһ„бҹ’бһҖбһҫбһҸ** бһҳбһ»бһҒбһ„бһ¶бһҡбһҮбһ¶бһ•бҹ’бһ‘бҹғбһҳбһ»бһҒбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ…бһ»бһ…бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ  
- **`background.js`**бҹ– **бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„** бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бҹ’бһ“бҹӮбһҖбһҖбҹ’бһҡбҹ„бһҷ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһҹбҹ’бһҠбһ¶бһ”бҹӢбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ  
- **`main.js`**бҹ– **бһ•бҹ’бһ‘бһ»бһҖ** JavaScript бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢ  
- **`src/index.js`**бҹ– **бһ•бҹ’бһ‘бһ»бһҖ** бһҖбһјбһҠбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҠбһҫбһҳбһҠбҹӮбһӣбһ…бҹҶбһҡбһјбһ„бһ‘бҹ…бһҮбһ¶ `main.js`

> рҹ’Ў **бһӮбһ“бҹ’бһӣбһ№бҹҮбһҡбҹҖбһ”бһ…бҹҶ**бҹ– бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбһјбһ“бһҹбҹ„ API бһ“бһ·бһ„бһҖбһјбһҠбһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ¶бҹҶбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҷбҹ„бһ„бһІбҹ’бһҷбһ„бһ¶бһҷбһ–бҹҒбһӣбһўбһ—бһ·бһңбһҢбҹ’бһҚбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ•бҹ’бһ“бҹӮбһҖбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

вң… **бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–**бҹ– бһҖбһ»бҹҶбһ’бҹ’бһңбһҫбһҮбһ¶бһҖбһ¶бһҡбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҖбһјбһ“бһҹбҹ„ API бһ¬бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ“бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ—бһ–бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһўбҹ’бһ“бһҖбһ–бһёбһҡбһ”бҹҖбһ”бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ“бҹ…бһҮбҹҶбһ бһ¶бһ“бһҖбҹ’бһҡбҹ„бһҷбҹ”

## бһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ

бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ” бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ–бһёбһҡбһ”бҹ’бһҡбҹӮбһңбһҗбһ¶бҹ– бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹҶбһ”бһјбһ„ бһ“бһ·бһ„бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ“бҹҒбҹҮбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҒбһүбһӣбҹҒбһүбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҡбһ…бһ“бһ¶бһ”бһ‘бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ…бһ¶бһ”бҹӢбһҸбһ¶бҹҶбһ„бһ–бһёбһҠбһҫбһҳбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ - бһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ бһ“бһ·бһ„вҖӢбһҮбһҳбҹ’бһҡбһҫбһҹвҖӢбһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҠбһҫбһҳбҹ’бһ”бһёбһҳбһ·бһ“бһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҡбһўбһ¶бһҖбҹӢбһҡбһўбһҪбһӣбҹ”

- **бһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶ** бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһ»бһҒбһ„бһ¶бһҡ JavaScript бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ  

### рҹ”„ **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶ**  
**бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ**бҹ– бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ»бһ“бһ”бһ“бҹ’бһҸбҹ–  
- вң… бһҸбһҫбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ“бҹғбһҳбһҪбһҷбһҜбһҖбһҹбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбһҳбҹ’бһҡбҹ„бһ„бһ”бһ¶бһ“бһ‘бҹҒ?  
- вң… бһҸбһҫбһўбҹ’бһ“бһҖбһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһ”бҹӢбһҖбһјбһҠбһ”бҹ’бһҡбһ—бһ–бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷ build process бһ¬бһ‘бҹҒ?  
- вң… бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶вҖӢбһҷбһҫбһ„бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ…бҹҶбһҺбһ»бһ…бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ“бһ·бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһ‘бҹ…бһҮбһ¶вҖӢбһ•бҹ’бһ“бҹӮбһҖ UI бһ•бҹ’бһҹбҹҒбһ„бҹ—?  
- вң… бһҸбһҫбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ”бҹӮбһ”бһ”бһ‘бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бһ„бҹ’бһҖбһҫбһ“бһ—бһ¶бһ–бһ„бһ¶бһҷбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһҠбҹ„бһҷбһ—бһ¶бһ–бһўбһ¶бһ…бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…?

бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбһҮбҹҶбһ бһ¶бһ“бһҠбҹҶбһ”бһјбһ„бһ“бҹғбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбҹ” бһҠбһјбһ…бһҮбһ¶ бһ”бҹ’бһўбһјбһ“бһ”бһ„бһ”бҹ’бһўбһјбһ“ Wright бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҷбһӣбҹӢбһ–бһёбһўбҹҒбһҡбҹүбһјбһҢбһёбһҺбһ¶бһҳбһ·бһ…бҹ’бһҷбһҳбһ»бһ“бһ–бҹҒбһӣбһ бҹ„бҹҮбһҮбһҫбһ„бһ”бһ¶бһ“ бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһӮбҹҶбһ“бһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ“бһ¶бҹҶбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡ JavaScript бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”

бһ”бҹ’бһҡбһҫбһҡбһ”бҹҖбһ” Agent бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһҪбһҸбһ“бҹҒбҹҮбҹ–  

**бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶:** бһ”бһ„бҹ’бһҖбһҫбһ“бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҠбҹ„бһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹӮбһ”бһ”бһ‘ бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫвҖӢ бһ–бҹҒбһӣбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹ„ API бһ“бһ·бһ„бһҖбһјбһҠбһҸбҹҶбһ”бһ“бҹӢбҹ”  

**бһҹбһ¶бһҡбһҺбһ¶:** бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ JavaScript бһҠбҹӮбһӣбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһңбһ¶бһӣбһҹбҹ„ API бһҳбһ¶бһ“бһҸбһҪбһўбһҖбҹ’бһҹбһҡбһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢ бҹўбҹ  бһҸбһҪ бһ“бһ·бһ„бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбһјбһҠбһҸбҹҶбһ”бһ“бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң (бһҠбһјбһ…бһҮбһ¶ 'US-NEISO')бҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһ»бһҒбһ„бһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбҹ„бһҷбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ–бһҺбҹҢбһҹбҹӮбһ„бһңбһ¶бһӣбһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһ–бһҺбҹҢбһ”бҹғбһҸбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһ–бһҺбҹҢбһҖбҹ’бһҡбһ бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһ»бһҒбһ„бһ¶бһҡбһ”бҹ’бһҠбһјбһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһү/бһӣбһ¶бһҖбҹӢбһҹбҹ„ API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ•бһ„бһҠбҹӮбһҡбҹ”  

бһҹбһјбһҳбһҡбҹҖбһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһё [agent mode](https://code.visualstudio.com/blogs/2025/02/24/introducing-copilot-agent-mode) бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбҹ”  

## рҹҡҖ бһ”бҹ’бһҡбһҖбһҪбһҸ  

бһҳбһҫбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһ бһҫбһҷбһҠбҹҶбһЎбһҫбһ„бһҳбһҪбһҷбһ‘бҹ…бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһңбһ¶бһ“бҹ…бһҡбһ”бҹҖбһ”бһӮбһҪбһҡбһұбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбҹ” бһҸбһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбһҖбһғбһҫбһүбһўбҹ’бһңбһё?

## бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһЎбһҫбһ„бһңбһ·бһү бһ“бһ·бһ„бһҡбҹҖбһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“  

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһ–бһёбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһңбҹӮбһ”бһҪбһҷбҹ—бҹЎбһҡбһҷбҹҲбһ–бҹҒбһӣ бһҸбҹ’бһҡбһјбһңбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бһёбһҖбһ¶бһҡбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбһ“бҹғбһӣбҹ„бһҖбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸ World Wide Web бһҠбҹ„бһҷбһўбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҡбһ”бһҹбҹӢбһңбһ¶бҹ” бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһңбҹӮбһ”бһҹбһ¶бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

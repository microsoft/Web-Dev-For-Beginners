ប្រើប្រាស់ API C02 Signal របស់ tmrow ដើម្បីតាមដានការប្រើប្រាស់អគ្គិសនី សាងសង់កម្មវិធីបន្ថែមក្នុងកម្មវិធីរកមើលវេបសាយ ដើម្បីឲ្យអ្នកមានការរំលឹកនៅក្នុងកម្មវិធីរកមើលវេបសាយរបស់អ្នកអំពីផ្ទៃដែនដែលប្រទេសរបស់អ្នកប្រើប្រាស់អគ្គិសនីធ្ងន់ធ្ងរ។

## ការចាប់ផ្តើម

អ្នកត្រូវតែមាន [npm](https://npmjs.com) តំឡើងរួច។ ទាញយកចម្លងកូដនេះទៅថតមួយនៅលើកុំព្យូទ័ររបស់អ្នក។

តំឡើងគ្រប់កញ្ចប់ដែលត្រូវការ៖

```
npm install
```

សាងសង់កម្មវិធីបន្ថែមពី webpack

```
npm run build
```

ដើម្បីតំឡើងលើ Edge ប្រើម៉ឺនុយ 'បីចុច' នៅ ជ្រុងខាងលើស្តាំនៃកម្មវិធីរកមើលវេបសាយ ដើម្បីស្វែងរកផ្ទាំងកម្មវិធីបន្ថែម។ ពីទីនោះ ជ្រើសរើស 'Load Unpacked' ដើម្បីផ្ទុកកម្មវិធីបន្ថែមថ្មី។ បើកថត 'dist' នៅក្នុងការបង្ហាញភាសា ហើយកម្មវិធីបន្ថែមនឹងត្រូវផ្ទុក។ ដើម្បីប្រើប្រាស់វា អ្នកត្រូវការខ្សែ API សម្រាប់ CO2 Signal's API ([ទទួលបាននៅទីនេះតាមអ៊ីមែល](https://www.co2signal.com/) - បញ្ចូលអ៊ីមែលរបស់អ្នកនៅប្រអប់នៅលើទំព័រនេះ) និង [កូដសម្រាប់តំបន់របស់អ្នក](http://api.electricitymap.org/v3/zones) ដែលសមាជិកនឹង [Electricity Map](https://www.electricitymap.org/map) (នៅ Boston, ឧទាហរណ៍ ខ្ញុំប្រើ 'US-NEISO')។

![installing](../../../../translated_images/km/install-on-edge.78634f02842c4828.webp)

នៅពេលដែលខ្សែ API និងតំបន់ត្រូវបានបញ្ចូលក្នុងចំណុចបញ្ចូលកម្មវិធីបន្ថែម ពណ៌ចុចនៅក្នងប៊ារកម្មវិធីបន្ថែមកម្មវិធីរកមើលវេបសាយគួរត្រូវបានបម្លែងដើម្បីពិចារណាការប្រើប្រាស់ថាមពលនៅតំបន់របស់អ្នក និងផ្តល់ជំនួយអ្នកលើសកម្មភាពដែលប្រើថាមពលធ្ងន់ដែលត្រូវបានសមស្របសម្រាប់អ្នកអនុវត្ត។ គំនិតពីប្រព័ន្ធ 'ចុច' នេះត្រូវបានផ្តល់ដោយកម្មវិធីបន្ថែម [Energy Lollipop extension](https://energylollipop.com/) សម្រាប់ការបញ្ចេញឧស្ម័នកាកសំណល់នៅ California។

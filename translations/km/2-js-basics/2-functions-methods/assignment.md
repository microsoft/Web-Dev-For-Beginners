# រីករាយជាមួយមុខងារ

## សេចក្តីណែនាំ

ក្នុងការងារនេះ អ្នកនឹងអនុវត្តបង្កើតមុខងារប្រភេទផ្សេងៗ ដើម្បីរឹតបន្ថែមយល់ដឹងអំពីមុខងារ JavaScript, ប៉ារ៉ាម៉ែត្រ, តម្លៃលំនាំដើម និងការបញ្ជូនតម្លៃត្រឡប់។

បង្កើតឯកសារ JavaScript មួយឈ្មោះ `functions-practice.js` ហើយអនុវត្តមុខងារដូចខាងក្រោម៖

### ផ្នែកទី 1: មុខងារមូលដ្ឋាន
1. **បង្កើតមុខងារដែលមានឈ្មោះ `sayHello`** ដែលមិនទទួលប៉ារ៉ាម៉ែត្រណាមួយ និងបង្ហាញ "Hello!" ទៅកុងសូល។

2. **បង្កើតមុខងារដែលមានឈ្មោះ `introduceYourself`** ដែលទទួលប៉ារ៉ាម៉ែត្រឈ្មោះ `name` ហើយបង្ហាញសារដូចជា "Hi, my name is [name]" ទៅកុងសូល។

### ផ្នែកទី 2: មុខងារដោយមានប៉ារ៉ាម៉ែត្រលំនាំដើម
3. **បង្កើតមុខងារដែលមានឈ្មោះ `greetPerson`** ដែលទទួលប៉ារ៉ាម៉ែត្រ ២: `name` (បង្ហាញបណ្ដាចាំបាច់) និង `greeting` (ជាជម្រើស, តម្លៃលំនាំដើម = "Hello")។ មុខងារនេះគួរបង្ហាញសារដូចជា "[greeting], [name]!" ទៅកុងសូល។

### ផ្នែកទី 3: មុខងារដែលត្រឡប់តម្លៃ
4. **បង្កើតមុខងារដែលមានឈ្មោះ `addNumbers`** ដែលទទួលប៉ារ៉ាម៉ែត្រ ២ (`num1` និង `num2`) ហើយត្រឡប់តម្លៃសរុបរបស់ពួកវា។

5. **បង្កើតមុខងារដែលមានឈ្មោះ `createFullName`** ដែលទទួលប៉ារ៉ាម៉ែត្រ `firstName` និង `lastName` ហើយត្រឡប់តម្លៃឈ្មោះពេញជា string តែមួយ។

### ផ្នែកទី 4: លាយបញ្ចូលគ្នា
6. **បង្កើតមុខងារដែលមានឈ្មោះ `calculateTip`** ដែលទទួលប៉ារ៉ាម៉ែត្រ ២: `billAmount` (បង្ហាញបណ្ដាចាំបាច់) និង `tipPercentage` (ជាជម្រើស, តម្លៃលំនាំដើម = 15)។ មុខងារនេះគួរគណនានិងត្រឡប់តម្លៃថ្លៃចំនេញ។

### ផ្នែកទី 5: សាកល្បងមុខងារ
បន្ថែមការហៅមុខងារដើម្បីសាកល្បងមុខងារនីមួយៗ និងបង្ហាញលទ្ធផលដោយប្រើ `console.log()` ។

**ឧទាហរណ៍ការហៅសាកល្បង៖**
```javascript
// សាកល្បង​ឧបករណ៍​របស់​អ្នក​នៅទីនេះ
sayHello();
introduceYourself("Sarah");
greetPerson("Alex");
greetPerson("Maria", "Hi");

const sum = addNumbers(5, 3);
console.log(`The sum is: ${sum}`);

const fullName = createFullName("John", "Doe");
console.log(`Full name: ${fullName}`);

const tip = calculateTip(50);
console.log(`Tip for $50 bill: $${tip}`);
```

## បញ្ហាបន្ថែម (ជាជម្រើស)

បើអ្នកចង់បញ្ចេញតម្រូវខ្លួនខាងមុខទៀត៖

1. **បង្កើតមុខងារ arrow function មួយនៃមុខងាររបស់អ្នក**
2. **បង្កើតមុខងារមួយដែលទទួលមុខងារផ្សេងទៀតជាប៉ារ៉ាម៉ែត្រ** (ដូចឧទាហរណ៍ setTimeout ពីមេរៀន)
3. **បន្ថែមការត្រួតពិនិត្យបញ្ចូល** ដើម្បីធានាថាមុខងាររបស់អ្នកដំណើរការជាប្រក្រតីនៅពេលបញ្ចូលខុសប្រក្រតី

# бһҖбһ¶бһҡбһ”бҹ’бһҡбһӣбһ„бһўбһ¶бһҡбҹҒ бһ“бһ·бһ„бһҡбһ„бҹ’бһңбһ·бһӣ

## бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶ

бһ”бһүбҹ’бһ…бһ”бҹӢбһӣбҹҶбһ бһ¶бһҸбҹӢбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҶбһ бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһўбһ¶бһҡбҹҒ бһ“бһ·бһ„бһҡбһ„бҹ’бһңбһ·бһӣбҹ” бһӣбҹҶбһ бһ¶бһҸбҹӢбһ“бһёбһҳбһҪбһҷбҹ—бһҹбһ„бҹӢбһӣбһҫбһӮбҹҶбһ“бһ·бһҸбһ–бһёбһҳбҹҒбһҡбҹҖбһ“ бһ бһҫбһҷбһӣбһҫбһҖбһ‘бһ№бһҖбһ…бһ·бһҸбҹ’бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҡбһ„бҹ’бһңбһ·бһӣбһ“бһ¶бһ“бһ¶ бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһўбһ¶бһҡбҹҒбһ‘бһ¶бҹҶбһ„бһЎбһ¶бһҷбҹ”

### бһӣбҹҶбһ бһ¶бһҸбҹӢ бҹЎбҹ– бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһӣбҹҶбһ“бһ¶бҹҶбһӣбҹҒбһҒ  
бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһүбҹ’бһҮбһёбһӣбҹҒбһҒбһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһўбһ„бҹ’бһӮбһ‘бһё 3 бһҡбһңбһ¶бһ„бһӣбҹҒбһҒ 1 бһҠбһӣбҹӢ 20 бһ бһҫбһҷбһ”бһ„бҹ’бһ бһ¶бһүбһңбһ¶бһӣбһҫбһҖбһ»бһ„бһҹбһјбһӣбҹ”

**бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹҲ**  
- бһ”бҹ’бһҡбһҫбһҡбһ„бҹ’бһңбһ·бһӣ `for` бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһӣбҹҒбһҒбһ”.custom  
- бһ”бһ„бҹ’бһ бһ¶бһүбһӣбҹҒбһҒбһҖбҹ’бһ“бһ»бһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ  
- бһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһ–бһ“бҹ’бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһҸбһҳбҹ’бһҡбһ№бһҳбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ

**бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбҹ–**  
```
3, 6, 9, 12, 15, 18
```
  
> **бһӮбҹҶбһ“бһ»бҹҶ:** бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҹбһҳбһёбһҖбһ¶бһҡбһҮбһ»бҹҶбһңбһ·бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ„бҹ’бһңбһ·бһӣ `for` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҶбһӣбһ„бһӣбҹҒбһҒбҹ”

### бһӣбҹҶбһ бһ¶бһҸбҹӢ бҹўбҹ– бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһўбһ¶бһҡбҹҒ  
бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһ¶бһҡбҹҒбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢ 8 бһӣбҹҒбһҒбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶ бһ бһҫбһҷбһҹбһҡбһҹбҹҒбһҡбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

**бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹҲ**  
- бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһ¶бһҡбҹҒбһҲбҹ’бһҳбҹ„бҹҮ `numbers` бһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢ 8  
- бһҹбһҡбһҹбҹҒбһҡбһҳбһ»бһҒбһ„бһ¶бһҡ `findMaximum()` бһҠбҹӮбһӣбһ”бһ„бҹ’бһңбһ·бһӣбһӣбҹҒбһҒбһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸ  
- бһҹбһҡбһҹбҹҒбһҡбһҳбһ»бһҒбһ„бһ¶бһҡ `findMinimum()` бһҠбҹӮбһӣбһ”бһ„бҹ’бһңбһ·бһӣбһӣбҹҒбһҒбһ‘бһ¶бһ”бһ”бҹҶбһ•бһ»бһҸ  
- бһҹбһҡбһҹбҹҒбһҡбһҳбһ»бһҒбһ„бһ¶бһҡ `calculateSum()` бһҠбҹӮбһӣбһ”бһ„бҹ’бһңбһ·бһӣбһҹбһҡбһ»бһ”бһӣбҹҒбһҒбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ  
- бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бһёбһҳбһҪбһҷбҹ— бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣ

**бһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһҪбһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ–** бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһӣбҹҒбһҒбһ‘бһёбһ–бһёбһҡбһҒбҹ’бһ–бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһўбһ¶бһҡбҹҒбҹ”

### бһӣбҹҶбһ бһ¶бһҸбҹӢ бҹЈбҹ– бһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡ бһўбһ¶бһҡбҹҒбһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡ  
бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһ¶бһҡбҹҒбһҲбҹ’бһҳбҹ„бҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ—бһ¶бһ–бһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҮбһ¶бһ—бһ¶бһ–бһҷбһ”бҹӢ/бһҹбҹҖбһңбһ—бҹ…/бһ…бһҳбҹ’бһҡбҹҖбһ„ бһ бһҫбһҷбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҡбһ„бҹ’бһңбһ·бһӣбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

**бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹҲ**  
- бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһ¶бһҡбҹҒбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡбһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢ 5  
- бһ”бҹ’бһҡбһҫбһҡбһ„бҹ’бһңбһ·бһӣ `for` бһ”бҹ’бһҡбһ–бҹғбһҺбһё бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ’бһ¶бһҸбһ»бһҮбһ¶бһӣбҹҒбһҒ (1. бһҲбҹ’бһҳбҹ„бҹҮбһ’бһ¶бһҸбһ»)  
- бһ”бҹ’бһҡбһҫбһҡбһ„бҹ’бһңбһ·бһӣ `for...of` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ’бһ¶бһҸбһ»бһҮбһ¶бһўбһҖбҹ’бһҹбһҡвҖӢбһ”бҹҶбһ”бҹӮбһҡбһҮбһ¶бһўбһҖбҹ’бһҹбһҡбһ’бҹҶ  
- бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ `forEach()` бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһ¶бһ”бҹӢ бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹҶбһ“бһҪбһ“бһҸбһҪбһўбһҖбҹ’бһҹбһҡбһ•бҹ’бһҸбһӣбҹӢбһ‘бҹ…

**бһӣбһ‘бҹ’бһ’бһ•бһӣбһӮбҹҶбһҡбһјбҹ–**

### бһӣбҹҶбһ бһ¶бһҸбҹӢ бҹӨбҹ– бһҖбһ¶бһҡбһҡбһҫбһҹбһҸбһҳбҹ’бһӣбҹғбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ)  
бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһҪбһҷбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ¶бһҡбҹҒбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһңбһҸбҹ’бһҗбһ»бһҸбҹҶбһҺбһ¶бһ„бһұбҹ’бһҷбһҹбһ·бһҹбҹ’бһҹбҹ”

**бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹҲ**  
- бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһ¶бһҡбҹҒбһҠбҹӮбһӣбһҳбһ¶бһ“бһңбһҸбҹ’бһҗбһ»бһҹбһ·бһҹбҹ’бһҹбһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢ 5 бһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲбҹҲ `name`, `age`, `grade`  
- бһ”бҹ’бһҡбһҫбһҡбһ„бҹ’бһңбһ·бһӣбһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҹбһ·бһҹбҹ’бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһ¶бһҷбһ» 18 бһҶбҹ’бһ“бһ¶бҹҶ бһ¬бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„  
- бһӮбһҺбһ“бһ¶бһҖбһҳбҹ’бһҡбһ·бһҸбһҮбһ¶бһ”бҹӢбһҸбһҳбҹ’бһҡбһјбһңбһ“бҹғбһ–бһ·бһ“бҹ’бһ‘бһ»бһҹбһҡбһ»бһ”бһҡбһ”бһҹбҹӢбһҹбһ·бһҹбҹ’бһҹбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ  
- бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһ¶бһҡбҹҒбһҗбҹ’бһҳбһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбҹӮбһҹбһ·бһҹбҹ’бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһ–бһ·бһ“бҹ’бһ‘бһ»бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„ 85 бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ

**бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбҹҶбһҡбһјбҹ–**  
```javascript
const students = [
  { name: "Alice", age: 17, grade: 92 },
  { name: "Bob", age: 18, grade: 84 },
  // бһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбһ·бһҹбҹ’бһҹбһ”бһ“бҹ’бһҗбҹӮбһҳ...
];
```


## бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбҹ–  
1. бһҡбһҸбҹӢбһӣбҹҶбһ бһ¶бһҸбҹӢбһ“бһёбһҳбһҪбһҷбҹ—бһ“бҹ…бһӣбһҫбһҖбһ»бһ„бһҹбһјбһӣбһҡбһ”бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ»бһҖбһҡбһҖ  
2. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбҹ’бһҡбһјбһңбһҸбһ¶бһҳбһҖбһ¶бһҡбһҡбҹҶбһ–бһ№бһ„  
3. бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҮбһ¶бһҳбһҪбһҷбһҸбһҳбҹ’бһӣбҹғбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бҹ—  
4. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһҡбһҺбһёбһҮбҹ’бһҡбҹ„бҹҮ (бһўбһ¶бһҡбҹҒ бһӮбҹ’бһҳбһ¶бһ“бһ’бһ¶бһҸбһ» бһ¬ бһҳбһ¶бһ“бһ’бһ¶бһҸбһ»бһҸбҹӮбһҳбһҪбһҷ) бһ”бһ¶бһ“

## бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҠбһ¶бһҖбҹӢбһҹбҹ’бһ“бһҫ

бһ”бһүбҹ’бһ…бһјбһӣбһўбҹ’бһңбһёбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҹбҹ’бһ“бһҫбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–  
- бһҖбһјбһҠ JavaScript бһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһӣбһҳбҹ’бһўбһ·бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҶбһ бһ¶бһҸбҹӢбһ“бһёбһҳбһҪбһҷбҹ—  
- бһҡбһјбһ”бһҗбһҸбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ бһ¬ бһӣбһ‘бҹ’бһ’бһ•бһӣбһўбһҖбҹ’бһҹбһҡбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ  
- бһ–бһҺбҹҢбһ“бһ¶бһҒбҹ’бһӣбһёбһўбҹҶбһ–бһёбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҡбһ„бҹ’бһңбһ·бһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ“бһёбһҳбһҪбһҷбҹ— бһ“бһ·бһ„бһ бҹҒбһҸбһ»бһ•бһӣ

| **бһҳбһ»бһҒбһ„бһ¶бһҡ** | бһӣбҹҶбһ бһ¶бһҸбҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бһүбҹ’бһ…бһ”бҹӢбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһҡбһҪбһҳбһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бһ”бһ“бҹ’бһҗбҹӮбһҳ | бһӣбҹҶбһ бһ¶бһҸбҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһҸбһҳбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң | бһӣбҹҶбһ бһ¶бһҸбҹӢбһҒбҹ’бһӣбҹҮбһҳбһ·бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ бһ¬бһҳбһ¶бһ“бһҖбҹҶбһ бһ»бһҹ |

| **бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ** | бһҳбһ¶бһ“бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһ–бҹҒбһүбһӣбҹҒбһүбһ–бһ“бҹ’бһҷбһӣбҹӢбһҖбһ¶бһҡбһӮбһ·бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸ | бһҳбһ¶бһ“бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ | бһҳбһ¶бһ“бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһҸбһ·бһ… бһ¬бһӮбҹ’бһҳбһ¶бһ“бһ‘бҹҒ |
| **бһ”бҹ’бһҡбһҫбһҡбһ„бҹ’бһңбһ·бһӣ** | бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһӣбҹ’бһўбһўбҹҶбһ–бһёбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҡбһ„бҹ’бһңбһ·бһӣбһ“бһ¶бһ“бһ¶ | бһ”бҹ’бһҡбһҫбһҡбһ„бҹ’бһңбһ·бһӣбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ·бһ“бһҳбһ¶бһ“бһ…бҹҶбһ“бһҪбһ“бһ”бҹҶбһ•бһ»бһҸ | бһ”бҹ’бһҡбһҫбһҡбһ„бҹ’бһңбһ·бһӣбһҒбһ»бһҹ бһ¬бһ”бҹ’бһҡбһҫбһҳбһ·бһ“бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– |

## бһҹбҹҶбһҺбһҪбһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһӣбҹҶбһ бһ¶бһҸбҹӢ бһҹбһјбһҳбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бҹ–  
1. бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҡбһ„бҹ’бһңбһ·бһӣбһҺбһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҗбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһ бҹҒбһҸбһ»бһўбҹ’бһңбһё?  
2. бһҸбһҫбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҲбһҳбһҳбһ»бһҒбһ–бҹҒбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһўбһ¶бһҡбҹҒ?  
3. бһҸбһҫбһҮбҹҶбһ“бһ¶бһүбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ‘бҹ…бһӣбһҫбһӮбҹҶбһҡбҹ„бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһңбһ·бһ”бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ–бһ·бһҸбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…?  
4. бһҸбһҫбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ?

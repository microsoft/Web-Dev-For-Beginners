бһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹ’бһ“бһјбһӣбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷ JavaScript бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбҹҒбһҒ бһўбһҸбҹ’бһҗбһ”бһ‘ бһҸбһҳбҹ’бһӣбҹғ true/false бһ“бһ·бһ„бһҷбһӣбҹӢбһҗбһ¶бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһӮбҹҶбһ“бһ·бһҸбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҳбһҫбһӣбһ‘бҹ…бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбһ·бһ“бһҳбҹӮбһ“бһ“бҹ…бһҠбҹҶбһ”бһјбһ„бһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бһҷбһӣбҹӢбһҠбһ№бһ„бһңбһ¶бһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбҹ”

бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бһ№бһ„бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһўбҹ’бһңбһёбҹ—бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ JavaScript бһӣбһҳбҹ’бһўбһ·бһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ” бһҠбһјбһ…бһҮбһ¶бһ”бһ»бһӮбҹ’бһӮбһӣбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҹбһ¶бһҡбһ’бһ¶бһҸбһ»бһҹбҹҶбһҺбһ„бҹӢбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бһҮбһ¶бһҳбһ»бһ“ бһҳбһ»бһ“бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһ–бҹ’бһҡбҹҮбһңбһ·бһ бһ¶бһҡ бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ“бһ№бһ„бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҹбһ„бҹӢбһ“бҹ…бһҒбһ¶бһ„бһҳбһ»бһҒбҹ”

## бһўбһҗбҹҒбһҡ

бһўбһҗбҹҒбһҡбһҮбһ¶бһҗбҹ’бһ“бһ¶бҹҶбһҹбҹҶбһҺбһ»бҹҶбһ•бҹ’бһ“бҹӮбһҖбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ” бһҠбһјбһ…бһҮбһ¶бһ’бһ»бһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһӣбһ¶бһ”бһҺбҹ’бһҠбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“бһҠбҹӮбһӣбһўбһ¶бһЎбҹ’бһӮбҹӮбһ“бһҳбҹҒбһҠбҹ’бһҷбһ¶бһңбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһҹбһ¶бһҡбһ’бһ¶бһҸбһ»бһ•бҹ’бһҹбҹҒбһ„бҹ— бһўбһҗбҹҒбһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҡбһҖбҹ’бһҹбһ¶бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһҲбҹ’бһҳбҹ„бҹҮбһ–бһҺбҹҢбһ“бһ¶ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҷбҹ„бһ„бһңбһ·бһүбһ”бһ¶бһ“бһ“бҹ…бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбҹ” бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ…бһ„бһ…бһ¶бҹҶбһўбһ¶бһҷбһ»бһҡбһ”бһҹбҹӢбһ“бһҡбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢ? бһҡбһҖбҹ’бһҹбһ¶бһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһҗбҹҒбһҡбһҲбҹ’бһҳбҹ„бҹҮ `age`бҹ” бһ…бһ„бҹӢбһҸбһ¶бһҳбһҠбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ? бһҡбһҖбҹ’бһҹбһ¶бһңбһ¶вҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„вҖӢбһўбһҗбҹҒвҖӢбһҡвҖӢбһҲбҹ’бһҳбҹ„бҹҮ `userName`бҹ”

бһҷбһҫбһ„бһ“бһ№бһ„бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҮбһ¶бһҳбҹүбһјбһҠбһҳбҹүбҹӮбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҗбҹҒбһҡвҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ JavaScriptбҹ” бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһҸбҹҶбһҺбһ¶бһ„бһІбҹ’бһҷбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ—бһ¶бһҹбһ¶бһҡбһҷбҹҲбһ–бҹҒбһӣбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҶбҹ’бһ“бһ¶бҹҶ бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҠбҹӮбһӣбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбҹ”

бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„ **бһ”бҹ’бһҡбһҖбһ¶бһҹ** бһўбһҗбҹҒбһҡбһҳбһ¶бһ“бһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ»бһ”бҹӮбһ”бһҡбҹҖбһ„бҹ—бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ **[keyword] [name]**бҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһ–бһёбһ…бҹҶбһҺбһ»бһ…бһ–бһёбһҡбһҠбҹӮбһӣбҹ–

- **бһ–бһ¶бһҖбҹ’бһҷбһӮбһ“бҹ’бһӣбһ№бҹҮ**бҹ” бһ”бҹ’бһҡбһҫ `let` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҗбҹҒбһҡбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҠбһјбһҡ бһ”бһҫбһҳбһ·бһ“бһҠбһјбһ…бҹ’бһ“бҹ„бҹҮбһ”бҹ’бһҡбһҫ `const` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһ“бҹ…бһҠбһҠбҹӮбһӣбҹ”
- **бһҲбҹ’бһҳбҹ„бҹҮбһўбһҗбҹҒбһҡ** бһҮбһ¶бһҲбҹ’бһҳбҹ„бҹҮбһ–бһҺбҹҢбһ“бһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһўбҹ’бһ“бһҖбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ”

вң… бһ–бһ¶бһҖбҹ’бһҷбһӮбһ“бҹ’бһӣбһ№бҹҮ `let` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„ ES6 бһ бһҫбһҷбһ•бҹ’бһҸбһӣбҹӢбһўбһҗбҹҒбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһҖбһ“бһјбһң _block scope_ бһҠбҹӮбһӣбһ бҹ…бһҗбһ¶бҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бҹ’бһҡбһҫ `let` бһ¬ `const` бһҮбҹҶбһ“бһҪбһҹ `var` бһҠбҹӮбһӣбһ…бһ¶бһҹбҹӢбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһўбҹҶбһ–бһё block scopes бһҮбһ¶бһҳбһҪбһҷбһ—бһ¶бһ–бһҮбҹ’бһҡбһ¶бһӣбһҮбҹ’бһҡбҹ…бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһҖбҹ’бһҡбҹ„бһҷбҹ”

### бһҖбһ¶бһҡбһ„бһ¶бһҡ - бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһҗбҹҒбһҡ

1. **бһ”бҹ’бһҡбһҖбһ¶бһҹбһўбһҗбҹҒбһҡ**бҹ” бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҗбҹҒбһҡбһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ–

    ```javascript
    let myVariable;
    ```

   **бһўбҹ’бһңбһёбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҠбҹ„бһҷбһ“бҹҒбҹҮбҹ–**
   - бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹ… JavaScript бһұбҹ’бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһёбһҸбһ¶бҹҶбһ„бһ•бҹ’бһ‘бһ»бһҖбһҠбҹӮбһӣбһ бҹ…бһҗбһ¶ `myVariable`
   - JavaScript бһ•бҹ’бһҠбһӣбҹӢбһҖбһ“бҹ’бһӣбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҗбҹҒбһҡбһ“бҹҒбҹҮ
   - бһўбһҗбҹҒбһҡбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҳбһ·бһ“бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҺбһ¶бһҳбһҪбһҷбһ‘бҹҒ (undefined)

2. **бһ•бҹ’бһҸбһӣбҹӢбһҸбһҳбҹ’бһӣбҹғбһңбһ¶**бҹ” бһҘбһЎбһјбһңLet's put something in our variable:

    ```javascript
    myVariable = 123;
    ```

   **бһҡбһ”бҹҖбһ”бһ•бҹ’бһҸбһӣбҹӢбһҸбһҳбҹ’бһӣбҹғбҹ–**
   - бһўбҹ’бһ“бһҖбһ”бҹ’бһҸбһјбһҡбһҸбһҳбҹ’бһӣбҹғ 123 бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһўбһ»бһёбһ•бһ¶бһҡ `=` бһ‘бҹ…бһўбһҗбҹҒбһҡбһҡбһ”бһҹбҹӢбһҷбһҫбһ„
   - бһўбһҗбҹҒбһҡбһҘбһЎбһјбһңбһҖбһ¶бһ“бҹӢбһҸбһҳбҹ’бһӣбҹғбһ“бҹҒбҹҮбһ”бҹ’бһҠбһјбһҡбһ–бһё undefined
   - бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҷбҹ„бһ„бһҸбһҳбҹ’бһӣбҹғбһ“бҹҒбҹҮбһҮбһ¶бһҳбһ’бҹ’бһҷбҹ„бһ”бһ¶бһҷ `myVariable` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

   > бһ‘бҹҶбһҡбһ„бҹӢбҹҲ бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ `=` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ "бһўбһ»бһёбһ•бһ¶бһҡвҖӢбһҸбҹҶбһҺбһ¶бһ„", бһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһ‘бҹ…бһўбһҗбҹҒбһҡбҹ” бһңбһ¶бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡвҖӢбһ‘бҹҒбҹ”

3. **бһ’бҹ’бһңбһҫбһңбһ¶бһҸбһ¶бһҳбһңбһ·бһ’бһёбһҶбҹ’бһӣбһ¶бһҸ**бҹ” бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҷбһҫбһ„бһ…бҹӮбһҖбһҡбһҪбһҳбһ–бһёбһҡбһ”бҹҖбһ”бһ“бҹ„бҹҮбһҮбһ¶вҖӢбһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бҹ–

    ```javascript
    let myVariable = 123;
    ```

    **бһңбһ·бһ’бһёбһ“бҹҒбҹҮбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ…бҹ’бһҡбһҫбһ“бҹ–**
    - бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҖбһ¶бһҹбһўбһҗбҹҒбһҡбһ бһҫбһҷбһ•бҹ’бһҸбһӣбҹӢбһҸбһҳбҹ’бһӣбҹғбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҳбһҪбһҷ
    - бһңбһ¶бһҮбһ¶бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҳбһ¶бһ“бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһ…бҹҶбһ–бҹ„бҹҮбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһ¶бһҡ
    - бһңбһ¶бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһјбһҠбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ”бҹ’бһҡбһңбҹӮбһ„ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһӣбһ»бһ”бһ”бҹҶбһ”бһ¶бһҸбҹӢбһ—бһ¶бһ–бһ…бҹ’бһҡбһ”бһјбһҖбһ…бҹ’бһҡбһ”бһӣбҹӢ

4. **бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҷбҹ„бһ”бһӣбҹӢ**бҹ” бһҸбһҫбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһ”бһҫбһ…бһ„бҹӢбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһӣбҹҒбһҒбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ?

   ```javascript
   myVariable = 321;
   ```

   **бһҷбһӣбҹӢбһ–бһёбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһҳбҹ’бһӣбҹғ ЧһЧ—Ч“Ч©бҹ–**
   - бһўбһҗбҹҒбһҡбһҘбһЎбһјбһңбһҖбһ¶бһ“бҹӢбһҸбһҳбҹ’бһӣбҹғ 321 бһҮбҹҶбһ“бһҪбһҹ 123
   - бһҸбһҳбҹ’бһӣбҹғбһҳбһ»бһ“бһ”бһ¶бһ“бһҮбҹҶбһ“бһҪбһҹ вҖ“ бһўбһҗбҹҒбһҡбһҡбһҖбҹ’бһҹбһ¶бһҸбһҳбҹ’бһӣбҹғбһҸбҹӮбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһ–бҹҒбһӣ
   - бһ—бһ¶бһ–бһўбһ¶бһ…бһ”бҹ’бһҠбһјбһҡбһ“бҹҒбҹҮбһӮбһәбһҮбһ¶ бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғбһўбһҗбҹҒбһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҖбһ¶бһҹбһҮбһ¶бһҳбһҪбһҷ `let`

   вң… бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһңбһ¶! бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡ JavaScript бһҸбҹ’бһҡбһ„бҹӢбһӣбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ”бһҫбһҖбһ”бһҺбҹ’бһҸбһ¶бһүбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһ“бһ·бһ„бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ”бҹ’бһҡбһўбһ”бҹӢ Developer Toolsбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бҹҶбһ–бҹҗбһҡбһ”бһүбҹ’бһҮбһ¶бһҡ (console), бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ; бһҹбһҡбһҹбҹҒбһҡ `let myVariable = 123`, бһ…бһ»бһ… Enter, бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҹбһҡбһҹбҹҒбһҡ `myVariable`бҹ” бһҸбһҫбһҳбһ¶бһ“бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„? бһ…бҹҶбһҺбһ¶бҹҶ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһӮбҹҶбһ“бһ·бһҸбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ—бҹ”

### рҹ§  **бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҮбҹҶбһ“бһ¶бһүбһўбһҗбҹҒбһҡбҹ– бһҖбһ¶бһҡбһҠбһ№бһ„бһ…бһ·бһҸбҹ’бһҸ**

**бһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҷбҹүбһ¶бһ„бһҺбһ¶бһўбҹҶбһ–бһёбһўбһҗбҹҒбһҡбҹ–**
- бһҸбһҫбһўбһ¶бһ…бһ–бһ“бҹ’бһҷбһӣбҹӢбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһ”бҹ’бһҡбһҖбһ¶бһҹ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһҸбһҳбҹ’бһӣбҹғбһўбһҗбҹҒбһҡбһ”бһ¶бһ“бһ‘бҹҒ?
- бһҸбһҫбһҳбһ¶бһ“бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ”бҹ’бһҡбһҫбһўбһҗбҹҒбһҡбһҳбһ»бһ“бһ–бҹҒбһӣбһ”бҹ’бһҡбһҖбһ¶бһҹбһңбһ¶?
- бһҸбһҫбһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҮбҹ’бһҡбһҫбһҹ `let` бһҮбҹҶбһ“бһҪбһҹ `const` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҗбҹҒбһҡ?

## бһўбһҗбҹҒбһҡбһҸбһҳбҹ’бһӣбҹғбһҗбҹҒбһҡ

бһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ·бһ“бһӮбһҪбһҡбһ”бҹ’бһҠбһјбһҡбһ“бҹ…бһ–бҹҒбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһӮбһ·бһҸбһўбһҗбҹҒбһҡбһҸбһҳбҹ’бһӣбҹғбһҗбҹҒбһҡбһҠбһјбһ…бһҮбһ¶бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бһҠбҹӮбһӣ Euclid бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһҖбҹ’бһҡбһ·бһ…бһ”бһ»бһҡбһ¶бһҺ вҖ“ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„ бһ“бһ·бһ„бһ”бһ¶бһ“бһ…бһ„бһҖбҹ’бһҡбһ„бһ бһҫбһҷ бһңбһ¶бһ“бҹ…бһҳбһ·бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһЎбһҫбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҷбҹ„бһ„бһ”бһ“бҹ’бһҠбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”

бһўбһҗбҹҒбһҡбһҸбһҳбҹ’бһӣбҹғбһҗбҹҒбһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҠбһјбһ…бһўбһҗбҹҒбһҡ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ– бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҸбһҳбҹ’бһӣбҹғбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒ бһңбһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ”бһ¶бһ“бһ‘бҹҒбҹ” бһ—бһ¶бһ–бһҳбһ·бһ“бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ“бҹҒбҹҮбһҮбһҪбһҷбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбһҡбһҺбһёбһҖбҹҶбһ бһ»бһҹбһҠбҹ„бһҷбһ…бҹғбһҠбһ“бҹ’бһҷбһ…бҹҶбһ–бҹ„бҹҮбһҸбһҳбҹ’бһӣбҹғбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘бһўбһҗбҹҒбһҡбһҸбһҳбҹ’бһӣбҹғбһҗбҹҒбһҡбһҸбһ¶бһҳбһӮбҹҶбһ“бһ·бһҸбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бһ№бһ„бһўбһҗбҹҒбһҡ бһҠбҹ„бһҷбһ•бҹ’бһҸбһӣбҹӢбһҸбҹӮбһ–бһ¶бһҖбҹ’бһҷбһӮбһ“бҹ’бһӣбһ№бҹҮ `const`бҹ” бһ”бһҫбһ’бһҳбҹ’бһҳбһҸбһ¶бһҮбһ¶бһўбһҗбҹҒбһҡбһҸбһҳбҹ’бһӣбҹғбһҗбҹҒбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҖбһ¶бһҹбһҮбһ¶бһҳбһҪбһҷбһўбһҖбҹ’бһҹбһҡбһ’бҹҶбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ”

```javascript
const MY_VARIABLE = 123;
```

**бһ“бҹҒбҹҮбһҮбһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбһјбһҠбһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбҹ–**
- **бһ”бһ„бҹ’бһҖбһҫбһҸ** бһўбһҗбҹҒбһҡбһҗбҹҒбһҡбһҲбҹ’бһҳбҹ„бҹҮ `MY_VARIABLE` бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғ 123
- **бһ”бҹ’бһҡбһҫ** бһҹбҹҶбһ–бһ¶бһ’бһҖбһ¶бһҡбһ“бҹғбһҲбҹ’бһҳбҹ„бҹҮбһўбһҗбҹҒбһҡбһҗбҹҒбһҡбһҠбҹ„бһҷбһўбһҖбҹ’бһҹбһҡбһ’бҹҶ
- **бһ…бҹҖбһҹбһңбһ·бһү** бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбһ“бҹғбһҸбһҳбҹ’бһӣбҹғбһ“бҹҒбҹҮ

бһўбһҗбҹҒбһҡбһҸбһҳбҹ’бһӣбҹғбһҗбҹҒбһҡбһҳбһ¶бһ“бһ…бҹ’бһ”бһ¶бһ”бҹӢбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ–бһёбһҡбһҠбҹӮбһӣбҹ–

- **бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһҸбһӣбҹӢбһҸбһҳбҹ’бһӣбҹғбһұбҹ’бһҷбһңбһ¶вҖӢбһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ‘бҹ…бһҳбһ»бһҒ** вҖ“ бһҳбһ·бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҳбһ¶бһ“бһўбһҗбҹҒбһҡбһҗбҹҒбһҡбһҗбҹ’бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһ‘бҹҒ!
- **бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһҸбһҳбҹ’бһӣбҹғбһ“бҹ„бҹҮбһ”бһ¶бһ“бһ‘бҹҒ** вҖ“ JavaScript бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹҶбһ бһ»бһҹ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбҹ” бһҳбһҖбһҳбһҫбһӣбһҗбһ¶бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„бҹ–

   **бһҸбһҳбҹ’бһӣбҹғбһҹбһ¶бһҳбһүбҹ’бһү** - бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһҳбһ·бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбҹ–
   
      ```javascript
      const PI = 3;
      PI = 4; // бһҳбһ·бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ
      ```

   **бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ…бһ„бһ…бһ¶бҹҶбҹ–**
   - **бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳ** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҠбһјбһҡбһҸбһҳбҹ’бһӣбҹғбһҗбҹҒбһҡбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹҶбһ бһ»бһҹ
   - **бһҖбһ¶бһҡбһ–бһ¶бһҡ** бһҸбһҳбҹ’бһӣбҹғбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ–бһёбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ…бҹғбһҠбһ“бҹ’бһҷ
   - **бһ’бһ¶бһ“бһ¶** бһҗбһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹ…бһҸбҹӮбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ‘бҹ…бһҸбһ¶бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

   **бһҖбһ¶бһҡбһҷбҹ„бһ„бһўбһҸбҹ’бһҗбҹҲбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһ¶бһҡбһ–бһ¶бһҡ** - бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҳбһ·бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбҹ–
   
      ```javascript
      const obj = { a: 3 };
      obj = { b: 5 } // бһҳбһ·бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ
      ```

   **бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһӮбҹҶбһ“бһ·бһҸбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ–**
   - **бһҖбһ¶бһҡбһ–бһ¶бһҡ** бһҖбһ¶бһҡбһҮбҹҶбһ“бһҪбһҹбһўбҹ’бһңбһёбһҳбһҪбһҷбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҮбһ¶бһҳбһҪбһҷбһңбһҸбҹ’бһҗбһ»бһҗбҹ’бһҳбһё
   - **бһҖбһ¶бһҡбһ–бһ¶бһҡ** бһҖбһ¶бһҡбһҷбҹ„бһ„бһ‘бҹ…бһңбһҸбҹ’бһҗбһ»бһҠбһҫбһҳ
   - **бһҡбһҖбҹ’бһҹбһ¶** бһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһңбһҸбҹ’бһҗбһ»бһҖбҹ’бһ“бһ»бһ„бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶ

    **бһҸбһҳбҹ’бһӣбҹғбһңбһҸбҹ’бһҗбһ»бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһ¶бһҡбһ–бһ¶бһҡ** - бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбҹ–
    
      ```javascript
      const obj = { a: 3 };
      obj.a = 5;  // бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ
      ```

      **бһ”бҹҶбһ”бҹӮбһҖбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбһҫбһҸбһЎбһҫбһ„бҹ–**
      - **бһҖбҹӮбһ”бҹ’бһҡбҹӮ** бһҸбһҳбҹ’бһӣбҹғбһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһҸбҹ’бһҗбһ»
      - **бһҡбһҖбҹ’бһҹбһ¶** бһҖбһ¶бһҡбһҷбҹ„бһ„бһңбһҸбҹ’бһҗбһ»бһҠбһҠбҹӮбһӣ
      - **бһ”бһ„бҹ’бһ бһ¶бһү** бһҗбһ¶бһҸбһҳбҹ’бһӣбҹғбһҖбҹ’бһ“бһ»бһ„бһңбһҸбҹ’бһҗбһ»бһҒбһ»бһҹбһ”бҹ’бһӣбҹӮбһҖбһҖбҹ’бһ“бһ»бһ„бһҒбһҺбҹҲбһҠбҹӮбһӣбһҖбһ¶бһҡбһҷбҹ„бһ„бһ“бҹ…бһҸбҹӮбһҗбҹҒбһҡ

   > бһ…бҹҶбһҺбһ¶бҹҶ `const` бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҖбһ¶бһҡбһҷбҹ„бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһ¶бһҡбһ–бһ¶бһҡ бһ–бһёбһҖбһ¶бһҡбһӣбҹғбһҸбһҳбҹ’бһҡбһјбһңбһЎбһҫбһ„бһңбһ·бһүбҹ” бһҸбҹӮбһҸбһҳбҹ’бһӣбҹғбһҒбҹ’бһӣбһҪбһ“бһңбһ¶бһҳбһ·бһ“бһҳбҹӮбһ“ _бһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҠбһјбһҡбһ”бһ¶бһ“_ бһ‘бҹҒ бһ бһҫбһҷбһҳбһ¶бһ“бһұбһҖбһ¶бһҹбһ”бҹ’бһҠбһјбһҡ бһ”бһ¶бһ“ бһҮбһ¶бһ–бһ·бһҹбҹҒбһҹбһ”бһҫбһңбһ¶бһҮбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҠбһјбһ…бһҮбһ¶ бһңбһҸбҹ’бһҗбһ»бҹ”

## бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

бһӣбҹҒбһҒбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһҪбһҷбһ„бһ¶бһҷбһ”бҹҶбһ•бһ»бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ JavaScriptбҹ” бһҳбһ·бһ“бһҗбһ¶бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҖбһ¶бһ“бҹӢбһҸбҹӮбһ“бҹ…бһҸбҹӮбһӣбҹҒбһҒбһӮбһҸбҹӢбһҠбһјбһ…бһҮбһ¶ 42, бһӣбҹҒбһҒбһ‘бһҹбһ—бһ¶бһӮбһҠбһјбһ…бһҮбһ¶ 3.14 бһ¬бһӣбҹҒбһҒбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“бһҠбһјбһ…бһҮбһ¶ -5, JavaScript бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһңбһ¶бһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бҹ”

бһ…бһ„бһ…бһ¶бҹҶбһўбһҗбҹҒбһҡбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһ–бһёбһҳбһ»бһ“бһ‘бҹҒ? бһӣбҹҒбһҒ 123 бһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһӮбһәбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӣбҹҒбһҒ:


- бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһӮбһҺбһ·бһҸбһӣбһҫбһўбһҗбҹҒбһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ
- бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҖбһ¶бһҹбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһҮбһ¶бһҡбһҝбһҷбҹ—

### бһҹбһҡбһҹбҹҒбһҡ

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ JavaScript бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҸбҹ’бһҗбһ”бһ‘бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһҮбһ¶бһҹбһҡбһҹбҹҒбһҡбҹ” бһ–бһ¶бһҖбҹ’бһҷ "string" бһҳбһҖбһ–бһёбһӮбҹҶбһ“бһ·бһҸбһўбһҖбҹ’бһҹбһҡбһҸбҹ„бһ“бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһӮбҹ’бһ“бһ¶ бһҠбһјбһ…бһҮбһ¶бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһ”бһ»бһҡбһ¶бһҺбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһўбһҖбҹ’бһҹбһҡбһ…бҹ’бһҡбһҫбһ“бһӮбҹ’бһ“бһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһ¶бһҖбҹ’бһҷ бһ“бһ·бһ„бһ”бҹ’бһҡбһҷбҹ„бһӮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹҖбһңбһ—бҹ…бһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбҹ”

бһҹбһҡбһҹбҹҒбһҡбһҮбһ¶бһӮбһ“бҹ’бһӣбһ„бһҳбһҪбһҷбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһҺбҹ’бһҸбһ¶бһүбҹ” бһҡбһ¶бһӣбҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһӣбһҫбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ вҖ“ бһҲбҹ’бһҳбҹ„бҹҮбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ, бһҹбҹ’бһӣбһ¶бһҖбһ”бҹҠбһјбһҸбһ»бһ„, бһҹбһ¶бһҡбһҖбҹҶбһ бһ»бһҹ, бһҳбһ¶бһҸбһ·бһҖбһ¶ вҖ“ бһӮбһәбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҮбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҡбһҹбҹҒбһҡбҹ” бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһё string бһӮбһәбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ”

бһҹбһҡбһҹбҹҒбһҡбһҮбһ¶бһҹбҹҶбһҺбһ»бҹҶбһҸбһҪбһўбһҖбҹ’бһҹбһҡбһҠбҹӮбһӣбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһ…бһ“бҹ’бһӣбҹ„бҹҮбһҹбһүбҹ’бһүбһ¶бһҠбһҖбһҹбҹ’бһҡбһ„бҹӢбһҳбҹ’бһҠбһ„бһ¬бһ–бһёбһҡбһҠбһ„бҹ”

```javascript
'This is a string'
"This is also a string"
let myString = 'This is a string value stored in a variable';
```

**бһҷбһӣбҹӢбһ–бһёбһӮбҹҶбһ“бһ·бһҸбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ–**
- **бһ”бҹ’бһҡбһҫ** бһ¬бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһүбҹ’бһүбһ¶бһҠбһҖбһҹбҹ’бһҡбһ„бҹӢ `'` бһ¬бһҹбһүбҹ’бһүбһ¶бһҠбһҖбһҹбҹ’бһҡбһ„бҹӢбһ–бһёбһҡ `"` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢ string
- **бһҡбһҖбҹ’бһҹбһ¶** бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһўбһ¶бһ…бһҳбһ¶бһ“бһўбһҖбҹ’бһҹбһҡ бһӣбҹҒбһҒ бһ“бһ·бһ„бһ“бһ·бһҳбһ·бһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶
- **бһ•бҹ’бһҠбһӣбҹӢ** бһҸбһҳбҹ’бһӣбҹғ string бһ‘бҹ…бһўбһҗбҹҒбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҷ
- **бһ‘бһ¶бһҳбһ‘бһ¶бһҡ** бһўбһҖбҹ’бһҹбһ¶бһҹбһүбҹ’бһүбһ¶бһҠбһҖбһҹбҹ’бһҡбһ„бҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҹбһӣбҹӢбһ“бһјбһңбһўбһҸбҹ’бһҗбһ”бһ‘бһ–бһёбһҲбҹ’бһҳбҹ„бҹҮбһўбһҗбҹҒбһҡ

### бһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбҹҖбһ”бһҹбһҡбһҹбҹҒбһҡ

бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ string бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ’бһ¶бһҸбһ»бһўбһҸбҹ’бһҗбһ”бһ‘бһ•бҹ’бһҹбҹҒбһ„бҹ— бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһўбһҗбҹҒбһҡ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹ„бһҷбһҘбһҸбһҒбҹ’бһҮбһ·бһӣ бһҠбҹӮбһӣбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ“бһ№бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ” бһңбһ·бһ’бһёбһ“бҹҒбҹҮбһўбһ¶бһ…бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ”

бһ–бҹҒбһӣбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ—бҹ’бһҮбһ¶бһ”бҹӢ string бһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶ вҖ“ бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹҒбҹҮбһ бҹ…бһҗбһ¶ concatenationбҹ”

бһҠбһҫбһҳбҹ’бһ”бһё **бһ—бҹ’бһҮбһ¶бһ”бҹӢ** string бһ–бһёбһ–бһёбһҡбһ¬бһ…бҹ’бһҡбһҫбһ“ бһ¬бһ—бҹ’бһҮбһ¶бһ”бҹӢбһңбһ¶бһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶ бһ”бҹ’бһҡбһҫбһўбһ»бһёбһ•бһ¶бһҡ `+`бҹ”

```javascript
let myString1 = "Hello";
let myString2 = "World";

myString1 + myString2 + "!"; //бһҹбһҪбһҹбҹ’бһҠбһёвҖӢбһ–бһ·бһ—бһ–бһӣбҹ„бһҖ!
myString1 + " " + myString2 + "!"; //бһҹбһҪбһҹбҹ’бһҠбһё бһ–бһ·бһ—бһ–бһӣбҹ„бһҖ!
myString1 + ", " + myString2 + "!"; //бһҹбһҪбһҹбҹ’бһҠбһё, бһ–бһ·бһ—бһ–бһӣбҹ„бһҖ!
```

**бһҮбҹҶбһ бҹҠбһ¶бһ“бһҠбҹ„бһҷбһҮбҹҶбһ бҹҠбһ¶бһ“ бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҖбһҫбһҸбһЎбһҫбһ„бһҳбһ¶бһ“бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–**
- **бһ”бһүбҹ’бһ…бһјбһӣ** бһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҗбҹҒбһҡ `+`
- **бһ—бҹ’бһҮбһ¶бһ”бҹӢ** бһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ…бһ“бҹ’бһӣбҹ„бҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбҹҶбһ”бһјбһ„
- **бһ”бһ“бҹ’бһҗбҹӮбһҳ** бһҸбһҪбһўбһҖбҹ’бһҹбһҡбһ…бһ“бҹ’бһӣбҹ„бҹҮ `" "` бһ“бҹ…бһ…бһ“бҹ’бһӣбҹ„бҹҮбһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһўбһ¶бһ“бһ„бһ¶бһҷ
- **бһ”бһүбҹ’бһ…бһјбһӣ** бһўбһҖбҹ’бһҹбһҡбһ–бһ·бһҹбҹҒбһҹбһҠбһјбһ…бһҮбһ¶вҖӢбһҹбһүбҹ’бһүбһ¶бһҖбҹ’бһ”бҹҖбһҹвҖӢ(comma) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң

вң… бһҳбһјбһӣбһ бҹҒбһҸбһ»бһҳбһ·бһ“бһӣбҹ’бһҳбһҳбһҗбһ¶ `1 + 1 = 2` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ JavaScript бһҳбһ·бһ“бһҠбһјбһ… `'1' + '1' = 11`? бһӮбһ·бһҸбһ–бһёбһңбһ¶бҹ” бһҸбһҫбһҳбҹүбҹҒбһ…бһ‘бҹ… `'1' + 1`?

**бһўбһҖбҹ’бһҹбһҡбһ‘бһҳбҹ’бһӣбһ¶бһ”бҹӢ** бһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡ бһӣбһ»бҹҮбһҸбҹ’бһҡбһ¶бһҸбҹӮбһ•бҹ’бһ‘бҹҒбһҡбһҖбһјбһҠбһҳбһ·бһ“бһ”бҹ’бһҡбһҫбһҹбһүбҹ’бһүбһ¶бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ‘бҹҒ бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһҹбһүбҹ’бһүбһ¶бһӮбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹ’бһҡбһ…бҹҖбһҖ backtick бҹ” бһўбҹ’бһңбһёбһҖбҹҸбһҠбҹ„бһҷбһҠбҹӮбһӣбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһўбһҖбҹ’бһҹбһҡбһ”бҹ’бһҡбһҖбһ”бһҖбһ¶бһҡ бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҠбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ `${ }` бҹ” бһңбһ¶бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһўбһҗбҹҒбһҡбһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһҮбһ¶бһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡбһ”бһ¶бһ“бҹ”

- **бһ•бҹ’бһҸбһӣбҹӢ** бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҠбҹҸбһҹбҹ’бһўбһ¶бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡбһӣбҹҶбһ”бһ¶бһҖбһҮбһ¶бһҳбһҪбһҷбһўбһҗбҹҒбһҡбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ

### рҹ”Ө **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҮбҹҶбһ“бһ¶бһүбһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡбҹ– бһ—бһ¶бһ–бһ‘бҹҶбһ“бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ…бҹҶбһ–бҹ„бҹҮбһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһўбһҸбҹ’бһҗбһ”бһ‘**

бһҸбһҳбҹ’бһӣбҹғбһ”бҹҠбһјбһӣбһўбһ¶бһ…бһҳбһ¶бһ“бһҸбҹӮбһ–бһёбһҡбһҸбһҳбҹ’бһӣбҹғбһӮбһҸбҹӢбһӮбһә `true` бһ¬ `false` бҹ” бһңбһ¶бһўбһ¶бһ…бһҮбһҪбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһҳбҹ’бһҡбҹҒбһ…бһҗбһ¶бһҸбҹ’бһҡбһ„бҹӢбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡбһҺбһ¶бһҳбһҪбһҷбһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҫбһҖбһ”бһҡбҹҗбһҺбһ–бҹҒбһӣбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ”бһ¶бһ“бһ”бҹҶбһ–бҹҒбһүбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“, [operators](#бһўбһ»бһёбһ•бһ¶бһҡвҖӢбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶) бһҮбһҪбһҷбһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһ”бҹҠбһјбһӣ бһ бһҫбһҷбһўбҹ’бһ“бһҖбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ“бһ№бһ„бһ”бһ¶бһ“бһ‘бһ‘бһҪбһӣ бһ“бһ·бһ„бһҹбһҡбһҹбҹҒбһҡбһўбһҗбҹҒбһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһӮбһәбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһ¬бһҸбһҳбҹ’бһӣбҹғбһҡбһ”бһҹбҹӢбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҪбһ“бһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷ operatorбҹ”

```javascript

let myFalseBool = false;
```

**бһ“бҹ…бһӣбһҫбһ“бҹҒбҹҮбһҷбһҫбһ„бһ”бһ¶бһ“бҹ–**
- **бһ”бһ„бҹ’бһҖбһҫбһҸ** бһўбһҗбҹҒбһҡбһҳбһҪбһҷбһҠбҹӮбһӣбһ•бҹ’бһ‘бһ»бһҖбһҸбһҳбҹ’бһӣбҹғбһ”бҹҠбһјбһӣ `true`
- **бһ”бһ„бҹ’бһ бһ¶бһү** бһҡбһ”бҹҖбһ”бһ•бҹ’бһ‘бһ»бһҖбһҸбһҳбҹ’бһӣбҹғбһ”бҹҠбһјбһӣ `false`
- **бһ”бҹ’бһҡбһҫ** бһ–бһ¶бһҖбҹ’бһҷбһӮбһ“бҹ’бһӣбһ№бҹҮбһҸбһјбһ…бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң `true` бһ“бһ·бһ„ `false` (бһҳбһ·бһ“бһҳбһ¶бһ“бһҹбһүбҹ’бһүбһ¶бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ‘бҹҒ)
- **бһҡбҹҖбһ”бһ…бҹҶ** бһўбһҗбҹҒбһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҡбһҮбһ¶бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢ

## бһ”бҹ’бһҡбһҷбһ»бһ‘бҹ’бһ’ GitHub Copilot Agent рҹҡҖ

бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠ Agent бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһ”бһүбҹ’бһ бһ¶бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

**бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бҹ–** бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ JavaScript бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһ–бҹҒбһӣбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ”

**бһ…бҹҶбһҺбһ¶бҹҶбҹ–** бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё JavaScript бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһңбһҸбҹ’бһҗбһ»бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҡбһјбһ”бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҳбһҪбһҷ бһҡбһҪбһҳбһҳбһ¶бһ“бҹ– бһҲбҹ’бһҳбҹ„бҹҮбһҳбһ“бһ»бһҹбҹ’бһҹ (бһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡ), бһўбһ¶бһҷбһ» (бһӣбҹҒбһҒ), бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҮбһ¶бһҹбһ·бһҹбҹ’бһҹ (бһ”бҹҠбһјбһӣ), бһ–бһҺбҹҢбһҠбҹӮбһӣбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҮбһ¶бһўбһ¶бһҡбҹүбҹҒ, бһ“бһ·бһ„бһўбһ¶бһҹбһҷбһҠбҹ’бһӢбһ¶бһ“бһҮбһ¶бһңбһҸбҹ’бһҗбһ»бһҳбһ¶бһ“бһ•бҹ’бһӣбһјбһң бһ‘бһёбһҖбҹ’бһҡбһ»бһ„ бһ“бһ·бһ„бһӣбҹҒбһҒбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбҹ” бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҳбһ»бһҒбһ„бһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҡбһјбһ” бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһңбһ¶бһӣбһҜбһҖбһ‘бҹҒбһҹ бһҒбһҺ:бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡ, бһўбһҖбҹ’бһҹбһҡбһ‘бһҳбҹ’бһӣбһ¶бһ”бҹӢ, бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ№бһҳбһҮбһ¶бһҳбһҪбһҷбһўбһ¶бһҷбһ» бһ“бһ·бһ„бһҸбһ»бһӣбҹ’бһҷбһҖбһҳбҹ’бһҳбһ”бҹҠбһјбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҮбһ¶бһҹбһ·бһҹбҹ’бһҹбҹ”

бһҹбһјбһҳбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһё [бһҳбҹүбһјбһҠ agent](https://code.visualstudio.com/blogs/2025/02/24/introducing-copilot-agent-mode) бһ‘бһёбһ“бҹҒбҹҮбҹ”

## рҹҡҖ бһ”бҹ’бһҡбһҷбһ»бһ‘бҹ’бһ’

JavaScript бһҳбһ¶бһ“бһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһҒбҹ’бһӣбҹҮбҹ—бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһҺбҹ’бһҸбһ¶бһӣбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбһ¶бһ“бһҖбһ¶бһҡбһ—бһ¶бһ“бҹӢбһ…бҹ’бһҡбһЎбҹҶбҹ” бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ”бһ»бһҡбһ¶бһҺбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбҹ– бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһңбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бһ„бһҹбһ»бһӣбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ– `let age = 1; let Age = 2; age == Age` бһ бһҫбһҷбһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбҹ” бһңбһ¶бһ”бһ„бҹ’бһ бһ¶бһү `false` вҖ“ бһҸбһҫбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“?

бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ“бҹғ JavaScript бһҠбҹӮбһӣбһӮбһҪбһҡбһҸбҹӮбһҷбһӣбҹӢбһҠбһ№бһ„бҹ” бһҖбһ¶бһҡбһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбҹ’бһӮбһ¶бһӣбҹӢбһӣбҹ’бһўбһ“бһ№бһ„бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһӣбҹӮбһҖбҹ—бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһ”бһ„бҹ’бһҖбһҫбһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһ”бһ¶бһ“бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ…бҹҶбһ–бҹ„бҹҮбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһ“бһ·бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бҹ”

## бһҹбҹҶбһҺбһҪбһҡбһ”бҹ’бһҡбһЎбһ„бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“

бһҳбһҫбһӣбһҸбһ¶бһҡбһ¶бһ„ [бһўбҹ’бһ“бһҖбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ JavaScript бһ“бҹҒбҹҮ](https://css-tricks.com/snippets/javascript/) бһ бһҫбһҷбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбһҪбһҷбҹ” бһҸбһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ?

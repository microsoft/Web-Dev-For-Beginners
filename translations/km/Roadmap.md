# ផែនទីមុខងារ​សម្រាប់​ឃ្លាំង Web-Dev-For-Beginners របស់ Microsoft

**ឃ្លាំងនេះផ្តល់ជូនផែនទីមុខងារសម្រាប់ការស្វែងយល់មូលដ្ឋានអំពីការអភិវឌ្ឍបណ្ដាញដោយផ្តោតលើ JavaScript, HTML, និង CSS។ មេរៀននេះមានភាពបត់បែន និងអាចយករៀនពេញមួយឬផ្នែកមួយ បានដោយមានមេរៀន ២៤ មេរៀនផ្ទាត់គ្នានៅក្នុងរយៈពេល ១២ សប្តាហ៍។**

## ដំណាក់កាលសំខាន់ៗ

* **សប្តាហ៍ ១-៣:**
   * ការណែនាំអំពីភាសាកម្មវិធីនិងឧបករណ៍ការងារ
   * មូលដ្ឋាន GitHub
   * ការចូលរួមរបស់មនុស្សមានពិការភាព
   * មូលដ្ឋាន JS៖ ប្រភេទទិន្នន័យ, មុខងារ និងវិធីសាស្រ្ត
   * ការធ្វើការសម្រេចចិត្តជាមួយ JS
* **សប្តាហ៍ ៤-៦:**
   * អារេ និង វដ្ត
   * Terrarium៖ HTML ក្នុងការប្រើប្រាស់
   * CSS ក្នុងការប្រើប្រាស់
   * បិទ JavaScript
   * ការបញ្ចូល DOM
* **សប្តាហ៍ ៧-៩:**
   * ល្បែងវាយអក្សរ៖ កម្មវិធីដែលគ្រប់គ្រងដោយព្រឹត្តិការណ៍
   * ការពង្រីកកម្មវិធី Green Browser៖ បំពេញការងារជាមួយកម្មវិធីរុករក
   * ការបង្កើតសំណុំបែបបទ, ហៅ API និងផ្ទុកអថេរនៅក្នុងឃ្លាំងតំបន់
   * ដំណើរការផ្ទៃខាងក្រោយនៅក្នុងកម្មវិធីរុករក
   * សមត្ថភាពបណ្ដាញ
* **សប្តាហ៍ ១០-១២:**
   * ល្បែងអាកាស៖ ការអភិវឌ្ឍល្បែងកម្រិតខ្ពស់ជាមួយ JavaScript
   * គូរនៅលើកង់វ៉ាស
   * ផ្លាស់ទីធាតុជុំវិញអេក្រង់
   * ការរកឃើញការប៉ះទង្គិច
   * ការធ្វើរកពិន្ទុ, បញ្ចប់ និងចាប់ផ្តើមល្បែងម្តងទៀត
   * កម្មវិធីធនាគារ៖ គំរូ HTML និងផ្លូវក្នុងកម្មវិធីវែប
   * បង្កើតសំណុំបែបបទចូលប្រើ និងចុះឈ្មោះ
   * វិធីសាស្រ្តនៃការទាញយក និងប្រើទិន្នន័យ
   * យល់ដឹងអំពីការគ្រប់គ្រងស្ថានភាព

## លទ្ធផលការរៀន

**ដោយបញ្ចប់ផែនទីនេះ និស្សិតនឹងទទួលបានបទពិសោធន៍ដល់ដៃក្នុងការបង្កើតល្បែងវាយអក្សរ, terrarium វិច្ចិច, ការពង្រីកកម្មវិធីរុករក បរិស្ថានបៃតង, ល្បែងអាកាសស្ទីល space-invader, និងកម្មវិធីធនាគារសម្រាប់អាជីវកម្ម។ ពួកគេចំណេះដឹងយ៉ាងល្អពីមូលដ្ឋានការអភិវឌ្ឍបណ្ដាញផងដែរ។**

## វត្ថុធាតុជំនួយបន្ថែម

* ឃ្លាំងនេះផ្តល់ជូនធនធានជាច្រើនសម្រាប់ការស្វែងយល់បន្ថែម រួមទាំងមេរៀន, ឧទាហរណ៍កូដ និងការប្រកួតប្រជែង។
* វេទិកា Microsoft Learn ផ្តល់ជូនវគ្គសិក្សា និងផ្លូវការសិក្សាក្នុងការអភិវឌ្ឍបណ្ដាញជាច្រើន។
* សហគមន៍អនឡាញដូចជា Stack Overflow និង MDN Web Docs ផ្តល់ជួយគាំទ្រ និងធនធានដល់អ្នកអភិវឌ្ឍន៍បណ្ដាញ។

**ខ្ញុំសង្ឃឹមថាផែនទីនេះនឹងជួយអ្នកក្នុងដំណើរការអភិវឌ្ឍបណ្ដាញរបស់អ្នក!**

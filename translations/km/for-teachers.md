### бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһјбһ”бһ„бҹ’бһҡбҹҖбһ“

бһўбҹ’бһ“бһҖбһҹбһјбһҳбһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбҹҖбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“бһҮбһ¶бһҳбһҪбһҷ GitHub Classroom бһ“бһ·бһ„бһңбҹҒбһ‘бһ·бһҖбһ¶ LMS бһҖбҹҶбһ–бһјбһӣбһ•бҹ’бһҹбҹҒбһ„бҹ— бһ бһҫбһҷбһңбһ¶бһҖбҹҸбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҮбһ¶бһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҜбһҖбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҳбһҪбһҷбһҹбһ·бһҹбҹ’бһҹбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ•бһ„бһҠбҹӮбһҡбҹ”

### бһ”бҹ’бһҡбһҫбһҮбһ¶бһҳбһҪбһҷ GitHub Classroom

бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҳбҹҒбһҡбҹҖбһ“ бһ“бһ·бһ„бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҖбҹ’бһҡбһ»бһҳ бһҹбһјбһҳбһ”бһ„бҹ’бһҖбһҫбһҸ repository бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһҳбһҪбһҷ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ GitHub Classroom бһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ“бһёбһҳбһҪбһҷбҹ—бһҠбҹ„бһҷбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбһ”бһ¶бһ“бҹ”

- Fork repo бһ“бҹҒбҹҮбһ‘бҹ…бһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
- бһ”бһ„бҹ’бһҖбһҫбһҸ repo бһ•бҹ’бһҹбҹҒбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бһёбһҳбһҪбһҷбҹ— бһҠбҹ„бһҷбһҠбһҖFolder бһҳбҹҒбһҡбҹҖбһ“бһ“бһёбһҳбһҪбһҷбҹ—бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ repository бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”
  - бһҮбһҳбҹ’бһҡбһҫбһҹ A: бһ”бһ„бҹ’бһҖбһҫбһҸ repo бһ‘бһ‘бҹҒ(бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһҳбһҪбһҷ) бһ бһҫбһҷбһ…бһҳбҹ’бһӣбһ„бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһ“бҹғ Folder бһҳбҹҒбһҡбҹҖбһ“бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„ repo бһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбҹ”
  - бһҮбһҳбҹ’бһҡбһҫбһҹ B: бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҗбһҸбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· Git бһ бһҫбһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· (бһ§. бһ”бҹҶбһ”бҹӮбһҖ Folder бһҳбһҪбһҷбһ‘бҹ… repo бһҗбҹ’бһҳбһё) бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”
- бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ GitHub Classroom бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бһёбһҳбһҪбһҷбҹ— бһ бһҫбһҷбһҠбһ¶бһҖбҹӢбһ‘бһ·бһҹбһҸбһ¶бһҳ repo бһҳбҹҒбһҡбҹҖбһ“бһ“бҹ„бҹҮбҹ”
- бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶбҹ–
  - бһҳбһҫбһӣбһғбһҫбһү Repository: бһҜбһҖбһҮбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһҹбһ·бһҹбҹ’бһҹбҹ”
  - бһ”бҹ’бһҡбһҫбһҖбһјбһҠбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҹбһ¶бһҒбһ¶бһ”бһҺбҹ’бһҸбһ¶бһүбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһ“бҹғ repo бһҳбҹҒбһҡбҹҖбһ“бҹ”
  - бһ”бһ“бҹ’бһҗбҹӮбһҳбһӮбҹҶбһҡбһјбһ”бһүбҹ’бһ бһ¶ бһ“бһ·бһ„ pull request бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ·бһүбҹ’бһүбһ¶бһҹбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҹбҹ’бһ“бһҫбҹ”
  - бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһ“ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢ autograding бһ“бһ·бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбҹ”
- бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹӮбһӣбһҮбһҪбһҷбһҠбһјбһ…бһҮбһ¶бҹ–
  - бһҲбҹ’бһҳбҹ„бҹҮ repo бһҠбһјбһ…бһҮбһ¶ lesson-01-intro, lesson-02-html бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”
  - бһҹбҹ’бһӣбһ¶бһ”: quiz, assignment, needs-review, late, resubmissionбҹ”
  - Tag/releases бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҖбҹ’бһҡбһ»бһҳ (бһ§. v2025-term1)бҹ”

 бһҷбҹ„бһ”бһӣбҹӢбҹ– бһҮбҹҖбһҹбһңбһ¶бһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ repository бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Folder бһҠбҹӮбһӣбһҹбһҳбһҖбһ¶бһӣбһҖбһҳбҹ’бһҳ (бһ§. OneDrive/Google Drive) бһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ”бҹүбҹҮбһ‘бһ„бҹ’бһӮбһ·бһ… Git бһ“бҹ…бһӣбһҫ Windowsбҹ”

### бһ”бҹ’бһҡбһҫбһҮбһ¶бһҳбһҪбһҷ Moodle, Canvas, бһ¬ Blackboard

бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮбһҳбһ¶бһ“бһҖбһүбҹ’бһ…бһ”бҹӢбһ•бҹ’бһ‘бһ»бһҖбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹ’бһҡбһ”бһҒбҹҗбһҺбҹ’бһҢ LMS бһ–бҹҒбһүбһ“бһ·бһҷбһҳбҹ”

- Moodleбҹ– бһ”бҹ’бһҡбһҫбһҜбһҖбһҹбһ¶бһҡ upload Moodle [Moodle upload file](../../../../../../../teaching-files/webdev-moodle.mbz) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһјбһӣбһңбһӮбҹ’бһӮбһ–бҹҒбһүбһӣбҹҒбһүбҹ”
- Common Cartridgeбҹ– бһ”бҹ’бһҡбһҫбһҜбһҖбһҹбһ¶бһҡ Common Cartridge [Common Cartridge file](../../../../../../../teaching-files/webdev-common-cartridge.imscc) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһҸбһҳбҹ’бһҡбһ»бһҷбһҠбҹӮбһӣбһҹбһ¶бһҖбһҹбһҳбһҮбһ¶бһ„бһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ LMS бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ”
- бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ¶бҹҶбҹ–
  - Moodle Cloud бһҳбһ¶бһ“бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҡбһ”бһҹбҹӢ Common Cartridge бһҳбһ·бһ“бһ…бҹ’бһҡбһҫбһ“бҹ” бһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҜбһҖбһҹбһ¶бһҡ Moodle бһҒбһ¶бһ„бһӣбһҫ бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ… Canvas бһ•бһ„бһҠбҹӮбһҡбҹ”
  - бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ“бһ¶бҹҶбһ…бһјбһӣ бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбҹүбһјбһҢбһ»бһӣ бһҖбһ¶бһӣбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘ бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ quiz бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһҹбһҳбһҹбҹ’бһҡбһ”бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһӣбһңбһ·бһ—бһ¶бһӮбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

### бһ”бҹ’бһҡбһҫ repo бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ (бһӮбҹ’бһҳбһ¶бһ“ Classroom)

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ„бҹӢбһ”бҹ’бһҡбһҫ GitHub Classroom бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбһ”бһ‘бһ–бһ¶бһҺ repo бһ“бҹҒбҹҮбһ”бһ¶бһ“бҹ”

- бһҡбһјбһ”бһҳбһ“бҹ’бһҸбһңбҹҒбһ”бһҹбһ¶бһҷ/бһ“бҹ…бһҸбһ¶бһҳбһўбһ“бһЎбһ¶бһү (Zoom/Teams)бҹ–
  - бһ”бһҫбһҖбһңбһӮбҹ’бһӮбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҒбҹ’бһӣбһёбҹ—бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһҺбҹӮбһ“бһ¶бҹҶ; бһ”бҹ’бһҡбһҫбһ”бһ“бҹ’бһ‘бһ”бҹӢбһ”бҹҶбһ”бҹӮбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ·бһүбҹ’бһүбһ¶бһҹбһ¶бҹ”
  - бһ”бҹ’бһҡбһҖбһ¶бһҹбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ·бһүбҹ’бһүбһ¶бһҹбһ¶; бһҹбһ·бһҹбҹ’бһҹбһҠбһ¶бһҖбҹӢбһ…бһҳбҹ’бһӣбһҫбһҷбһҮбһ¶ GitHub Issuesбҹ”
  - бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶ бһҹбһ·бһҹбҹ’бһҹбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ“бҹ… repo бһҳбҹҒбһҡбҹҖбһ“бһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲ бһ бһҫбһҷбһ”бһҫбһҖ pull requestsбҹ”

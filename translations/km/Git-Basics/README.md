## бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғ GIT бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ’бҹ’бһңбһҫбһ”бһҺбҹ’бһҸбһ¶бһүрҹ‘¶


## Git бһҮбһ¶бһўбҹ’бһңбһё?
      1. Git бһӮбһәбһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҺбҹ’бһҸбһ»бҹҶбһҮбһ¶бһҸбһ·бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһјбһҠбҹ”
      2. бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҖбһјбһҠбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ бһ“бһ·бһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўwickdeveloper бһҡбһ¶бһӣбҹӢбһӮбҹ’бһ“бһ¶ 
         бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҸбҹҶбһ”бһ“бҹӢбһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ”
      3. бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбҹӮ (VCS) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ”
      
* бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбҹӮбһ…бҹӮбһҖбһ…бһ¶бһҷ
* бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһҹбһ бһҖбһ¶бһҡбһ–бһёбһўбҹ’бһ“бһҖбһўwickdeveloper бһ…бҹ’бһҡбһҫбһ“
* бһҸбһҫбһ“бһҡбһҺбһ¶бһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһўбҹ’бһңбһё бһ“бһ·бһ„бһ–бҹҒбһӣбһҺбһ¶
* бһҖбһ¶бһҡбһ”бһ„бҹ’бһңбһ·бһӣбһҸбҹ’бһҡбһЎбһ”бҹӢбһҖбҹ’бһҡбҹ„бһҷбһ“бҹ…бһ–бҹҒбһӣбһҺбһ¶бһҳбһҪбһҷ
* бһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҶбҹ’бһ„бһ¶бһҷ

## бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ“бҹғ GIT
* бһҸбһ¶бһҳбһҠбһ¶бһ“бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһјбһҠ
* бһҷбһҖбһҡбһјбһ”бһҗбһҸ "snapshots" бһ“бҹғбһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
* бһўбҹ’бһ“бһҖбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ–бҹҒбһӣбһҷбһҖвҖӢбһҡбһјбһ”бһҗбһҸ бһҠбҹ„бһҷбһ’бҹ’бһңбһҫ "commit"
* бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҡбһјбһ”бһҗбһҸбһҺбһ¶бһҳбһҪбһҷбһ“бҹ…бһ–бҹҒбһӣбһҺбһ¶бһҳбһҪбһҷ
* бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбҹҖбһ”бһ…бҹҶбһҜбһҖбһҹбһ¶бһҡбһҳбһ»бһ“бһ–бҹҒбһӣбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ commit

### бһ‘бһ‘бһҪбһӣбһҷбһҖ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„

| бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ | бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶ |
| ------- | ----------- |
| `git init` | бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһғбҹ’бһӣбһ¶бҹҶбһ„ Git бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ |
| `git clone ssh://git@github.com/[username]/[repository-name].git` | бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹ’бһ”бһ¶бһ”бҹӢ lokalen бһ–бһёбһғбҹ’бһӣбһ¶бҹҶбһ„бһҶбҹ’бһ„бһ¶бһҷ |

### бһҡбһјбһ”бһҗбһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

| бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ | бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶ |
| ------- | ----------- |
| `git status` | бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– |
| `git add [file-name.txt]` | бһ”бһүбҹ’бһ…бһјбһӣбһҜбһҖбһҹбһ¶бһҡбһ…бһјбһӣбһҸбҹҶбһ”бһ“бҹӢбһҡбҹҖбһ”бһ…бҹҶ staging |
| `git add -A` | бһ”бһ“бҹ’бһҗбҹӮбһҳбһҜбһҖбһҹбһ¶бһҡбһҗбҹ’бһҳбһё бһ“бһ·бһ„бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ‘бҹ… staging |

| `git rm -r [file-name.txt]` | бһӣбһ»бһ”бһҜбһҖбһҹбһ¶бһҡ (бһ¬бһҗбһҸ) |
| `git push` | бһ”бһҫбһҖбһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһғбҹ’бһӣбһ¶бҹҶбһ„бһҶбҹ’бһ„бһ¶бһҷ |
| `git pull` | бһ‘бһ¶бһүбһҷбһҖбһ”бҹҶбһӣбҹӮбһ„бһҗбҹ’бһҳбһёбһ–бһёбһғбҹ’бһӣбһ¶бҹҶбһ„бһҶбҹ’бһ„бһ¶бһҷ |

# бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһёбһңбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ VSCode.dev

бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһўбһ“бһ¶бһӮбһҸбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһҖбһҹбһ¶бһ„бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһёбһңбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҮбҹҶбһ“бһ¶бһү бһ“бһ·бһ„бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҡбһјбһ”бһҡбһ¶бһ„бһ”бҹӮбһ”бһўбһ“бҹ’бһҸбһҡбһҮбһ¶бһҸбһ· бһ“бһ·бһ„бһ‘бҹҶбһ“бһҫбһ”бҹ” бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһ•бҹ’бһүбһҫ PDF бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһ–бҹғбһҺбһё бһҹбһјбһҳбһ“бһ№бһҖбһҹбҹ’бһҡбһҳбҹғбһ–бһёбһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһўбҹ’бһ“бһҖбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ“бһ·бһҷбҹ„бһҮбһ·бһҸбһҠбҹ„бһҷбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһ”бһҗбһҹбҹ’бһўбһ¶бһҸ бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ”бһ¶бһ“бһӣбҹ’бһў бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ‘бһ¶бҹҶбһ„бһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶ бһ“бһ·бһ„бһҮбҹҶбһ“бһ¶бһүбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбҹ”

бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹӮбһ”бһ“бҹҒбҹҮ бһ“бһ№бһ„бһҠбһ¶бһҖбҹӢбһҮбҹҶбһ“бһ¶бһү VSCode.dev бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҒбһҺбҹҲбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһ·бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ¶бһӮбһҸбһҖбһ¶бһҡбһ„бһ¶бһҡбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ–бҹҒбһүбһӣбҹҒбһүбһ“бҹғбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ вҖ“ бһ…бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ repository бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ вҖ“ бһ‘бһ¶бҹҶбһ„бһЎбһ¶бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһҠбҹ„бһҷбһ”бһүбҹ’бһ…бһ”бҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ¶бһ“бһңбһҸбҹ’бһҸбһҳбһ¶бһ“бһўбһ“бһЎбһ¶бһүбһҮбҹҶбһ“бһҪбһҹбһҠбҹӮбһӣбһўбһ¶бһ…бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ”бһ¶бһ“бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҮбһ¶бһҳбһҪбһҷбһ“бһ·бһҷбҹ„бһҮбһ·бһҖбһҳбһ¶бһ“бһҹбһҖбҹ’бһҸбһ¶бһ“бһ»бһ–бһӣ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ–бҹҒбһӣбһҮбҹҶбһ“бһ¶бһүбһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“ бһ бһҫбһҷбһўбһ¶бһ…бһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһҳбҹүбһ¶бһҖбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһӮбҹҶбһҡбһјбһӮбһҳбҹ’бһҡбҹ„бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҮбҹҶбһ“бһ¶бһүбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…:

- **бһ”бһ„бҹ’бһҖбһҫбһҸ** бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ–бҹҒбһүбһӣбҹҒбһүбһ”бҹ’бһҡбһҫ VSCode.dev
- **бһҡбҹҖбһ”бһ…бҹҶ** бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ’бһ¶бһҸбһ» HTML бһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ“бҹҗбһҷ
- **бһҸбһ»бһ”бһҸбҹӮбһ„** бһҳбһ¶бһҸбһ·бһҖбһ¶бһўбҹ„бһҷбһҶбһ”бһӮбҹ’бһ“бһ¶бһ“бһ№бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһӣбһҫбһҖбһҳбҹ’бһҡбһ¶бһҹбҹӢбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ CSS бһ‘бҹҶбһ“бһҫбһ”
- **бһўбһ“бһ»бһңбһҸбҹ’бһҸ** бһҳбһ»бһҒбһ„бһ¶бһҡбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“
- **бһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бһ¶бһү** бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҡбһҹбҹӢбһҠбҹӮбһӣбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҸбһ¶бһҳ URL бһҠбҹӮбһӣбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ”бһ¶бһ“
- **бһ”бһ„бҹ’бһ бһ¶бһү** бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ version control бһҠбһ¶бһ…бҹӢбһҒбһ¶бһҸбһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ

## бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһҮбһ¶бһҳбһ»бһ“

бһҳбһ»бһ“бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһҖбһ»бҹҶбһ—бҹ’бһӣбҹҒбһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶ бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бҹ–

- бһӮбһҺбһ“бһё GitHub (бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ¶бһ“бһ“бҹ… [github.com](https://github.com/) бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ·бһ“бһҳбһ¶бһ“)
- бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“ VSCode.dev бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһҮбһ“бҹ’бһӣбһ¶бһҹбҹӢбһҲбҹ’бһҳбҹ„бҹҮ бһ“бһ·бһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“
- бһҷбһӣбҹӢбһҠбһ№бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһўбҹҶбһ–бһёбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“ HTML бһ“бһ·бһ„бһӮбҹҶбһ“бһ·бһҸбһҸбһ»бһ”бһҸбҹӮбһ„ CSS

## бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһӮбһҳбҹ’бһҡбҹ„бһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ Repository

бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҠбҹҶбһҺбһҫбһҡбһ“бҹҒбҹҮбһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ–бһ·бһҸ бһҠбҹӮбһӣбһӮбһҳбҹ’бһҡбҹ„бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ repository бһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҷбҹүбһ¶бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

4. **бһ”бһ“бҹ’бһҗбҹӮбһҳ** бһ–бһҺбҹҢбһ“бһ¶ бһҒбҹ’бһӣбһёбҹ– "бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһёбһңбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷ HTML бһ“бһ·бһ„ CSS"

- **бһҳбһҫбһӣбһҳбһ»бһ“бһҳбһ¶бһ“бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ**: **бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷ** бһўбҹ’бһ“бһҖбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҸбҹҶбһҺ бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһ•бҹ’бһҹбҹҒбһ„бҹ—

### бһҮбһҳбҹ’бһҡбһҫбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бһ¶бһү

бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһңбһ·бһүбһ”бһ¶бһ“бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ“бһ·бһҷбҹ„бһҮбһҖ бһҹбһјбһҳбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһ”бһ„бҹ’бһ бһ¶бһүбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

**GitHub Pages (бһҺбҹӮбһ“бһ¶бҹҶ):**
1. бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ Settings бһҡбһ”бһҹбҹӢ repository бһ“бҹ… GitHub
2. бһҡбһҫбһҹбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢ "Pages"
3. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ "Deploy from a branch" бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹбһҹбһ¶бһҒбһ¶ "main"
4. бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҸбһ¶бһҳ `https://your-username.github.io/my-resume`

**бһңбҹҒбһ‘бһ·бһҖбһ¶бһҮбҹҶбһ“бһҪбһҹ:**
- **Netlify**: бһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҮбһ¶бһҳбһҪбһҷбһҠбҹӮбһ“бһ”бҹ’бһҸбһјбһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“
- **Vercel**: бһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһӣбһҝбһ“бһҮбһ¶бһҳбһҪбһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһ•бҹ’бһ‘бҹҮбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ‘бҹҶбһ“бһҫбһ”
- **GitHub Codespaces**: бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҳбһҫбһӣбһҳбһ»бһ“бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ

**бһўбһ—бһ·бһ”бһ¶бһ”бҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбҹ–** бһҖбҹ’бһ“бһ»бһ„бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҮбһёбһңбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһҪбһ…бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҳбһ»бһҒбһ„бһ¶бһҡбһ‘бҹҶбһ“бһҫбһ”бҹ—:

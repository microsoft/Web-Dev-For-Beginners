# ការរួមចំណែក

គម្រោងនេះស្វាគមន៍ការរួមចំណែក និងការផ្តល់អំណានយោបល់។ ការរួមចំណែកភាគច្រើនទាមទារឱ្យអ្នកយល់ព្រមនូវកិច្ចសន្យាសិទ្ធិរបស់អ្នករួមចំណែក (CLA) ដែលថ្លែងថាអ្នកមានសិទ្ធិ
និងពិតជាអនុញ្ញាតឱ្យយើងប្រើប្រាស់ការរួមចំណែករបស់អ្នក។ សម្រាប់ព័ត៌មានលម្អិតបន្ថែម សូមចូលទៅកាន់
[https://cla.microsoft.com](https://cla.microsoft.com/?WT.mc_id=academic-77807-sagibbon)។

ពេលអ្នកដាក់សំណើទាញយក (pull request) CLA-bot នឹងកំណត់ដោយស្វ័យប្រវត្តិនៅតើអ្នកត្រូវ
ផ្តល់ CLA និងតែងតាំង PR ឱ្យសមរម្យ (ឧ. ស្លាក, យោបល់)។ ត្រឹមត្រូវតែអនុវត្តតាម
សេចក្ដីណែនាំដែលប៉ុស្តិកាដោយបុត។ អ្នកត្រូវតែធ្វើបែបនេះតែម្តងគត់សម្រាប់គ្រប់រ៉េបូស៊ីតូរីដែលប្រើប្រាស់ CLA របស់យើង។ សូមព្យាយាមប្រាប់យើងថាហេតុអ្វីបានធ្វើការផ្លាស់ប្តូរនោះ ដើម្បីដែលយើងអាចយល់ច្បាស់ពីសំណើរបស់អ្នក។

គម្រោងនេះបានទទួលយក [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/?WT.mc_id=academic-77807-sagibbon)។
សម្រាប់ព័ត៌មានបន្ថែម សូមមើល [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/?WT.mc_id=academic-77807-sagibbon)
ឬទំនាក់ទំនង [opencode@microsoft.com](mailto:opencode@microsoft.com) ប្រសិនបើមានសំណួរ ឬមតិយោបល់បន្ថែម។

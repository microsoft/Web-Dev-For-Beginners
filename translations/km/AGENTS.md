- **бһӮбһҳбҹ’бһҡбҹ„бһ„бһҷбҹүбһ¶бһ„бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„**бҹ– Terrarium, Typing Game, Browser Extension, Space Game, Banking App, Code Editor, бһ“бһ·бһ„ AI Chat Assistant
- **бһҖбһ¶бһҡбһ”бҹ’бһҡбһӣбһ„бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ**бҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһӣбһ„ бҹӨбҹЁ бһ”бҹ’бһҡбһҖбһ¶бһҡ бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһ“бһёбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһҹбҹҶбһҺбһҪбһҡ бҹЈ (бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҳбһ»бһ“/бһҖбҹ’бһҡбҹ„бһҷбһҳбҹҒбһҡбҹҖбһ“)
- **бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“**бҹ– бһ”бһҖбһ”бҹ’бһҡбҹӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„ бҹҘбҹ  бһҮбһ¶бһҳбһҪбһҷ GitHub Actions
- **бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶**бҹ– HTML, CSS, JavaScript, Vue.js 3, Vite, Node.js, Express, Python (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„ AI)

### бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһјбһӣбһҡбһҪбһҳбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҳбһ¶бһҸбһ·бһҖбһ¶

3. **бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ¶бһҒбһ¶бһҗбҹ’бһҳбһё** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҳбҹ’бһҡбҹӮбһ”бһҳбҹ’бһҡбһҪбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

### бһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷ Quiz App (Azure Static Web Apps)

Quiz App бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷ Azure Static Web Appsбҹ–

```bash
cd quiz-app
npm run build      # бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸ dist/
# бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһ¶бһҡбһ”бҹҶбһ–бҹҒбһүбһҸбһ¶бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё GitHub Actions бһ“бҹ…бһ–бҹҒбһӣбһ…бһ»бһ…бһҠбһ¶бһҖбҹӢбһ‘бҹ… main
```

бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ–бһ·бһҹбҹҒбһҹ Azure Static Web Apps:
- **бһ‘бһёбһҸбһ¶бҹҶбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё**: `/quiz-app`
- **бһ‘бһёбһҸбһ¶бҹҶбһ„бһ…бҹҒбһү**: `dist`
- **бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ**: `.github/workflows/azure-static-web-apps-ashy-river-0debb7803.yml`

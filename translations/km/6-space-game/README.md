# បង្កើតហ្គេមអាកាសយាន

ហ្គេមអាកាសយានដើម្បីបង្រៀនមូលដ្ឋាន JavaScript ដែលមានកម្រិតខ្ពស់ជាងនេះ

នៅក្នុងមេរៀននេះ អ្នកនឹងរៀនពីរបៀបបង្កើតហ្គេមអាកាសយានផ្ទាល់ខ្លួន។ ប្រសិនបើអ្នកធ្លាប់លេងហ្គេម "Space Invaders" ហ្គេមនេះមានគំនិតដូចគ្នា៖ ដើម្បីបញ្ជាលេងយន្តយានអាកាស ហើយបាញ់ទៅលើម្ជុលដែលចុះមកពីលើ។ នេះជារូបរាងហ្គេមដែលបានបញ្ចប់៖

![Finished game](../../../6-space-game/images/pewpew.gif)

នៅក្នុងមេរៀនប្រាំមួយនេះ អ្នកនឹងរៀនដូចតទៅ៖

- **អន្តរាគមន៍** ជាមួយធាតុ Canvas ដើម្បីគូរពីរបីនៅលើអេក្រង់
- **យល់ដឹង** អំពីប្រព័ន្ធប្រភេទកូអរដោនេត Cartesian
- **រៀន** លំនាំ Pub-Sub ដើម្បីបង្កើតស្ថាបត្យកម្មហ្គេមដែលរឹងមាំ និងងាយស្រួលថែទាំ និងពង្រីក
- **ប្រើប្រាស់** Async/Await ដើម្បីផ្ទុកធនធានហ្គេម
- **ដោះស្រាយ** ព្រឹត្តិការណ៍ក្តារចុច

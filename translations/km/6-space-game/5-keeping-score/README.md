2. **бһҸбһҳбҹ’бһӣбһҫбһ„бһўбһҗбҹҒбһҡбһ бҹ’бһӮбҹҒбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ**бҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһ¶бһҳбһҠбһ¶бһ“бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһ“бҹ’бһ‘бһ»бһҹбһҡбһ»бһ” (бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһё 0) бһ“бһ·бһ„бһҮбһёбһңбһ·бһҸбһ“бҹ…бһҹбһӣбҹӢ (бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһё 3)бҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһңбһ¶бһ‘бҹ…бһӣбһҫбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһҖбһёбһЎбһ¶бһҖбһҡбһ”бһ¶бһ“бһҠбһ№бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһ–бһҪбһҖбһӮбҹҒбһ“бҹ…бһҮбһ¶бһ“бһ·бһ…бҹ’бһ…бҹ”

3. **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһҖбһ¶бһҡбһ”бҹүбҹҮбһ‘бһ„бҹ’бһӮбһ·бһ…**бҹ” бһ–бһ„бҹ’бһҡбһёбһҖбһҳбһ»бһҒбһ„бһ¶бһҡ `updateGameObjects()` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бҹҒбһӣбһҠбҹӮбһӣбһҹбһҸбҹ’бһҡбһјбһңбһ”бҹүбҹҮбһ‘бһ„бҹ’бһӮбһ·бһ…бһңбһёбһҡбҹҲбһҡбһјбһ”бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

### рҹ”„ **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸ**

**бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ бҹ’бһӮбҹҒбһҳ**бҹ– бһҳбһ»бһ“бһ“бһ№бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ·бһӣбҹҗбһҷ бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһўбҹ’бһ“бһҖбһҷбһӣбҹӢбһҠбһјбһ…бһҮбһ¶бҹ–

- вң… бһ”бһүбҹ’бһүбһ¶бһҹбҹ’бһҡбҹ„бһ…бһҹбҹ’бһҡбһ–бһ…бҹҶбһ–бҹ„бҹҮбһҸбһҳбҹ’бһӣбҹғбһ–бһ·бһ“бҹ’бһ‘бһ» бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҮбһёбһңбһ·бһҸ

*бһ…бһҳбҹ’бһӣбһҫбһҷбҹ– бһ…бҹҶбһ“бһҪбһ“бһҡбһ„бҹ’бһңбһ„бҹӢбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһёбһЎбһ¶бһҖбһҡбһӮбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһ…бһ·бһҸбҹ’бһҸ бһ бһҫбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ„бҹ’бһңбһ¶бһ“бҹӢбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸбһҠбҹӮбһӣбһ”бҹҶбһ–бҹҒбһүбһ…бһ·бһҸбҹ’бһҸ*

**бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ**бҹ– бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ–  
- **бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҸбһҪбһўбһҖбҹ’бһҹбһҡбһҸбһ¶бһҳбһҡбһҷбҹҲбһңбһ·бһ…бһ·бһҸбҹ’бһҡбһҹбһ·бһӣбҹ’бһ”бҹҲ**бҹ– бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһҖбҹӮбһ„бһҳбһ»бһҒ
- **бһ”бҹ’бһҡбһҸбһ·бһҖбһҳбҹ’бһҳбһ—бҹ’бһӣбһ¶бһҳбҹ—**бҹ– бһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бһ·бһҸбһ“бҹғбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҖбһёбһЎбһ¶бһҖбһҡ
- **бһ•бҹ’бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸ**бҹ– бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһҹбҹ’бһҡбһ¶бһӣбһҹбһ¶бһҳбһүбҹ’бһү
- **бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһ¶бһҡбһҳбһ¶бһҸбҹ’бһҡ**бҹ– бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ бһ“бһ·бһ„бһ–бһҺбҹҢбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷбһҖбһёбһЎбһ¶бһҖбһҡ

1. **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ·бһӣбҹҗбһҷбһ бҹ’бһӮбҹҒбһҳ бһ“бһ·бһ„бһҡбһ„бҹ’бһңбһ¶бһ“бҹӢ**бҹ” бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҖбһёбһЎбһ¶бһҖбһҡбһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ“бҹҗбһҷбҹ–

   1. **бһҖбһ¶бһҡбһ”бҹүбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһҸбҹӢбһҮбһёбһңбһ·бһҸ**бҹ” бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҠбҹӮбһӣбһңбһёбһҡбҹҲбһҡбһјбһ”бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһӮбһ¶бһ”бҹӢбһ”бҹүбҹҮбһҮбһ¶бһҳбһҪбһҷбһҹбһҸбҹ’бһҡбһјбһң бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһҸбҹӢбһҮбһёбһңбһ·бһҸбһҳбһҪбһҷбҹ”
   
      бһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ“бҹҒбҹҮбһ‘бҹ…бһҗбҹ’бһ“бһ¶бһҖбҹӢ `Hero` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

        ```javascript
        decrementLife() {
          this.life--;
          if (this.life === 0) {
            this.dead = true;
          }
        }
        ```

   2. **бһҖбһ¶бһҡбһ”бһ¶бһүбҹӢбһҹбһҸбҹ’бһҡбһјбһңбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ–бһ·бһ“бҹ’бһ‘бһ»**бҹ” бһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ”бһ¶бһүбҹӢбһҠбҹӮбһӣбһҮбҹ„бһӮбһҮбҹҗбһҷ бһҲбһ¶бһ“бһҠбһӣбҹӢбһ–бһ·бһ“бҹ’бһ‘бһ» бҹЎбҹ бҹ  бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһ”бҹ’бһҡбһҸбһ·бһҖбһҳбҹ’бһҳбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“бһ—бҹ’бһӣбһ¶бһҳбҹ—бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ¶бһүбҹӢбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

### рҹӣ пёҸ бһҹбһ„бҹ’бһҒбҹҒбһ”бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ…бһ“бһ¶бһ бҹ’бһӮбҹҒбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бҹ–
- **бһ…бһ·бһҸбҹ’бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бһҖбһёбһЎбһ¶бһҖбһҡ**бҹ– бһҖбһ¶бһҡбһҷбһӣбҹӢбһ…бҹҶбһҺбһ„бҹӢбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸ, бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ/бһңбһ·бһӣбҹҗбһҷ бһ“бһ·бһ„бһ…бҹ’бһҡбһңбһ№бһ„бһ…бһјбһӣбһҡбһҪбһҳ
- **бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһңбһ·бһҠбҹҒбһўбһј**бҹ– бһҡбһ…бһ“бһ¶бһ•бҹ’бһ‘бҹғбһ”бҹ’бһҡбһҫбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһўбһҖбҹ’бһҹбһҡ, бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ бһ“бһ·бһ„бһ”бҹ’бһӣбһ„бҹӢ
- **бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҸбһ”бһҹбҹ’бһ“бһ¶бһҖбҹӢ**бҹ– бһҡбһ”бһҷбһ“бҹ’бһҸбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бһ·бһҸбһ“бҹғбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҖбһёбһЎбһ¶бһҖбһҡ бһ“бһ·бһ„бһ бҹ’бһӮбҹҒбһҳ
- **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–**бҹ– бһҸбһ¶бһҳбһҠбһ¶бһ“ бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ бҹ’бһӮбҹҒбһҳбһІбҹ’бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
- **бһӮбһјбһҡбһўбһҖбҹ’бһҹбһҡбһӣбһҫбһҖбһ„бҹӢбһңбҹүбһ¶бһҹ**бҹ– бһ”бһ„бҹ’бһ бһ¶бһүбһўбһҖбҹ’бһҹбһҡбһҠбҹ„бһҷбһҮбҹҶбһ“бһ¶бһүбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҸбһ»бһ”бһҸбҹӮбһ„бһ“бһ·бһ„бһ‘бһёбһҸбһ¶бҹҶбһ„
- **бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ**бҹ– бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ‘бҹ…бһңбһ·бһӣбҹҗбһҷбһҳбһ¶бһ“бһ“бҹҗбһҷбһ“бҹғбһ бҹ’бһӮбҹҒбһҳ
- **бһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһ бҹ’бһӮбҹҒбһҳ**бҹ– бһҡбһ…бһ“бһ¶бһўбһ¶бһҖбһ¶бһҡбҹҲбһӣбҹҶбһ”бһ¶бһҖбһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“

**бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ–бһ·бһҸ**бҹ– бһҮбҹҶбһ“бһ¶бһүбһҡбһ…бһ“бһ¶бһ бҹ’бһӮбҹҒбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ‘бҹ…бҹ–
- **бһҡбһ…бһ“бһ¶бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹ’бһ‘бҹғбһ”бҹ’бһҡбһҫбһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ¶бһүбһ“бһ·бһ„бһ„бһ¶бһҷбһҷбһӣбҹӢ
- **бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ•бһӣбһ·бһҸбһ•бһӣ**бҹ– бһҷбһӣбҹӢбһ…бһ·бһҸбҹ’бһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһ”бҹ’бһҡбһҸбһ·бһҖбһҳбҹ’бһҳ
- **бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ·бһҖбҹ’бһҹбһ¶**бҹ– бһҖбһ¶бһҡбһҡбһҸбҹӢбһ бҹ’бһӮбҹҒбһҳбһ“бһ·бһ„бһҖбҹҶбһҡбһ»бһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶
- **бһҖбһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ**бҹ– бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ„бһ¶бһҷбһҷбһӣбҹӢбһ„бһ¶бһҷбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸ
- **бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһӣбҹҗбһҸ**бҹ– бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ
- **бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҳбҹүбһ¶бһҸбһҒбҹҒбһ‘бһёбһ„**бҹ– бһҷбһӣбҹӢбһ…бһ·бһҸбҹ’бһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҡбһ¶бһ”бһҳбҹ’бһӣбҹӮбһ„

**бһҮбҹҶбһ“бһ¶бһүбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“**бҹ– бһўбҹ’бһ“бһҖбһўбһ¶бһ…бҹ–
- **бһҡбһ…бһ“бһ¶**бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҠбҹӮбһӣбһӣбһҫбһҖбһҖбһҳбҹ’бһ–бһҹбҹӢбһ…бһ·бһҸбҹ’бһҸбһ…бһјбһӣбһҡбһҪбһҳ
- **бһўбһ“бһ»бһңбһҸбҹ’бһҸ**бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһҳбһҫбһӣбһҗбҹӮбһ‘бһ¶бҹҶбһҘбһҡбһ·бһҷбһ¶бһ”бһҗбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһ—бһ¶бһ–
- **бһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–**бһҖбһ¶бһҡбһ”бҹ’бһҡбһҲбһҳбһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ
- **бһ”бһ„бҹ’бһҖбһҫбһҸ**бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҳбһҫбһӣбһғбһҫбһүбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ»бһҳбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ•бҹ’бһҹбҹҒбһ„бҹ—
- **бһңбһ·бһ—бһ¶бһӮ**бһҘбһҡбһ·бһҷбһ¶бһ”бһҗбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бһ“бҹ’бһҸбһӣбһҫбһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһҡбһ…бһ“бһ¶

**бһӮбҹҶбһ“бһ·бһҸбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ бҹ’бһӮбҹҒбһҳбһҠбҹӮбһӣбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ**бҹ–
- **бһҖбҹҶбһӣбһ¶бҹҶбһ„бһ…бҹҶбһҺбһ„бҹӢбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҖбһёбһЎбһ¶бһҖбһҡ**бҹ– бһҷбһӣбҹӢбһ–бһёбһўбҹ’бһңбһёбһҠбҹӮбһӣбһ”бһҺбҹ’бһҸбһ¶бһӣбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶

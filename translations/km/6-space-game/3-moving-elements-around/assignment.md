# សរសេរកំណត់ចំណាំលើកូដរបស់អ្នក

## សេចក្តីណែនាំ

កូដស្អាត មានឯកសារបញ្ជាក់ល្អគឺចាំបាច់សម្រាប់ការរក្សាទុក និងចែករំលែកគម្រោងរបស់អ្នក។ នៅក្នុងការបញ្ចូននេះ អ្នកនឹងហ្វឹកហាត់ទម្លាប់ដ៏សំខាន់ៗមួយរបស់អ្នកអភិវឌ្ឍន៍ជំនាញ: សរសេរកំណត់ចំណាំច្បាស់លាស់ និងមានប្រយោជន៍ដែល​ពន្យល់ពីគោលបំណង និងមុខងាររបស់កូដរបស់អ្នក។

សូមពិនិត្យឯកសារ `app.js` បច្ចុប្បន្នរបស់អ្នកក្នុងថតហ្គេម រួចស្វែងរកវិធីសាស្រ្តសម្រាប់បញ្ចូលចំណាំ និងរៀបចំវាឱ្យល្អ។ វាស្រួលណាស់សម្រាប់កូដឲ្យចេញពីការត្រួតត្រា ហើយឥឡូវនេះជាឱកាសល្អឱ្យបន្ថែមកំណត់ចំណាំដើម្បីធានាថាអ្នកមានកូដដែលអាចអានបាន ដូច្នេះអ្នកអាចប្រើវាបានបន្តក្រោយ។

**ភារកិច្ចរបស់អ្នកមាន៖**
- **បន្ថែមកំណត់ចំណាំ** ដែលពន្យល់ពីតួនាទីនៃផ្នែកកូដធ मुख्यៗ
- **ឯកសារពិពណ៌នា** មុខងារជាមួយការពន្យល់ច្បាស់លាស់ពីគោលបំណង និងប៉ារ៉ាម៉ែត្រ
- **រៀបចំកូដ** ជាផ្នែកត្រឹមត្រូវដោយមានចំណងជើងផ្នែក
- **លុប** កូដដែលមិនបានប្រើឬកូដខ្ពស់មិនចាំបាច់
- **ប្រើឈ្មោះត្រួតត្រា** សម្រាប់អថេរ និងមុខងារ

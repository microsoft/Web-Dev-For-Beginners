бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһҮбһҳбҹ’бһҡбһҫбһҹбһҗбҹ’бһҳбһё бһҹбһјбһҳбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ "GitHub Copilot Agent Challenge рҹҡҖ" бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ–бһјбһҖбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бҹ” бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһҗбҹ’бһҳбһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ GitHub Copilot бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ Agent modeбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ”бһ¶бһ“бһ”бҹ’бһҡбһҫ Agent mode бһҳбһ»бһ“бһ“бҹҒбҹҮ бһңбһ¶бһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҖбҹ’бһҹбһҡ бһ‘бҹҒ бһҸбҹӮбһҗбҹӮбһҳбһ‘бһ¶бҹҶбһ„бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҜбһҖбһҹбһ¶бһҡ, бһҡбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ бһ“бһ·бһ„бһҠбһјбһ…бҹ’бһ“бҹ„бҹҮбһ‘бҹҖбһҸбҹ”

### рҹ“Ј бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бҹ’бһҡбһҖбһ¶бһҹ - _бһӮбһҳбҹ’бһҡбҹ„бһ„бһҗбҹ’бһҳбһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷ Generative AI_

* бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹ„бһҷбһӮбһҳбҹ’бһҡбҹ„бһ„
* бһҹбҹҶбһ“бһҪбһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢ

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҡбҹҖбһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ JavaScript, HTML бһ“бһ·бһ„ CSS бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҗбҹ’бһҳбһёбҹ—бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһңбҹҒбһ”бһҹбһ¶бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ“бҹ…бһҗбҹ’бһ„бҹғбһ“бҹҒбҹҮбҹ” бһҹбһ·бһҹбҹ’бһҹбһ“бһ№бһ„бһҳбһ¶бһ“бһұбһҖбһ¶бһҹбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҠбҹғбһӮбһјбһҠбҹ„бһҷбһҹбһ¶бһ„бһҹбһ„бҹӢбһ бҹ’бһӮбҹҒбһҳбһңбһ¶бһҷбһўбһҖбҹ’бһҹбһҡ Terrarium бһҸбһ¶бһҳбһўбҹҠбһ·бһ“бһ’бһәбһҡбһҺбҹҒбһҸ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӣбҹҶбһ“бһ¶бҹҶ Firefox бһҳбһ·бһҸбҹ’бһҸбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһ бҹ’бһӮбҹҒбһҳбһӣбҹ’бһҒбҹ…бһӣбҹ’бһҒбҹ…бһӣбһҫбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бһ“бһ¶бһӮбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбҹ” бһ”бһүбҹ’бһ…бһ”бҹӢбһҹбҹҠбҹҒбһҡбһёбһ“бҹҒбҹҮ бһҹбһ·бһҹбҹ’бһҹбһ“бһ№бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҮбҹ’бһҡбһ¶бһӣбһҮбҹ’бһҡбҹ…бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһңбҹҒбһ”бҹ”

> рҹҺ“ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҷбһҖбһҳбҹҒбһҡбҹҖбһ“бһҠбҹҶбһ”бһјбһ„бҹ—бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮбһҮбһ¶бһ•бҹ’бһӣбһјбһң [Learn Path](https://docs.microsoft.com/learn/paths/web-development-101/?WT.mc_id=academic-77807-sagibbon) бһ“бҹ…бһӣбһҫ Microsoft Learn!

бһҠбҹ„бһҷбһ’бһ¶бһ“бһ¶бһҗбһ¶бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҹбҹ’бһҡбһ”бһ‘бҹ…бһҸбһ¶бһҳбһӮбһҳбҹ’бһҡбҹ„бһ„ бһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбҹҖбһ“бһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ·бһҹбҹ’бһҹ бһ“бһ·бһ„бһҖбҹӮбһӣбһҳбҹ’бһўбһҖбһ¶бһҡбһ…бһ„бһ…бһ¶бҹҶбһӮбҹҶбһ“бһ·бһҸбҹ” бһҷбһҫбһ„бһҖбҹҸбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҳбҹҒбһҡбҹҖбһ“бһҠбҹҶбһ”бһјбһ„бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһңбһ·бһҹбҹҗбһҷ JavaScript бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһӮбҹҶбһ“бһ·бһҸбһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһңбһёбһҠбҹҒбһўбһјбһ–бһё "[Beginners Series to: JavaScript](https://channel9.msdn.com/Series/Beginners-Series-to-JavaScript/?WT.mc_id=academic-77807-sagibbon)" бһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶбһңбһёбһҠбҹҒбһўбһјбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ·бһ–бһ“бҹ’бһ’бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ”бһ¶бһ“бһҡбһҪбһҳбһ…бҹҶбһҺбҹӮбһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮбҹ”

бһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бһёбһ“бҹҒбҹҮ бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбһҪбһҷбһҳбһ»бһ“бһҗбҹ’бһ“бһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бһ¶бһҸбҹӢбһҠбһ¶бһҖбҹӢбһ”бҹҶбһҺбһ„бһҖбҹ’бһҸбһёбһ…бһ„бҹӢбһҡбҹҖбһ“бһҡбһ”бһҹбҹӢбһҹбһ·бһҹбҹ’бһҹбһҳбһҪбһҷ бһҮбһ¶бһҳбһҪбһҷбһҹбҹҶбһ“бһҪбһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ‘бһёбһ–бһёбһҡбһ–бһёбһҡбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһӮбһәбһ’бһ¶бһ“бһ¶бһҖбһ¶бһҡбһ…бһ„бһ…бһ¶бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮбһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһұбҹ’бһҷбһ”бһҸбҹӢбһ”бҹӮбһ“ бһ“бһ·бһ„бһҡбһёбһҖбһҡбһ¶бһҷ бһ бһҫбһҷбһўбһ¶бһ…бһҷбһҖбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһ¬бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ…бҹҶбһҺбҹӮбһҖбһҳбһҖбһўбһ¶бһ“бһ”бһ¶бһ“бҹ” бһӮбһҳбҹ’бһҡбҹ„бһ„вҖӢбһҸбһјбһ…бҹ— бһҸбҹӮбһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“бһҖбһ¶бһ“бҹӢбһҸбҹӮбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҸбһ¶бһҳбһҡбһҷбҹҲбһҡбһҷбҹҲбһ–бҹҒбһӣ 12 бһҹбһ”бҹ’бһҠбһ¶бһ бҹҚбҹ”

бһ“бҹ…бһ–бҹҒбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбһ·бһ“бһҺбҹӮбһ“бһ¶бҹҶбһҒбҹ’бһ“бһ¶бһҸ JavaScript frameworks бһ‘бҹҒ бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҮбҹҶбһ“бһ¶бһүбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбһ¶бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһңбҹҒбһ”бһҳбһ»бһ“бһ–бҹҒбһӣбһҷбһҖ framework бһҳбһҪбһҷбҹ” бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһӣбҹ’бһўбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮбһӮбһәбһҡбҹҖбһ“бһўбҹҶбһ–бһё Node.js бһҸбһ¶бһҳбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶбһңбһёбһҠбҹҒбһўбһјбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ– "[Beginner Series to: Node.js](https://channel9.msdn.com/Series/Beginners-Series-to-Nodejs/?WT.mc_id=academic-77807-sagibbon)"бҹ”

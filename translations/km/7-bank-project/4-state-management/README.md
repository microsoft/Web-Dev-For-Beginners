# бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бһ“бһ¶бһӮбһ¶бһҡ бһ—бһ¶бһӮ 4бҹ– бһӮбҹҶбһ“бһ·бһҸбһ“бҹғбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–

## вҡЎ бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ 5 бһ“бһ¶бһ‘бһёбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ

**бһ•бҹ’бһӣбһјбһңбһҹбһҡбһ»бһ”бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҠбҹӮбһӣбһ—бҹ’бһӣбһҖбҹӢбһ–бҹҒбһӣ**

```mermaid
flowchart LR
    A[вҡЎ 5 бһ“бһ¶бһ‘бһё] --> B[бһ”бһүбҹ’бһҮбһјбһ“бһ”бһүбҹ’бһ бһ¶вҖӢбһҡбһҠбҹ’бһӢ]
    B --> C[бһ”бһ„бҹ’бһҖбһҫбһҸвҖӢбһўбһ…бһӣбһ“бһңбһҸбҹ’бһҗбһ»вҖӢбһҡбһҠбҹ’бһӢбһҖбһҺбҹ’бһҠбһ¶бһӣ]
    C --> D[бһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһ»бһҒбһ„бһ¶бһҡ updateState]
    D --> E[бһҳбһҫбһӣвҖӢбһғбһҫбһүвҖӢбһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ—бҹ’бһӣбһ¶бһҳбҹ—]
```
- **бһ“бһ¶бһ‘бһёбһ‘бһё 1**бҹ– бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһүбҹ’бһ бһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ - бһӣбҹҒбһ„бһҖбҹ’бһ“бһ»бһ„бһӮбһҺбһ“бһё, бһ”бһ“бҹ’бһҸбһ‘бҹҶбһ–бҹҗбһҡ, бһҹбһ„бҹ’бһҖбҹҒбһҸбһҖбһ¶бһҡбһ…бҹҒбһүбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’
- **бһ“бһ¶бһ‘бһёбһ‘бһё 2**бҹ– бһҮбҹҶбһ“бһҪбһҹ `let account = null` бһҮбһ¶ `let state = { account: null }`
- **бһ“бһ¶бһ‘бһёбһ‘бһё 3**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡ `updateState()` бһҹбһ¶бһҳбһүбҹ’бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһҪбһҸбһҸбҹ’бһҡбһ¶
- **бһ“бһ¶бһ‘бһёбһ‘бһё 4**бҹ– бһ”бһ“бҹ’бһҸбһ”бһүбҹ’бһ…бһјбһӣбһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһҪбһҷбһІбҹ’бһҷбһ”бҹ’бһҡбһҫбһӮбҹҶбһҡбһјбһҗбҹ’бһҳбһё
- **бһ“бһ¶бһ‘бһёбһ‘бһё 5**бҹ– бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ—бһ¶бһ–бһўбһ¶бһ…бһ‘бһ¶бһҷбһ”бһ¶бһ“бһҖбһ¶бһ“бҹӢбһҸбҹӮбһӣбҹ’бһў бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бһүбҹ’бһ бһ¶

**бһҸбҹҒбһҹбҹ’бһҸбһҡбҹүбһ¶бһ”бҹӢбһҡбһ„бһӣбһҝбһ“**бҹ–
```javascript
// бһҳбһ»бһ“бҹ– бһҡбһҠбҹ’бһӢбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһ…бҹҒбһү
let account = null; // бһ”бһ¶бһҸбҹӢбһҒбһ¶бһҸбһ–бҹҒбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһЎбһҫбһ„бһңбһ·бһү!

// бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ– бһҡбһҠбҹ’бһӢбһҖбһҺбҹ’бһҸбһ¶бһӣ
let state = Object.freeze({ account: null }); // бһҸбҹ’бһҡбһҪбһҸбһҸбҹ’бһҡбһ¶ бһ“бһ·бһ„бһўбһ¶бһ…бһҸбһ¶бһҳбһҠбһ¶бһ“бһ”бһ¶бһ“!
```

**бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһңбһ¶вҖӢбһҹбҹҶбһҒбһ¶бһ“бҹӢ**бҹ– бһҖбҹ’бһ“бһ»бһ„ 5 бһ“бһ¶бһ‘бһё бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ¶бһ“бһҮбһҪбһ”бһ”бҹ’бһҡбһ‘бҹҮбһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„бһ–бһёбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһјбһ„бһ…бһҳбҹ’бһҡбһ¶бһҹбһ‘бҹ…бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҠбҹӮбһӣбһўбһ¶бһ…бһ‘бһ¶бһҷбһ”бһ¶бһ“ бһ“бһ·бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бһүбҹ’бһ бһ¶бҹ” бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһӮбҹ’бһҡбһ№бҹҮбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһўбһ¶бһ…бһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶бһ”бһ¶бһ“бҹ”

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһӮбһәбһҠбһјбһ…бһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҺбһ¶бһңбһёбһ бҹ’бһӮбҹҒбһҹбһ·бһ“бһӣбһҫбһҷбһ¶бһ“бһҷбһ“бҹ’бһҸ Voyager вҖ“ бһ–бҹҒбһӣбһўбҹ’бһңбһёбһӮбҹ’бһҡбһ”бҹӢбһҷбҹүбһ¶бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбҹ’бһў бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҹбһ„бҹ’бһҖбҹҒбһҸбһғбһҫбһүбһңбһ¶бһ‘бҹҒбҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ–бҹҒбһӣбһўбҹ’бһңбһёбһҳбһ·бһ“бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ бһңбһ¶бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ…бһҳбҹ’бһ”бһ„бһҡбһңбһ¶бһ„бһҖбһ¶бһҡбһҲбһ¶бһ“бһҠбһӣбҹӢбһӣбҹҶбһ бһҖбҹ’бһҡбҹ…бһ•бҹӮбһ“бһҠбһё бһ“бһ·бһ„бһҡбһ»бһүбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһҖбҹ’бһ“бһ»бһ„бһҹбһҸбҹ’бһңбһ–бҹ’бһҡбҹғбһӮбһҳбҹ’бһ„бһ¶бһҸбҹӢбһўбһ¶бһҖбһ¶бһҹбһҷбһ¶бһ“бҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҸбҹҶбһҺбһ¶бһ„бһІбҹ’бһҷбһўбҹ’бһңбһёбҹ—бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ…бһ¶бҹҶбҹ– бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ, бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷаёҹаҘүаӨ°аҘҚаӨ®, бһ”бҹ’бһҡбһңбһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһҡбһ»бһҖбһҡбһҖ, бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“бҹ”

бһ–бҹҒбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бһ“бһ¶бһӮбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹҒбһүбһ–бһёаёҹбҹ’бһҡбһ»бһ„бһҸбҹӮбһҳбһҫбһӣбһӣбҹҒбһ„бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ‘бҹ…бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһӣбһҳбҹ’бһўбһ·бһҸ бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһ”бһ¶бһ“бһҮбһҪбһ”бһ”бҹ’бһҡбһ‘бҹҮбһ”бһүбҹ’бһ бһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҠбҹӮбһӣбһҮбһҪбһ”бһЎбһҫбһ„бһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбҹ” бһ”бһ“бҹ’бһҸбһ‘бҹҶбһ–бҹҗбһҡбһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ…бҹҒбһүбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹ„бһҷбһҳбһ·бһ“бһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбҹ” бһ”бһ·бһ‘бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһ бһҫбһҷбһҖбһ¶бһҡбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбҹ” бһҖбҹӮбһ”бһүбҹ’бһ бһ¶бһӣбҹҶбһ”бһ¶бһҖ бһ бһҫбһҷбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһҪбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ‘бҹ…бһңбһ·бһүбһ‘бҹ…бһҳбһҖбҹ”

бһ“бҹҒбҹҮбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҡбһҝбһ„бһҖбһјбһҠбһҗбһ¶бһ”бһ¶бһ“бһ”бһүбҹ’бһ бһ¶вҖ”бһңбһ¶бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһҲбһәбһ…бһ¶бһ”бҹӢбһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ·бһ“бҹғбһҖбһ¶бһҡбҹүбһҸбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһ“бҹ…бһ–бҹҒбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһҖбһҠбһӣбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҳбһҪбһҷбҹ” бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҸбҹ’бһҡбһјбһңбһҮбһҪбһ”бһ”бҹ’бһҡбһ‘бҹҮбһ”бһүбҹ’бһ бһ¶бһ¶бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҠбһјбһ…бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһ”бҹҶбһӣбҹӮбһ„бһ–бһё "бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„" бһ‘бҹ…бһҮбһ¶ "бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ"бҹ”

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҳбһҪбһҷбһ…бҹҶбһҺбһ»бһ… бһҠбҹӮбһӣбһ”бһҳбҹ’бһӣбҹӮбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бһ“бһ¶бһӮбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“ бһ“бһ·бһ„бһҳбһ¶бһ“бһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ…бһҡбһ¶бһ…бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҡбһ”бҹҖбһ”бһўбһ¶бһ…бһ‘бһ¶бһҷбһ”бһ¶бһ“ бһҡбһҖбҹ’бһҹбһ¶бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбһҳбҹҗбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҡбһӣбһјбһ“бһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҹбһ¶бһҷбһҹбһҳбҹҗбһҷбһҗбҹ’бһҳбһёбһҸбһҳбҹ’бһҡбһјбһңбҹ”

## бһҖбһ¶бһҡбһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“

бһҳбһ»бһ“бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбҹҶбһ“бһ·бһҸбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҷбҹүбһ¶бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бһ“бһ¶бһӮбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҹбһ¶бһ„бһҹбһ„бҹӢбһ”бһ“бҹ’бһҸбһҸбһ¶бһҳбһӮбҹҶбһ“бһ·бһҸ бһ“бһ·бһ„бһҖбһјбһҠбһ–бһёбһ•бҹ’бһ“бҹӮбһҖбһҳбһ»бһ“бҹ—бһ“бҹғбһҹбҹҠбҹҒбһҡбһёбһ“бҹҒбҹҮбҹ”

бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ’бһ¶бһҸбһ»бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҡбһҪбһ…бһҮбһ¶бһҹбҹ’бһҡбҹҒбһ…бһҳбһ»бһ“бһ”бһ“бҹ’бһҸбҹ–

**бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбҹ–**
- бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“ [бһ‘бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ](../3-data/README.md) - бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбһҺбһ“бһёбһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷ
- бһҠбҹҶбһЎбһҫбһ„ [Node.js](https://nodejs.org) бһ“бҹ…бһӣбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҸбҹӢ API бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ

## бһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бһүбҹ’бһ бһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“

бһҠбһјбһ…бһҮбһ¶ Sherlock Holmes бһҖбҹҶбһ–бһ»бһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ“бҹ’бһӣбҹӮбһ„бһ бһ·бһ„бҹ’бһҹбһ¶ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбһўбҹҶбһ–бһёбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбһҫбһҸбһЎбһҫбһ„бһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһўбһ“бһ»бһҳбҹҗбһҸбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҳбһ»бһ“бһ–бҹҒбһӣбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҹбһҳбҹҗбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбҹ”

бһҷбһҫбһ„бһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҹбһ¶бһҳбһүбҹ’бһүбһҳбһҪбһҷ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ”бһүбҹ’бһ бһ¶бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бҹ–

**рҹ§Ә бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҸбҹҒбһҹбҹ’бһҸбһҡбҹүбһ¶бһ”бҹӢбһҡбһ„бһ“бҹҒбҹҮбҹ–**

2. бһ”бһ“бҹ’бһҸбһ‘бҹҶбһ–бҹҗбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё
3. бһҳбһҫбһӣбһҗбһ¶бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„бһ…бҹҶбһ–бҹ„бҹҮбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ…бһјбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһ”бһҫбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһјбһ“бһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ…бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ…бһјбһӣ бһңбһ¶бһҮбһ¶бһўбһҸбҹ’бһҗбһҹбһүбҹ’бһүбһ¶бһ“бҹғбһ”бһүбҹ’бһ бһ¶бһҡбһҖбҹ’бһҹбһ¶бһҹбһҳбҹҗбһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ” бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ“бҹҒбҹҮбһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ”бҹ’бһҡбһҫбһўбһҗбҹҒбһҡбһҮбһ¶бһ—бһ¶бһҹбһ¶ JavaScript бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһЎбһҫбһ„бһңбһ·бһүбһҠбҹ„бһҷбһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ”бһ“бҹ’бһҸбһ‘бҹҶбһ–бҹҗбһҡбҹ”

бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ‘бҹҶбһҡбһ„бҹӢ `account` бһҹбһ¶бһҳбһүбҹ’бһүбҹ–

```js
let account = null;
```

бһҮбһ¶бһҳбһҪбһҷбһңбһҸбҹ’бһҗбһ»бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ–

```js
let state = {
  account: null
};
```

**бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ“бҹҒбҹҮбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ–**
- **бһҖбһҺбҹ’бһҸбһ¶бһӣбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҸбҹӮбһҳбһҪбһҷ**
- **бһҡбҹҖбһ”бһ…бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹ…бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷ**
- **бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһҡбһңбһ¶бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– бһ“бһ·бһ„бһўбһҗбҹҒбһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—**
- **бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһҡбһјбһҠбҹӮбһӣбһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ‘бҹ…бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бҹҶбһ‘бһјбһӣбһ¶бһҷ**

**бһҮбҹҶбһ бһ¶бһ“ 2бҹ– бһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷбһӮбҹҶбһҡбһјбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–**

бһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһІбҹ’бһҷбһ”бҹ’бһҡбһҫбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҗбҹ’бһҳбһёбҹ–

**бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `register()` бһ“бһ·бһ„ `login()`** бһҮбҹҶбһ“бһҪбһҹбҹ–
```js
account = ...
```

бһҮбһ¶бһҳбһҪбһҷбҹ–
```js
state.account = ...
```

**бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ `updateDashboard()`** бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бҹҒбҹҮбһ“бҹ…бһӣбһҫбһҖбҹҶбһ–бһјбһӣбҹ–
```js
const account = state.account;
```

**бһўбҹ’бһңбһёбһҠбҹӮбһӣбһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһ”бһ¶бһ“бҹ–**
- **бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҳбһ»бһҒбһ„бһ¶бһҡбһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҠбҹ„бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӣбҹ’бһў**
- **бһҡбҹҖбһ”бһ…бҹҶбһІбҹ’бһҷбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҹбҹҠбҹ’бһңбһ·бһҸбһҖбҹ’бһҡбҹ„бһҷ**
- **бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһҡбһјбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–**
- **бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһҺбҹ’бһҸбһ¶бһӣ**

> рҹ’Ў **бһ…бҹҶбһҺбһ¶бҹҶ**бҹ– бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ“бҹҒбҹҮбһҳбһ·бһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҗбҹ’бһ„бҹғбһ“бҹҒбҹҮбһ‘бҹҒ, бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹӮбһӣбһҳбҹ’бһўбһҠбҹҸбһҳбһ¶бһ“бһҹбһҖбҹ’бһҸбһ¶бһ“бһ»бһ–бһӣбһ“бҹ…бһҗбҹ’бһ„бҹғбһҖбҹ’бһҡбҹ„бһҷ!

### рҹҺҜ бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ’бҹ’бһңбһҫбһҖбһҺбҹ’бһҸбһ¶бһӣ

**бһҲбһ”бҹӢ бһ“бһ·бһ„бһӮбһ·бһҸбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶**: бһўбҹ’бһ“бһҖбһ‘бһҫбһ”бһҸбҹӮбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҳбһҪбһҷбһҖбһҺбҹ’бһҸбһ¶бһӣбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҳбһҪбһҷбҹ”

**бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһҡбһ бҹҗбһҹ**бҹ–
- бһҸбһҫбһўбҹ’бһ“бһҖбһ”бһҖбһҹбҹ’бһҡбһ¶бһҷбһ”бһ¶бһ“бһ‘бҹҒбһҗбһ¶бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһҖбһ¶бһҡбһҖбһҺбҹ’бһҸбһ¶бһӣбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһңбһҸбҹ’бһҗбһ»бһҳбһҪбһҷбһӣбҹ’бһўбһҮбһ¶бһ„бһўбһҗбҹҒбһҡбһ…бһҸбҹӢбһ…бһҸ?
- бһҸбһҫбһўбҹ’бһңбһёбһўбһ¶бһ…бһҖбһҫбһҸбһЎбһҫбһ„ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ—бҹ’бһӣбҹҒбһ…бһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫ `state.account`?
- бһҸбһҫбһӮбҹҶбһҡбһјбһ“бҹҒбҹҮбһҡбҹҖбһ”бһ…бҹҶбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһҒбҹ’бһ–бһҹбҹӢбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…?

**бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„**бҹ– бһӮбҹҶбһҡбһјбһҖбһҺбҹ’бһҸбһ¶бһӣбһ“бҹҒбҹҮбһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҹбһҳбҹҗбһҷбһҗбҹ’бһҳбһёбһҠбһјбһ…бһҮбһ¶ Redux, Vuex, бһ“бһ·бһ„ React Contextбҹ” бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҹбһ¶бһ„бһҹбһ„бҹӢбһӮбҹҶбһ“бһ·бһҸбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһҳбһ¶бһ“бһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бҹҶбһ“бҹ…бһ–бһ·бһ—бһ–бһӣбҹ„бһҖбҹ”

**бһҹбҹҶбһҺбһҪбһҡбһҹбһҳбҹ’бһҡбһҪбһӣ**бҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ“бҹ’бһҗбҹӮбһҳбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ (бһ–бһёбһҡбһ”бҹҖбһ”бһ•бҹ’бһ‘бһ¶бҹҶбһ„, бһ—бһ¶бһҹбһ¶) бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҸбһҫбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бһ‘бһёбһҺбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–? бһңбһ¶бһ“бһ№бһ„бһ–бһ„бҹ’бһҡбһёбһҖбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…?

## бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһҪбһҸбһҸбҹ’бһҡбһ¶

бһҠбҹ„бһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһ”бһ¶бһ“бһҖбһҺбҹ’бһҸбһ¶бһӣ бһҡбһ”бҹҖбһ”бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһӮбһәбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһҸбҹ’бһҡбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһңбһ¶бһ“бһ№бһ„бһ’бһ¶бһ“бһ¶бһҗбһ¶бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһўбһ¶бһ…бһ‘бһ¶бһҷбһ”бһ¶бһ“ бһ“бһ·бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бһүбҹ’бһ бһ¶бҹ”

бһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһҹбҹ’бһ“бһјбһӣбһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ…бһҡбһ¶бһ…бһҡбһ бҹ„бҹҮбһ бһҫбһҡбҹ– бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһІбҹ’бһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹ„бһҷбһЎбҹӮбһҖ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бһүбҹ’бһҮбһјбһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҸбһ¶бһҳбһҡбһҷбҹҲбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷ бһҠбҹ„бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһҸбҹ’бһҡбһ¶бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ” бһӮбҹҶбһҡбһјбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһҸбҹ’бһҡбһ¶ бһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһҗбһ¶ бһ–бҹҒбһӣбһҺбһ¶бһ бһҫбһҷбһҸбһҫбһҠбһјбһ…бһҳбҹ’бһҠбҹҒбһ…бһҠбҹӮбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбҹ”

**бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹӮбһӣбһҳбһ·бһ“бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ”бһ¶бһ“**

бһҷбһҫбһ„бһ“бһ№бһ„бһ…бһ·бһүбҹ’бһ…бһ№бһҳ `state` бһҮбһ¶ [*бһўбһ…бһӣбһ“бһңбһҸбҹ’бһҗбһ»*](https://en.wikipedia.org/wiki/Immutable_object) бһҠбҹ„бһҷбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҷбһҫбһ„бһҳбһ·бһ“бһ”бҹ’бһҸбһјбһҡбһҸбһҳбҹ’бһӣбҹғбһңбһ¶бһ”бһүбҹ’бһүбһ¶бһҖбҹӢбһ‘бҹ…бһӣбһҫбһңбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбһ„бһ‘бҹҒбҹ” бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ”бһ¶бһ“бһӣбһҫбһҖбһ‘бһ№бһҖбһ…бһ·бһҸбҹ’бһҸбһІбҹ’бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһңбһҸбҹ’бһҗбһ»бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҗбҹ’бһҳбһёбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһҸбҹ’бһҡбһҪбһҸбһҸбҹ’бһҡбһ¶бҹ”

бһ”бһҫбһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһңбһ·бһ’бһёбһ“бҹҒбҹҮбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹҒбһ…бһ¶бһҸбҹӢбһ‘бһ»бһҖбһҗбһ¶бһҳбһ·бһ“бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҮбһ¶бһҳбһ»бһ“ бһҖбҹҸбһҠбҹ„бһҷбһңбһ¶бһ•бҹ’бһҸбһӣбҹӢбһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бһүбҹ’бһ бһ¶ бһҖбһ¶бһҡбһ”бҹ’бһҡбһЎбһ„ бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ‘бһ¶бһҷбһ”бһ¶бһ“бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ”

**бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ“бҹғбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһўбһ…бһӣбһ“бһңбһҸбҹ’бһҗбһ»бҹ–**

|---------|-------------|--------|
| **бһўбһ¶бһ…бһ‘бһ¶бһҷбһ”бһ¶бһ“** | бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ’бҹ’бһңбһҫбһҸбҹӮбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһҸбҹ’бһҡбһ¶ | бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бһүбҹ’бһ бһ¶бһ“бһ·бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ |
| **бһҸбһ¶бһҳбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·** | бһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһ„бҹ’бһҖбһҫбһҸбһңбһҸбҹ’бһҗбһ»бһҗбҹ’бһҳбһё | бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһӣбһ»бһ”бһ…бҹ„бһӣ/бһ’бҹ’бһңбһҫбһҳбҹ’бһҠбһ„бһ‘бҹҖбһҸ |
| **бһҖбһ¶бһҡбһ–бһ¶бһҡбһ”бһүбҹ’бһ бһ¶бһ”бҹ’бһӣбҹӮбһҖбҹ—** | бһҳбһ·бһ“бһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ…бҹғбһҠбһ“бҹ’бһҷ | бһ”бһ„бҹ’бһҖбһ¶бһҡбһ”бһүбҹ’бһ бһ¶бһҠбһӣбҹӢбһұбҹ’бһҷбһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ |
| **бһўбһ»бһ”бҹ’бһ‘бһёбһҳбҹүбһ¶бһүбҹ’бһҷбҹҒбһҹбһ·бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–** | бһ„бһ¶бһҷбһҹбҹ’бһҖбҹҒбһ“бһ–бҹҒбһӣбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ | бһўбһ¶бһ…бһўбһ¶бһ”бҹӢбһҠбҹҒбһҸ UI бһҠбҹ„бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– |

**JavaScript бһ“бһ·бһ„ Object.freeze() бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ…бһӣбһ“бһңбһҸбҹ’бһҗбһ»бҹ–**

JavaScript бһ•бҹ’бһҸбһӣбҹӢ [`Object.freeze()`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object/freeze) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһңбһҸбҹ’бһҗбһ»бҹ–

```js
const immutableState = Object.freeze({ account: userData });
// бһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҺбһ¶бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ immutableState бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹҶбһ бһ»бһҹ
```

**бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбһҫбһҸбһЎбһҫбһ„бҹ–**
- **бһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡ** бһҖбһ¶бһҡбһ…бҹҶбһҺбһҸбһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһӣбһ»бһ”
- **бһ”бҹ„бҹҮбһ”бһ„бҹӢ** бһҖбһ¶бһҡбһҒбһјбһ…бһҒбһ¶бһҸбһҠбҹӮбһӣбһ…бһ„бҹӢбһҖбһҫбһҸбһЎбһҫбһ„
- **бһ’бһ¶бһ“бһ¶** бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһҸбҹ’бһҡбһ¶
- **бһ”бһ„бҹ’бһҖбһҫбһҸ** бһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһҷбһ¶бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–

> рҹ’Ў **бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҮбҹ’бһҡбҹ…**бҹ– бһҹбһјбһҳбһҡбҹҖбһ“бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖ *shallow* бһ“бһ·бһ„ *deep* бһҡбһ”бһҹбҹӢбһўбһ…бһӣбһ“бһңбһҸбҹ’бһҗбһ»бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [бһҜбһҖбһҹбһ¶бһҡ MDN](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object/freeze#What_is_shallow_freeze)аҘӨ бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ“бҹҒбҹҮбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҺбһ¶бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбҹ”

### бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…

бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡ `updateState()` бһҗбҹ’бһҳбһёбҹ–

```js
function updateState(property, newData) {
  state = Object.freeze({
    ...state,
    [property]: newData
  });
}
```

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһңбһҸбҹ’бһҗбһ»бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҗбҹ’бһҳбһё бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһ¶бһ”бҹӢбһ…бһҳбҹ’бһӣбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҳбһ»бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ [* spread (`...`) operator*](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Operators/Spread_syntax#Spread_in_object_literals). бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһҷбһҫбһ„бһ”бҹ’бһҸбһјбһҡбһҸбһҳбҹ’бһӣбҹғбһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҳбһҪбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһҸбҹ’бһҗбһ»бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ [bracket notation](https://developer.mozilla.org/docs/Web/JavaScript/Guide/Working_with_Objects#Objects_and_properties) `[property]` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҸбһҳбҹ’бһӣбҹғбҹ” бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһ”бһ·бһ‘бһңбһҸбҹ’бһҗбһ»бһ“бҹҒбҹҮбһҠбҹ„бһҷ `Object.freeze()` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбҹ” бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһҷбһҫбһ„бһҳбһ¶бһ“бһҸбҹӮбһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲ `account` бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһҸбҹӮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲбһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹҒбҹҮбҹ”

бһҷбһҫбһ„бһҖбҹҸбһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷбһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ `state` бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹҶбһ”бһјбһ„бһӮбһәбһ”бһ¶бһ“бһҸбҹҶбһҡбһ№бһ„бһ•бһ„бһҠбҹӮбһҡбҹ–

```js
let state = Object.freeze({
  account: null
});
```

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷбһҳбһ»бһҒбһ„бһ¶бһҡ `register` бһҠбҹ„бһҷбһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһӣ `state.account = result;` бһҮбһ¶бҹ–

```js
updateState('account', result);
```

бһ’бҹ’бһңбһҫбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһҮбһ¶бһҳбһҪбһҷбһҳбһ»бһҒбһ„бһ¶бһҡ `login` бһҮбҹҶбһ“бһҪбһҹ `state.account = data;` бһҮбһ¶бҹ–

```js
updateState('account', data);
```

бһҷбһҫбһ„бһ“бһ№бһ„бһ…бһ¶бһ”бҹӢбһұбһҖбһ¶бһҹбһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҹбһҮбһ»бһӣбһ”бһүбҹ’бһ бһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбһҺбһ“бһёбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбҹ’бһўбһ¶бһҸбһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ *Logout*бҹ”

бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡ `logout()` бһҗбҹ’бһҳбһёбҹ–

```js
function logout() {
  updateState('account', null);
  navigate('/login');
}
```

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `updateDashboard()`, бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһ”бһ„бҹ’бһңбҹ’бһҷбһ“бҹӢ `return navigate('/login');` бһҮбһ¶ `return logout();`

бһҹбһјбһҳбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһӮбһҺбһ“бһёбһҗбҹ’бһҳбһё бһ…бҹҒбһүбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ бһҫбһҷбһ…бһјбһӣбһңбһ·бһү бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһңбһёбҹ—бһ“бҹ…бһҸбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбҹ’бһўбҹ”

> бһҮбҹҶбһ“бһҪбһҷбҹ– бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҠбҹ„бһҷбһ”бһ“бҹ’бһҗбҹӮбһҳ `console.log(state)` бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ `updateState()` бһ бһҫбһҷбһ”бһҫбһҖ console бһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

## бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһҷбҹҲбһ–бҹҒбһӣбһҷбһјбһҡ

бһ”бһүбҹ’бһ бһ¶бһҒбһ¶бһҸбһ”бһ„бҹӢбһҹбһҳбҹҗбһҷбһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ–бһёбһҳбһ»бһ“ бһҸбһҳбҹ’бһҡбһјбһңбһІбҹ’бһҷбһҳбһ¶бһ“бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҠбҹӮбһӣбһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҮбһ¶бһ‘бҹҖбһ„бһ‘бһ¶бһҸбҹӢбһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһҹбһҳбҹҗбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбҹ” бһңбһ¶бһ”бһҳбҹ’бһӣбҹӮбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһ–бһёбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“бһ‘бҹ…бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“бҹ”

бһҹбһјбһҳбһӮбһ·бһҸбһІбҹ’бһҷбһ”бһ¶бһ“бһҗбһ¶ бһ“бһ¶бһЎбһ·бһҖбһ¶бһўбһ¶бһҸбһјбһҳбһҸбһ·бһҖбһҖбһ¶бһ“бҹӢбһҸбҹӮбһҳбһ¶бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ‘бҹ„бҹҮбһ”бһёбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһҗбһ¶бһҳбһ–бһӣбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ»бһҖбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҳбҹүбһјбһҡбһёбһҳбһ·бһ“бһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбҹ” бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҹбһ¶бһҷбһҸбһҳбҹ’бһҡбһјбһңбһІбҹ’бһҷбһҳбһ¶бһ“бһҳбҹҒбһҖбһ¶бһ“бһ·бһ…бһ•бҹ’бһ‘бһ»бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһҹбһҳбҹҗбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһ“бһ·бһ„бһ”бһ“бҹ’бһҸбһ‘бҹҶбһ–бҹҗбһҡбҹ”

**бһҹбҹҶбһҺбһҪбһҡбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһҷбҹҲбһ–бҹҒбһӣбһҷбһјбһҡбҹ–**

бһҳбһ»бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ бһҹбһјбһҳбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҡбһҝбһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

| бһҹбҹҶбһҺбһҪбһҡ | бһ”бһҡбһ·бһ”бһ‘бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бһ“бһ¶бһӮбһ¶бһҡ | бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҮбһ¶бһҳбһ»бһ“бһҹбһҳбҹ’бһҡбҹҒбһ… |
|----------|-------------------|----------------|
| **бһҸбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһ‘бҹҒ?** | бһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһӮбһҺбһ“бһё, бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ | бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһңбһ·бһ’бһёбһ•бҹ’бһ‘бһ»бһҖбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– |
| **бһҸбһҫбһңбһ¶бһҸбҹ’бһҡбһјбһңбһҡбһҖбҹ’бһҹбһ¶бһҡбһҷбҹҲбһ–бҹҒбһӣбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“?** | бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ…бһјбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһҮбһ¶бһҳбһҪбһҷбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“ | бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҡбһҷбҹҲбһ–бҹҒбһӣбһ•бҹ’бһ‘бһ»бһҖбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң |
| **бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһңбһ¶бһ‘бҹҒ?** | бһҸбһҪбһўбһҖбҹ’бһҹбһҡбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ бһ“бһ·бһ„ бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ UI | бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡвҖӢбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖ |

**бһҮбһҳбҹ’бһҡбһҫбһҹбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбҹ–**

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһ‘бҹҶбһ“бһҫбһ”бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҳбҹҒбһҖбһ¶бһ“бһ·бһ…бһ•бҹ’бһ‘бһ»бһҖбһҒбҹ’бһӣбҹҮбҹ— бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бҹ–

**API бһ•бҹ’бһ‘бһ»бһҖбһҠбһҫбһҳбҹ–**

1. **[`localStorage`](https://developer.mozilla.org/docs/Web/API/Window/localStorage)**бҹ– бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖ [key/value](https://en.wikipedia.org/wiki/Key%E2%80%93value_database) бһҠбҹҸбһҗбҹҒбһҡбһ…бһ“бҹ’бһӣбҹ„бҹҮбһңбһӮбҹ’бһӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ
   - **бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ** бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһңбһӮбҹ’бһӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ
   - **бһӮбҹ’бһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶** бһ“бҹ…бһ–бҹҒбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһ”бһ·бһ‘ бһ“бһ·бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһЎбһҫбһ„бһңбһ·бһү
   - **бһҳбһ¶бһ“бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢ** бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһҠбҹӮбһ“бһ“бҹғбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ–бһ·бһҹбҹҒбһҹ
   - **бһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮ** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҶбһҺбһ„бҹӢбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ…бһјбһӣбһӮбһҺбһ“бһё

2. **[`sessionStorage`](https://developer.mozilla.org/docs/Web/API/Window/sessionStorage)**бҹ– бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһңбһӮбҹ’бһӮбһҖбһ¶бһӣбһ”бһҺбҹ’бһҠбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“
   - **бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ** бһҠбһјбһ… `localStorage` бһ“бҹ…бһ–бҹҒбһӣбһңбһӮбҹ’бһӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҶбһҖбһ·бһӣ
   - **бһӣбһ»бһ”бһ…бҹ„бһӣ** бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ–бҹҒбһӣбһ”бһ·бһ‘бһ…бһ»бһ„бһҸбһ¶бһ”бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ
   - **бһҹбһҳбһҡбһҳбҹ’бһҷ** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһ”бһҺбҹ’бһҠбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“бһҠбҹӮбһӣбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһӮбҹҒбһ…бһ„бһ…бһ¶бҹҶ

3. **[HTTP Cookies](https://developer.mozilla.org/docs/Web/HTTP/Cookies)**бҹ– бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ–бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ
   - **бһ”бһүбҹ’бһ…бһјбһ“бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·** бһҮбһ¶бһҳбһҪбһҷбһҹбҹҶбһҺбһҫбһҡвҖӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҡбһ¶бһӣбҹӢбһҠбһ„
   - **бһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮ** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһҪбһўбһҖбҹ’бһҹбһҡбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ
   - **бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ‘бҹҶбһ бҹҶ** бһ бһҫбһҷбһўбһ¶бһ…бһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһӣбһҫбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·

**бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡвҖӢбһҹбҹҶбһӣбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ–**

бһ‘бһ¶бҹҶбһ„ `localStorage` бһ“бһ·бһ„ `sessionStorage` бһ•бҹ’бһ‘бһ»бһҖбһҸбҹӮбһ“бһјбһң [бһўбһҖбҹ’бһҹбһҡ](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/String) бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹҲ

```js
// бһ”бһҳбҹ’бһӣбҹӮбһ„бһңбһҸбҹ’бһҗбһ»бһ‘бҹ…бһҮбһ¶бһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡ JSON бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ
const accountData = { user: 'john', balance: 150 };
localStorage.setItem('account', JSON.stringify(accountData));

// бһ”бһҖбһ”бҹ’бһҡбҹӮбһҖбһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡ JSON бһңбһ·бһүбһ‘бҹ…бһҮбһ¶бһңбһҸбҹ’бһҗбһ»бһ–бҹҒбһӣбһҷбһҖбһҳбһҖбһңбһ·бһү
const savedAccount = JSON.parse(localStorage.getItem('account'));
```

**бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҖбһ¶бһҡбһҹбҹҶбһӣбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ–**
- **бһ”бһҳбҹ’бһӣбҹӮбһ„** бһӮбҹ’бһҡбһҝбһ„бһўбҹҒбһЎбһ·бһ…бһҸбҹ’бһҡбһјбһ“бһ·бһ… JavaScript бһ‘бҹ…бһҮбһ¶бһўбһҖбҹ’бһҹбһҡ JSON бһҸбһ¶бһҳбһ”бҹ’бһҡбһҫ [`JSON.stringify()`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/JSON/stringify)
- **бһҖбһҹбһ¶бһ„бһЎбһҫбһ„бһңбһ·бһү** бһңбһҸбҹ’бһҗбһ»бһ–бһё JSON бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ [`JSON.parse()`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/JSON/parse)

- **бһ”бһҡбһ¶бһҮбҹҗбһҷ** бһ“бҹ…бһӣбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ·бһ“бһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„бһҷбҹ„бһ„бһңбһ„бҹӢбһ…бһ„

### бһҖбһ¶бһҡбһ„бһ¶бһҡбҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡвҖӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ `localStorage` бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҗбҹҒбһҡ

бһҷбһҫбһ„бһ“бһ№бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҸбҹӢбһ•бҹ’бһ‘бһ»бһҖбһҗбҹҒбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ“бҹ…бһҸбҹӮбһ…бһјбһӣбһӮбһҺбһ“бһёбһҡбһ бһјбһҸбһҠбһӣбҹӢбһ–бһҪбһҖбһӮбҹҒбһ…бҹҒбһүбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫ `localStorage` бһҠбһ¶бһҖбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбһҺбһ“бһёбһҖбҹ’бһ“бһ»бһ„бһңбһӮбҹ’бһӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбҹ”

**бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 1бҹ– бһҖбҹҶбһҺбһҸбҹӢбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһ‘бһ»бһҖ**

```js
const storageKey = 'savedAccount';
```

**бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҗбҹҒбһҡбһ“бҹҒбҹҮбһ•бҹ’бһҠбһӣбҹӢбһҮбһјбһ“бҹ–**
- **бһ”бһ„бҹ’бһҖбһҫбһҸ** бһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһҜбһҖбһҹбһ·бһ‘бҹ’бһ’бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖ
- **бһ‘бһ”бҹӢбһҹбҹ’бһҖбһ¶бһҸбҹӢ** бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҖбһјбһҠбһҒбһ»бһҹбһҖбҹ’бһ“бһ»бһ„бһҖбһјбһ“бһӮбҹ’бһҡбһ¶бһ”бҹӢ key
- **бһ’бҹ’бһңбһҫбһұбҹ’бһҷ** бһӣбҹӮбһ„бһҠбһјбһҡбһҖбһјбһ“бһӮбҹ’бһҡбһ¶бһ”бҹӢ key бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҹбҹ’бһҡбһҪбһӣбһ”бһҫбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ
- **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳ** бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһјбһҠбһҠбҹӮбһӣбһ„бһ¶бһҷбһҗбҹӮбһ‘бһ¶бҹҶ

**бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 2бҹ– бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҗбҹҒбһҡбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·**

бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бҹҒбҹҮбһ“бҹ…бһ…бһ»бһ„бһҹбһҳбһ¶бһҹбһ—бһ¶бһ– `updateState()` бҹ–

```js
localStorage.setItem(storageKey, JSON.stringify(state.account));
```

**бһ”бҹҶбһ”бҹӮбһҖбһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбҹ–**
- **бһ”бһҳбҹ’бһӣбҹӮбһ„** бһўбһ…бһӣбһ“бһңбһҸбҹ’бһҗбһ»бһӮбһҺбһ“бһёбһ‘бҹ…бһҮбһ¶ JSON string бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖ
- **бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ** бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһјбһ“бһӮбҹ’бһҡбһ¶бһ”бҹӢ storage key бһҜбһҖбһҹбһ·бһ‘бҹ’бһ’
- **бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ** бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһёбһҸбһ¶бҹҶбһ„
- **бһ’бһ¶бһ“бһ¶** бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһ‘бһ»бһҖбһҸбҹӮбһ„бһҸбҹӮбһҹбһҳбһҹбҹ’бһҡбһ”бһҮбһ¶бһҳбһҪбһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“

> рҹ’Ў **бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбҹ–** бһ–бҹ’бһҡбҹ„бҹҮбһҷбһҫбһ„бһ”бһ¶бһ“бһ•бҹ’бһҠбҹ„бһҸбһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҸбһ¶бһҳ `updateState()` бһҳбһҪбһҷбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһ…бҹҶбһҺбһ»бһ…бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһўбһ“бһ»бһ—бһ¶бһ–бһ“бҹғбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҡбҹҖбһ”бһ…бҹҶбһӣбҹ’бһў!

**бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 3бҹ– бһҹбҹ’бһҠбһ¶бһҡбһЎбһҫбһ„бһңбһ·бһүбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ–бҹҒбһӣбһ•бҹ’бһ‘бһ»бһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё**

бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҠбһ¶бһҡбһЎбһҫбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбҹ–

```js
function init() {
  const savedAccount = localStorage.getItem(storageKey);
  if (savedAccount) {
    updateState('account', JSON.parse(savedAccount));
  }

  // бһҖбһјбһҠбһ•бҹ’бһҠбһҫбһҳбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбһҫбһҳбһҡбһ”бһҹбҹӢбһҷбһҫбһ„
  window.onpopstate = () => updateRoute();
  updateRoute();
}

init();
```

**бһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ initializationбҹ–**
- **бһ‘бһ¶бһүбһҷбһҖ** бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбһҺбһ“бһёбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ–бһё `localStorage`
- **бһ”бһҳбҹ’бһӣбҹӮбһ„бһҸбҹ’бһҡбһЎбһ”бҹӢ** JSON string бһ‘бҹ…бһҮбһ¶бһўбһ…бһӣбһ“бһңбһҸбҹ’бһҗбһ» JavaScript
- **бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–** бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҸбһ¶бһҳбһҳбһ»бһҒбһ„бһ¶бһҡбһ”бҹ’бһҸбһјбһҡбһ•бҹ’бһ“бҹӮбһҖбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
- **бһҹбҹ’бһҠбһ¶бһҡбһЎбһҫбһ„** бһңбһӮбҹ’бһӮбһҖбһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹ…бһ–бҹҒбһӣбһ•бҹ’бһ‘бһ»бһҖбһ‘бҹҶбһ–бҹҗбһҡ
- **бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһ»бһ“** бһҖбһ¶бһҡбһ”бҹ’бһҠбһјбһҡбһ–бһёбһҡбһ“бһ¶бҹҶбһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһҳбҹ’бһӣбһ„

- **бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ** dashboard бһҡбҹҖбһ”бһ…бҹҶбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ authentication
- **бһ”бһүбҹ’бһҮбһјбһ“бһҸбҹ’бһҡбһЎбһ”бҹӢ** бһ…бһјбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһ”бһҫбһҳбһ·бһ“бһҳбһ¶бһ“бһңбһӮбҹ’бһӮбһҖбһ¶бһӣбһҹбҹ’бһ”бҹӮбһҖбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ
- **бһ”бһ„бҹ’бһҖбһҫбһҸ** бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбһјбһ…бһҮбһ¶бһҷбһјбһҡбһўбһ„бҹ’бһңбҹӮбһ„

**бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–**

1. бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бһ“бһ¶бһӮбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

4. бһ”бһ·бһ‘бһ бһҫбһҷбһ”бһҫбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһЎбһҫбһ„бһңбһ·бһү
5. бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһўбҹ’бһ“бһҖбһ“бҹ…бһҸбҹӮбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“

рҹҺү **бһҹбһҳбһ·бһ‘бһ·бҹ’бһ’бһ”бһ¶бһ“бһ…бҹҶбһ–бҹҒбһү**бҹ– бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҗбҹҒбһҡбһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷ! бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҘбһЎбһјбһңбһҳбһ¶бһ“бһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһҠбһјбһ…бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбҹ”

### рҹҺҜ бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бҹ– бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ persistence

**бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ**бҹ– бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢ persistence бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһҠбҹӮбһӣбһ…бһҳбҹ’бһҡбһ»бҹҮбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҮбһ¶бһҳбһҪбһҷбһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ“бҹғбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

**бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹҶбһ“бһ·бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбҹ–**
- **JSON Serialization**бҹ– бһ”бһҳбҹ’бһӣбҹӮбһ„бһңбһҸбҹ’бһҗбһ»бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ‘бҹ…бһҮбһ¶string бһҠбҹӮбһӣбһўбһ¶бһ…бһ•бҹ’бһ‘бһ»бһҖбһ”бһ¶бһ“
- **Synchronization бһҮбһ¶бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·**бҹ– бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһ„бҹ’бһҖбһ”бһ“бҹ’бһ‘бһ»бһҖбһҗбҹҒбһҡбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
- **бһҖбһ¶бһҡбһҹбҹ’бһҠбһ¶бһҡбһңбһӮбҹ’бһӮбһҖбһ¶бһӣ**бҹ– бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ¶бһ…бһҹбҹ’бһҠбһ¶бһҡбһ”бһҡбһ·бһ”бһ‘бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҳбһ¶бһ“бһҖбһ¶бһҡбһҮбҹ’бһҡбҹҒбһҖбһ…бһ·бһҸбҹ’бһҸ
- **Persistence бһҖбһҺбҹ’бһҠбһ¶бһӣ**бҹ– бһҳбһ»бһҒбһ„бһ¶бһҡбһ”бҹ’бһҸбһјбһҡбһҸбҹӮбһҳбһҪбһҷбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ

**бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ§бһҹбҹ’бһҹбһ¶бһ бһҖбһҳбҹ’бһҳ**бҹ– бһҡбһ…бһ“бһ¶бһ”бһҺбҹ’бһҸбһ»бҹҶ persistence бһ“бҹҒбҹҮбһӮбһәбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҡбһҪбһҳ (PWAs), бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹ’бһҡбҹ…бһ”бһҺбҹ’бһҸбһ¶бһүбһҮбһ¶бһҳбһ»бһ“, бһ“бһ·бһ„бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһңбҹҒбһ”бһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһ‘бҹҶбһ“бһҫбһ”бҹ” бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҖбһҹбһ¶бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбһҳбҹ’бһҡбһ·бһҸбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ”

**бһҹбҹҶбһҺбһҪбһҡбһҸбҹ’бһҡбһӣбһ”бҹӢбһҳбһҖбһңбһ·бһү**бҹ– бһҸбһҫбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ“бҹҒбҹҮбһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ… бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӮбһҺбһ“бһёбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ…бҹ’бһҡбһҫбһ“бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбһҠбҹӮбһӣ? бһҹбһјбһҳбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡ бһ—бһ¶бһ–бһҜбһҖбһҮбһ“ бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ”

## бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ” persistence бһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹ’бһҡбһҹбҹӢ

бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ persistence бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҡбһҪбһ…бһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһңбһӮбҹ’бһӮбһҖбһ¶бһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“ бһҹбһјбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһүбҹ’бһ бһ¶бһҗбҹ’бһҳбһёбҹҲ бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһ¶бһҹбҹӢ (data staleness)бҹ” бһ–бҹҒбһӣбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ¬бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бҹ’бһҡбһҫбһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ local cached бһ“бһ№бһ„бһҖбҹ’бһӣбһ¶бһҷбһ‘бҹ…бһҮбһ¶бһ…бһ¶бһҹбҹӢбҹ”

бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹҒбҹҮбһҠбһјбһ…бһҳбҹ’бһҠбҹҒбһ…бһ“бҹғбһўбҹ’бһ“бһҖбһҠбһ№бһҖбһ“бһ¶бҹҶбһҠбһ„бһҖбһ¶бҹҶбһ—бҹ’бһӣбһҫбһ„ Viking бһҠбҹӮбһӣбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ•бҹӮбһ“бһ‘бһёбһ–бҹ’бһҡбһӣбһ№бһ„бһ•бҹ’бһҖбһ¶бһҷбһҠбҹҶбһ”бһјбһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ„бҹ’бһҖбһҸбҹӢбһҳбһҫбһӣбһ•бҹ’бһҖбһ¶бһҷбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ” бһ•бҹӮбһ“бһ‘бһёбһ•бҹ’бһҠбһӣбҹӢбһ“бһјбһңбһ—бһ¶бһ–бһҗбҹҒбһҡ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбҹ’бһҡбһ»бһҳбһҠбһ№бһҖбһ“бһ¶бҹҶбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһ„бҹ’бһҖбһҸбҹӢбһҳбһҫбһӣбһҗбҹ’бһҳбһёбһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбҹ” бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бҹҒбҹҮ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҗбҹҒбһҡбһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ”

### бһҖбһ¶бһҡбһ„бһ¶бһҡбҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһё

бһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҚбһҗбҹ’бһҳбһёбһ–бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһҒбһҺбҹҲбһҠбҹӮбһӣбһҡбһҖбҹ’бһҹбһ¶бһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ persistenceбҹ”

**бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 1бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбһҺбһ“бһё**

```js
async function updateAccountData() {
  const account = state.account;
  if (!account) {
    return logout();
  }

  const data = await getAccount(account.user);
  if (data.error) {
    return logout();
  }

  updateState('account', data);
}
```

- **бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ** бһҗбһ¶бһҸбһҫбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһ…бһјбһӣбһӮбһҺбһ“бһё (state.account бһҳбһ¶бһ“)
- **бһ”бһүбҹ’бһҮбһјбһ“бһҸбҹ’бһҡбһӣбһ”бҹӢ** бһ…бҹҒбһүбһ…бһјбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ·бһ“бһҳбһ¶бһ“бһңбһӮбҹ’бһӮбһҖбһ¶бһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
- **бһҷбһҖ** бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбһҺбһ“бһёбһҗбҹ’бһҳбһёбһ–бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ `getAccount()`
- **бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ** бһҖбҹҶбһ бһ»бһҹбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҠбҹ„бһҷбһ…бҹҒбһүбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбһҺбһ“бһёбһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
- **бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–** бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбһҸбһ¶бһҳбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ update бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„
- **бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҸбһ¶бһҳбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·** бһҠбҹ„бһҷаёңа№ҲаёІаёҷбһҳбһ»бһҒбһ„бһ¶бһҡ `updateState()`

**бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 2бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһё бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ– dashboard**

  await updateAccountData();
  updateDashboard();

- **бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ** бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ refresh бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ– UI
- **бһҡбһ„бҹӢбһ…бһ¶бҹҶ** бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбһҳбһҖбһҠбһӣбҹӢ бһҳбһ»бһ“бһ–бҹҒбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣ
- **бһ’бһ¶бһ“бһ¶** бһҗбһ¶ dashboard бһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷбһ”бҹҶбһ•бһ»бһҸ
- **бһҡбһҖбҹ’бһҹбһ¶** бһҖбһ¶бһҡбһ…бҹӮбһҖбһ•бҹ’бһ“бҹӮбһҖбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһҡбһңбһ¶бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„ UI

**бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 3бҹ– бһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бһёбһҸбһ¶бҹҶбһ„**

**бһҡбһ”бҹҖбһ”бһ”бһүбҹ’бһ…бһјбһӣбһ“бҹҒбҹҮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ–**
- **бһўбһ“бһ»бһңбһҸбҹ’бһҸ** бһҳбһ»бһҒбһ„бһ¶бһҡ refresh бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҠбҹӮбһӣ route dashboard бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖ
- **бһ’бһ¶бһ“бһ¶** бһҮбһ¶бһҹбҹ’бһҡбҹҒбһ…бһҗбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбһҸбҹӮбһ„бһҸбҹӮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ‘бҹ…бһҖбһ¶бһ“бҹӢ dashboard
- **бһҡбһҖбҹ’бһҹбһ¶** бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ‘бһёбһҸбһ¶бҹҶбһ„бһ…бһ¶бһҹбҹӢбһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҖбҹ’бһҡбҹ„бһҳбһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹ’бһҡбһҹбҹӢ
- **бһ•бҹ’бһҠбһӣбҹӢ** бһӮбҹҶбһҡбһјбһҸбҹӮбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ initialization бһ‘бҹ…бһҸбһ¶бһҳбһҸбҹҶбһҺбһ¶бһҖбҹӢбһҸбҹҶбһҺбҹӮбһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„

**рҹҺ“ бһҮбҹҶбһ“бһ¶бһ“бҹӢбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶**бҹ– бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҖбһҹбһ¶бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ–бҹҒбһүбһӣбҹҒбһүбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҠбһјбһ… Redux, Vuex, бһ“бһ·бһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” бһӣбҹҶбһ“бһ¶бҹҶбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһӣбҹҒбһ…бһ’бҹ’бһӣбҹ„бһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбһјбһ…бһҠбһӣбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ”бҹӢбһҡбҹҶбһ’бҹҶбҹ”

**рҹ”„ бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбһҳбҹ’бһҡбһ·бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ–**
- бһҡбҹҖбһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҮбһ¶бһҳбһҪбһҷ framework (Redux, Zustand, Pinia)
- бһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бҹҒбһӣ-бһ–бһ·бһҸбһҮбһ¶бһҳбһҪбһҷ WebSockets
- бһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Progressive Web Apps бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҖбҹ’бһҡбҹ…бһ”бһҺбҹ’бһҸбһ¶бһү
- бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҶбһ“бһ¶бҹҶбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҠбһјбһ…бһҮбһ¶ machine states бһ“бһ·бһ„ observers

## бһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“бһ§бһ‘бҹ’бһ‘бҹҒбһҹбһ…бһҡбһҺбҹҚ GitHub Copilot рҹҡҖ

бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠ Agent бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһ”бһүбҹ’бһ бһ¶бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

**бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бҹ–** бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡ undo/redo бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бһ“бһ¶бһӮбһ¶бһҡбҹ” бһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӮбҹҶбһ“бһ·бһҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһ“бҹғбһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–, бһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһўбһ…бһӣбһ“бһңбһҸбҹ’бһҗбһ»бһҠбҹӮбһӣбһҳбһ·бһ“бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ, бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ UIбҹ”

**бһ”бһүбҹ’бһҮбһ¶бҹ–** бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹӮбһӣбһҳбһ¶бһ“бҹ– бҹЎ) бһўбһ¶бһҡбҹҒбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ; бҹў) бһҳбһ»бһҒбһ„бһ¶бһҡ undo бһ“бһ·бһ„ redo бһҠбҹӮбһӣбһўбһ¶бһ…бһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ…бһҡбһҠбҹ’бһӢбһҳбһ»бһ“бҹ—; бҹЈ) бһ”бҹҠбһјбһҸбһ»бһ„ UI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢoperaundo/redo бһ“бҹ…бһӣбһҫ dashboard; бҹӨ) бһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбҹҶбһҡбһ¶бһ”бҹӢбһҸбҹ’бһҡбһ№бһҳ бҹЎбҹ  бһҡбһҠбҹ’бһӢ бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„бһ”бһүбҹ’бһ бһ¶бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶ; бһ“бһ·бһ„ бҹҘ) бһҹбһҳбҹ’бһўбһ¶бһҸбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ…бҹҒбһүбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ” бһ’бһ¶бһ“бһ¶бһҗбһ¶бһҳбһ»бһҒбһ„бһ¶бһҡ undo/redo бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢ бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ”бһ¶бһ“бһҖбҹ’бһҡбҹ„бһҳбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһЎбһҫбһ„бһңбһ·бһүбһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбҹ”

бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһё [agent mode](https://code.visualstudio.com/blogs/2025/02/24/introducing-copilot-agent-mode) бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбҹ”

## рҹҡҖ бһ”бһүбҹ’бһ бһ¶бҹ– бһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҷбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖ

бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҘбһЎбһјбһңбһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһңбһӮбҹ’бһӮбһҖбһ¶бһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһё бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ” бһҸбҹӮбһҹбһјбһҳбһ–бһ·бһ…бһ¶бһҡбһҺаёІаё§а№ҲаёІаё§аёҙаёҳаёөбһҹбһҳбҹ’бһҡбһҪбһӣбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһ’бҹ’бһңбһҫбһўбҹ„бһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһ¬бһ‘бҹҒбҹ”

бһҠбһјбһ…бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҠбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ·бһҷбһ¶бһӣбҹҗбһҷбһҠбһ¶бһҖбҹӢбһ’бһ¶бһҸбһ»бһ“бһ·бһҷбһҳбһ…бһҳбҹ’бһҡбһјбһ„бһ…бһҳбҹ’бһҡбһ¶бһҹбҹӢ бһ“бһ·бһ„бһҖбһјбһ“бһҹбһҸбҹ’бһңбһӣбҹҒбһ„бһ…бҹҶбһҡбһ»бҹҮ бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бҹ’бһҡбһҹбһҫбһҡбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶бһҸбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҺбһ¶бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһ“бһ·бһ„бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бҹҶбһ–бҹҒбһүбһҠбҹ„бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹ’бһҡбһҹбҹӢбһ–бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҡбһ”бҹҖбһ”бһҮбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶбҹ”

**бһңбһ·бһ—бһ¶бһӮбһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҷбҹ–**

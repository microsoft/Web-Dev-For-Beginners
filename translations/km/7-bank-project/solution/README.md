# កម្មវិធីធនាគារ

> ដំណោះស្រាយគំរូសម្រាប់គម្រោងកម្មវិធីធនាគារ ដែលបានកសាងជាមួយ HTML5 សុទ្ធ CSS និង JavaScript (មិនប្រើប្រាស់ស៊ាំវើកឬបណ្ណាល័យណាមួយទេ)។

## រត់កម្មវិធី

ជាមុនសិន សូមធ្វើឱ្យប្រាកដថាអ្នកមាន [ម៉ាស៊ីនបម្រើ API](../api/README.md) កំពុងរត់។

ម៉ាស៊ីនបម្រើវេបណែនាំណាមួយអាចប្រើសម្រាប់រត់កម្មវិធី ប៉ុន្តែពីព្រោះអ្នកគួរតែមាន [Node.js](https://nodejs.org) ដើម្បីរត់ API យ៉ាងហោចណាស់ អ្នកអាចធ្វើបាន៖

1. Git លីនខមាស៊ីននេះ។
2. បើកបន្ទប់បញ្ជា ពង្រីកទៅថតនេះ បន្ថែមទៀតរត់ `npx lite-server .`។ វានឹងចាប់ផ្តើមម៉ាស៊ីនបម្រើវេបអភិវឌ្ឍន៍នៅកំពង់ផែ `3000`
3. បើក `http://localhost:3000` នៅក្នុងកម្មវិធីរុករក ដើម្បីរត់កម្មវិធី។

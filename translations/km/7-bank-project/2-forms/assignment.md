# бһҖбһ¶бһҡбһҸбһ»бһ”бһҸбҹӮбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бһ“бһ¶бһӮбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҳбһҪбһҷ CSS бһ‘бҹҶбһ“бһҫбһ”

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһӮбһҳбҹ’бһҡбҹ„бһ„

бһ”бһҳбҹ’бһӣбҹӮбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бһ“бһ¶бһӮбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһұбҹ’бһҷбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹӮбһ”бһҳбһ¶бһ“бһҡбһјбһ”бһҡбһ¶бһ„бһ‘бһ¶бһҖбҹӢбһ‘бһ¶бһү бһ“бһ·бһ„бһҳбһ¶бһ“бһҡбһјбһ”бһҡбһ¶бһ„бһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ CSS бһ‘бҹҶбһ“бһҫбһ”бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҜбһҖбһҹбһҺбҹ’бһӢбһ¶бһ“бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҒбһҺбҹҲбһ–бҹҒбһӣбһҡбһҖбҹ’бһҹбһ¶бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһ”бһ‘бһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бҹ”

бһҖбһ¶бһҡбһҸбҹӮбһ„бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһҸбһҹбҹҠбһјбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹҶбһ“бһ¶бҹҶбһҡбһ…бһ“бһ¶бһңбҹӮбһ”бһ‘бҹҶбһ“бһҫбһ” бһўбһ“бһ»бһңбһҸбҹ’бһҸбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бҹҮбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ“бһ№бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһ‘бһ¶бһҖбҹӢбһ‘бһ¶бһүбҹ”

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҺбҹӮбһ“бһ¶бҹҶ

### бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 1бҹ– бһҡбҹҖбһ”бһ…бҹҶбһҹбҹ’бһ‘бһёбһӣбһҹбҹҠбһёбһҸ

**бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ CSS бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–**

1. **бһ”бһ„бҹ’бһҖбһҫбһҸ** бһҜбһҖбһҹбһ¶бһҡбһҗбҹ’бһҳбһёбһҲбҹ’бһҳбҹ„бҹҮ `styles.css` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
2. **бһ—бҹ’бһҮбһ¶бһ”бҹӢ** бһҹбҹ’бһ‘бһёбһӣбһҹбҹҠбһёбһҸбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `index.html` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–
   ```html
   <link rel="stylesheet" href="styles.css">
   ```
3. **бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ** бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһІбҹ’бһҷ CSS бһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҗбҹ’бһҳбһё бһ“бһ·бһ„бһҸбһҳбҹ’бһӣбҹғбһӣбҹҶбһ“бһ№бһ„бһ‘бҹҶбһ“бһҫбһ”бҹ–
   ```css
   /* Modern CSS reset and base styles */
   * {
     margin: 0;
     padding: 0;
     box-sizing: border-box;
   }
   
   body {
     font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
     line-height: 1.6;
     color: #333;
   }
   ```

### бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 2бҹ– бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ…бһ“бһ¶

**бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ’бһ¶бһҸбһ»бһҡбһ…бһ“бһ¶бһҠбҹҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ–**

#### бһ–бһҺбҹҢ
- **бһ–бһҺбҹҢбһҹбҹҶбһҒбһ¶бһ“бҹӢ**бҹ– бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ–бһҺбҹҢбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹҠбһјбһҸбһ»бһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһ·бҹҮбһӮбһ“бҹӢ
- **бһ–бһҺбҹҢбһӮбһјбһҹбһҳбһ бҹҒбһҸбһ»**бҹ– бһ–бһҺбҹҢбһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбһҖбҹ’бһҒбһҺбҹҲбһ•бҹ’бһҹбҹҒбһ„бҹ— бһ“бһ·бһ„бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҡбһ„
- **бһ–бһҺбҹҢбһўбһҗбҹҒбһҡ**бҹ– бһ–бһҺбҹҢбһҹбҹ’бһҳбһҫбһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҖбҹ’бһҹбһҡ бһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһ–бҹ’бһҡбҹҶ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ
- **бһ–бһҺбҹҢбһҮбҹ„бһӮбһҮбҹҗбһҷ/бһҖбҹҶбһ бһ»бһҹ**бҹ– бһ–бһҺбҹҢбһ”бҹғбһҸбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҮбҹ„бһӮбһҮбҹҗбһҷ бһ–бһҺбҹҢбһҖбҹ’бһҡбһ бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһ бһ»бһҹ

#### бһ•бҹ’бһ“бҹӮбһҖбһўбһҖбҹ’бһҹбһҡ
- **бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҖбҹ’бһ”бһ¶бһӣ**бҹ– бһ•бҹ’бһҹбһ¶бһүбҹӢбһ”бҹҶбһ”бҹӮбһҖбһ…бҹ’бһ”бһ¶бһҹбҹӢбһ…бһ“бҹ’бһӣбҹ„бҹҮ H1, H2, бһ“бһ·бһ„ H3
- **бһўбһҸбҹ’бһҗбһ”бһ‘бһҒбҹ’бһӣбһё**бҹ– бһ‘бҹҶбһ бҹҶбһўбһҖбҹ’бһҹбһҡбһўбһ¶бһ…бһўбһ¶бһ“бһ”бһ¶бһ“ (бһҷбҹүбһ¶бһ„бһҸбһ·бһ… 16px) бһ“бһ·бһ„бһҖбһҳбҹ’бһ–бһҹбҹӢбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһҳбһҡбһҳбҹ’бһҷ
- **бһҹбҹ’бһӣбһ¶бһҖбһ”бҹӮбһ”бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–**бҹ– бһҡбһ…бһ“бһ¶бһўбһҖбҹ’бһҹбһҡбһҸбһ¶бһҳбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ“бһ·бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ

#### бһ‘бһёбһҸбһ¶бҹҶбһ„ бһ“бһ·бһ„бһ…бһҳбҹ’бһ„бһ¶бһҷбһҡбһңбһ¶бһ„бһ’бһ¶бһҸбһ»
- **бһ…бһҳбҹ’бһ„бһ¶бһҷбһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶**бҹ– бһ”бҹ’бһҡбһҫбһңбһ·бһҳбһ¶бһҸбҹ’бһҡбһ…бһҳбҹ’бһ„бһ¶бһҷ (8px, 16px, 24px, 32px)
- **бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһӣбһ„бҹӢбһ”бҹ’бһҡбһўбһ”бҹӢ**бҹ– бһҡбҹҖбһ”бһ…бҹҶбһ”бҹ’бһӣбһ„бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһҳбҹ’бһҡбһ„бҹӢ бһ“бһ·бһ„бһ•бҹ’бһ“бҹӮбһҖбһҳбһ¶бһҸбһ·бһҖбһ¶
- **бһҡбһ…бһ“бһ¶бһ”бһ‘бһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҶбҹ’бһӣбһҫбһҷбһҸбһ”**бҹ– бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҳбһ»бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһӣбҹҗбһҸбһҮбһ¶бһҳбһҪбһҷбһ…бҹҶбһҺбһ»бһ…бһӣбҹҶбһ бһјбһҡ

### бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 3бҹ– бһҖбһ¶бһҡбһҸбһ»бһ”бһҸбҹӮбһ„бһ’бһ¶бһҸбһ»бһ•бҹ’бһҹбҹҶбһ–бһ·бһҹбҹҒбһҹ

**бһҸбһ»бһ”бһҸбҹӮбһ„бһ’бһ¶бһҸбһ»бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ–**

#### бһ‘бһҳбҹ’бһҡбһ„бҹӢ
- **бһ”бҹ’бһҡбһўбһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣ**бҹ– бһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһ–бҹ’бһҡбҹҶбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲ, бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ•бҹ’бһҸбҹ„бһҸ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһўбҹ’бһ“бһҖбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
- **бһ”бҹҠбһјбһҸбһ»бһ„**бҹ– бһ”бҹ’бһҡбһҸбһ·бһҖбһҳбҹ’бһҳбһ–бҹҒбһӣбһҖбһ¶бһҖбҹӢ, бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ–бһ»бҹҶбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ, бһ“бһ·бһ„бһҹбһүбҹ’бһүбһ¶бһҸбһҳбҹ’бһӣбһҫбһ„
- **бһҹбҹ’бһӣбһ¶бһҖ**бҹ– бһ‘бһёбһҸбһ¶бҹҶбһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһ“бһ·бһ„бһҹбһүбҹ’бһүбһ¶бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһңбһ¶бһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ
- **бһҹбһ¶бһҡвҖӢбһҖбҹҶбһ бһ»бһҹ**бҹ– бһҡбһ…бһ“бһ¶бһҖбҹҶбһ бһ»бһҹбһҠбҹӮбһӣбһҳбһҫбһӣбһғбһҫбһүбһ”бһ¶бһ“ бһ“бһ·бһ„бһҹбһ¶бһҡбһҮбһҪбһҷбһ”бһ„бҹ’бһ бһ¶бһү

#### бһҖбһ¶бһҡбһҡбһ»бһҖбһҡбһҖ
- **бһҖбҹ’бһ”бһ¶бһӣ**бҹ– бһҸбҹҶбһ”бһ“бҹӢбһҡбһ»бһҖбһҡбһҖбһҹбҹ’бһўбһ¶бһҸ бһ“бһ·бһ„бһҳбһ¶бһ“бһҳбҹүбһ¶бһҖбһ–бһ¶бһҺбһ·бһҮбҹ’бһҮбһҖбһҳбҹ’бһҳ
- **бһҸбҹҶбһҺбһ—бҹ’бһҮбһ¶бһ”бҹӢ**бҹ– бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһ¶бһҖбҹӢбһ…бҹ’бһ”бһ¶бһҹбҹӢ бһ“бһ·бһ„бһҹбһүбҹ’бһүбһ¶бһҹбһҖбһҳбҹ’бһҳ
- **бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„/бһ…бҹҶбһҺбһ„бһҮбһҫбһ„**бҹ– бһ’бһ¶бһҸбһ»бһҳбҹүбһ¶бһҖбһ–бһ¶бһҺбһ·бһҮбҹ’бһҮбһҖбһҳбҹ’бһҳбһҠбҹҸбһҡбһ№бһ„бһҳбһ¶бҹҶ

#### бһ•бҹ’бһ“бҹӮбһҖбһҳбһ¶бһҸбһ·бһҖбһ¶
- **бһ•бҹ’бһ“бҹӮбһҖ**бҹ– бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһҮбһ¶бһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ¶бһүбһҡбһңбһ¶бһ„бһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҒбһ„бҹ—
- **бһҖбһ¶бһҸ**бҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ”бҹ’бһҡбһҫбһ”бҹ’бһӣбһ„бҹӢбһҖбһ¶бһҸ бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҡбһ“бҹ’бһ’бһүбҹҗбһҡбһ“бһ·бһ„бһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһ–бҹ’бһҡбҹҶ
- **бһ•бҹ’бһ‘бҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ**бҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӮбһ“бҹ’бһӣбһ„бһ–бһҺбҹҢбһҹбһ“бһ·бһ„бһ•бҹ’бһ‘бҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷбһҹбҹ’бһҡбһ‘бһ“бҹӢ

### бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 4бҹ– бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһ”бһ“бҹ’бһҗбҹӮбһҳ (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ)

**бһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҠбҹҸбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ–**
- **бһҡбһ”бҹҖбһ”бһ„бһ„бһ№бһҸ**бҹ– бһ”бҹ’бһҠбһјбһҡбһҡбһңбһ¶бһ„бһҡбһ”бҹҖбһ”бһҹбҹ’бһҡбһ¶бһӣбһ“бһ·бһ„бһ„бһ„бһ№бһҸ
- **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһ…бһӣбһ“бһ¶**бҹ– бһ”бҹ’бһҡбһҸбһ·бһҖбһҳбҹ’бһҳбһҹбҹ’бһҡбһ‘бһ“бҹӢбһ“бһ·бһ„бһ…бһӣбһ“бһ¶бһҸбһјбһ…бҹ—
- **бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ•бҹ’бһ‘бһ»бһҖ**бҹ– бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһңбһёбһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„бһ–бҹҒбһӣбһ”бһүбҹ’бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢ
- **бһҡбһјбһ”бһ—бһ¶бһ–бһҶбҹ’бһӣбһҫбһҷбһҸбһ”**бҹ– бһҡбһјбһ”бһ—бһ¶бһ–бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бҹҶбһ бҹҶбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶

## бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһҡбһ…бһ“бһ¶

**бһӣбһҖбҹ’бһҒбһҺбҹҲбһӣбҹҒбһ…бһ’бҹ’бһӣбҹ„бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ’бһ“бһ¶бһӮбһ¶бһҡбһ‘бҹҶбһ“бһҫбһ”бҹ–**
- **бһҡбһ…бһ“бһ¶бһҹбҹ’бһўбһ¶бһҸ бһ“бһ·бһ„бһ‘бҹҶбһ“бҹҒбһҡбһ‘бһёбһҸбһ¶бҹҶбһ„бһҒбһ¶бһ„бһҹ** бһҮбһ¶бһҳбһҪбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ
- **бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ–бһҺбҹҢбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲ** (бһ–бһҺбҹҢбһҒбҹҖбһң бһ”бҹғбһҸбһ„ бһ¬бһ–бһҺбҹҢбһҹбҹ’бһҳбһҫбһҡбһҹбҹ’бһ‘бһ¶бһҸбҹӢ)
- **бһӣбҹҶбһҠбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹ’бһ”бһ¶бһҹбҹӢ** бһҮбһ¶бһҳбһҪбһҷбһ”бҹҠбһјбһҸбһ»бһ„бһ бҹ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹӮбһӣбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ
- **бһўбһҸбҹ’бһҡбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ** бһҠбҹӮбһӣбһӮбҹ„бһҡбһ–бһҸбһ¶бһҳбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ WCAG
- **бһ”бҹ’бһӣбһ„бҹӢбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ“бҹ…бһӣбһҫбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘** бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ

| **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ** | бһ”бҹ’бһҡбһҫбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ CSS бһ‘бҹҶбһ“бһҫбһ” бһҮбһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҖбһјбһҠбһ”бһ¶бһ“бһӣбҹ’бһў бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң | бһўбһ“бһ»бһңбһҸбҹ’бһҸ CSS бһ”бһ¶бһ“бһӣбҹ’бһўбһҮбһ¶бһҳбһҪбһҷбһҡбҹҖбһ”бһ…бҹҶбһ”бһ¶бһ“бһӣбҹ’бһў бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһҳбһҡбһҳбҹ’бһҷ | CSS бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҒбҹ’бһңбҹҮбһҡбҹҖбһ”бһ…бҹҶбһ¬бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ‘бҹҶбһ“бһҫбһ” | бһўбһ“бһ»бһңбһҸбҹ’бһҸ CSS бһҳбһ·бһ“бһ”бһ¶бһ“бһӣбҹ’бһў бһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ бһ¬бһҳбһ·бһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҸбһ¶бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ |
| **бһҡбһ…бһ“бһ¶бһ”бһ‘бһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҶбҹ’бһӣбһҫбһҷбһҸбһ”** | бһҡбһ…бһ“бһ¶бһ”бһ‘бһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ–бҹҒбһүбһӣбҹҒбһү бһ’бҹ’бһңбһҫбһҖбһ¶бһҡвҖӢбһ”бһ¶бһ“бһӣбҹ’бһўбһӣбһҫбһ‘бҹҶбһ бҹҶбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ | бһҳбһ¶бһ“бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһӣбҹ’бһўбһҮбһ¶бһҳбһҪбһҷбһ”бһүбҹ’бһ бһ¶бһҸбһ·бһ…бһҸбһҪбһ…бһӣбһҫбһҒбҹ’бһӣбҹҮбһ‘бҹҶбһ бҹҶбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ | бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһҮбһ¶бһҳбһҪбһҷбһ”бһүбҹ’бһ бһ¶бһ”бҹ’бһӣбһ„бҹӢбһҒбҹ’бһӣбҹҮ | бһҳбһ·бһ“бһҶбҹ’бһӣбһҫбһҷбһҸбһ” бһ¬бһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бһҡбһ»бһҖбһҡбһҖбһҖбһ¶бһ“бҹӢбһҸбҹӮбһҠбһ¶бһ…бҹӢбһҒбһ¶бһҸбһӣбһҫбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘ |
| **бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ** | бһӮбҹ„бһҡбһ–бһҸбһ¶бһҳбһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶ WCAG бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҡбһ»бһҖбһҡбһҖбһҖбҹ’бһҸбһ¶бһҡбһ…бһ»бһ…бһӣбҹ’бһў бһ“бһ·бһ„бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһўбһ¶бһ“бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ | бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбҹ’бһў бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ” бһ“бһ·бһ„бһҹбһүбҹ’бһүбһ¶бһ•бҹ’бһҸбҹ„бһҸбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң | бһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһҮбһ¶бһҳбһҪбһҷбһ’бһ¶бһҸбһ»бһҒбҹ’бһңбҹҮ | бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһ·бһ“бһӣбҹ’бһў бһ–бһ·бһ”бһ¶бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ–бһ·бһҖбһ¶бһҡбһ—бһ¶бһ– |

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҺбҹӮбһ“бһ¶бҹҶбһҠбһ¶бһҖбҹӢбһҹбҹ’бһ“бһҫ

**бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҹбҹ’бһ“бһҫбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–**
- **styles.css**бҹ– бһҹбҹ’бһ‘бһёбһӣбһҹбҹҠбһёбһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһӣбҹӢбһ–бҹҒбһүбһӣбҹҒбһү
- **HTML бһ”бһ¶бһ“бһҖбҹӮбһ”бҹ’бһҡбҹӮ**бҹ– бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһўбҹ’бһңбһёбҹ—бһҖбҹ’бһ“бһ»бһ„ HTML бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
- **бһҡбһјбһ”бһҗбһҸбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ**бҹ– бһҡбһјбһ”бһ—бһ¶бһ–бһ”бһ„бҹ’бһ бһ¶бһүбһҡбһ…бһ“бһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһ“бһ·бһ„бһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘
- **README**бҹ– бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһҒбҹ’бһӣбһёбһўбҹҶбһ–бһёбһҮбһҳбҹ’бһҡбһҫбһҹбһҡбһ…бһ“бһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ–бһҺбҹҢ

**бһ–бһ·бһ“бҹ’бһ‘бһ»бһ”бһјбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбҹ–**
- **бһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲ CSS** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбҹӮбһ”бҹ’бһҡбҹӮ
- **бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹ CSS бһҒбҹ’бһ–бһҹбҹӢ** бһҠбһјбһ…бһҮбһ¶ Grid, Flexbox, бһ¬бһ…бһӣбһ“бһ¶бһ•бҹ’бһ‘бҹғ CSS
- **бһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһ–бһ·бһҹбҹҒбһҹ** бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ CSS бһ”бҹӮбһ”бһ”бҹ’бһҡбһҹбһҫбһҡбһ“бһ№бһ„бһ‘бҹҶбһ бҹҶбһҜбһҖбһҹбһ¶бһҡбһҸбһ·бһ…бһ”бҹҶбһ•бһ»бһҸ
- **бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҡбһңбһ¶бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ** бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ…бҹҶбһ–бҹ„бҹҮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶

# បានគឹង API

> បានគឹង API ត្រូវបានកសាងជាមួយ [Node.js](https://nodejs.org) + [Express](https://expressjs.com/) ។

API ត្រូវបានកសាងរួចសម្រាប់អ្នកហើយមិនមែនជាផ្នែកនៃវគ្គសិក្សានេះទេ។

បើតែអ្នកចង់រៀនពីរបៀបកសាង API យ៉ាងនេះ អ្នកអាចតាមដានស៊េរីវីដេអូនេះបាន៖ https://aka.ms/NodeBeginner (វីដេអូ ១៧ ដល់ ២១ ពាក់ព័ន្ធនឹង API នេះដោយផ្ទាល់)។

អ្នកក៏អាចមើលសៀវភៅបង្រៀនអន្តរកម្មនេះ៖ https://aka.ms/learn/express-api

## ការរត់ម៉ាស៊ីនបម្រើ

ប្រាកដថាអ្នកបានដំឡើង [Node.js](https://nodejs.org) ។

១. Git clone ហាងហ្គោនេះ [The Web-Dev-For-Beginners](https://github.com/microsoft/Web-Dev-For-Beginners)។
២. បើកម៉ាស៊ីនបញ្ជារ និងចូលទៅក្នុងថត `Web-Dev-For-Beginners/7-bank-project/api`
២. រត់ពាក្យបញ្ជា `npm install` ហើយរង់ចាំការដំឡើងកញ្ចប់(អាចចំណាយពេលខ្លះនៅលើគុណភាពរបស់ការតភ្ជាប់អ៊ីនធឺណិត)។
៣. ពេលដំឡើងរួចរាល់ រត់ពាក្យបញ្ជា `npm start` ហើយអ្នកអាចប្រើបាន។

ម៉ាស៊ីនបម្រើគួរតែនាំស្តាប់នៅលើច្រក `5000`។
*ម៉ាស៊ីនបម្រើនេះនឹងរត់រួមជាមួយម៉ាស៊ីនបម្រើកម្មវិធីធនាគារមេ (ស្តាប់នៅលើច្រក `3000`) មិនចាប់ផ្តើមបិទវាទេ។*

> សម្គាល់៖ មាតិកាទាំងអស់ត្រូវបានផ្ទុកនៅក្នុងចងចាំមិនត្រូវបានរក្សាទុកយ៉ាងម៉ត់ចត់ ដូច្នេះពេលម៉ាស៊ីនបម្រើត្រូវបានបញ្ឈប់ ទិន្នន័យទាំងអស់នឹងបាត់បង់។

## ព័ត៌មានលំអិតអំពី API

Route                                       | ការពិពណ៌នា
--------------------------------------------|------------------------------------
GET    /api/                                | យកព័ត៌មានម៉ាស៊ីនបម្រើ
POST   /api/accounts/                       | បង្កើតគណនី ឧទាហរណ៍៖ `{ user: 'Yohan', description: 'My budget', currency: 'EUR', balance: 100 }`
GET    /api/accounts/:user                  | យកទិន្នន័យទាំងអស់សម្រាប់គណនីដែលបានគោលបំណង
DELETE /api/accounts/:user                  | លុបគណនីដែលបានគោលបំណង
POST   /api/accounts/:user/transactions     | បន្ថែមប្រតិបត្តិការ ឧទាហរណ៍៖ `{ date: '2020-07-23T18:25:43.511Z', object: 'Bought a book', amount: -20 }`
DELETE  /api/accounts/:user/transactions/:id | លុបប្រតិបត្តិការដែលបានគោលបំណង

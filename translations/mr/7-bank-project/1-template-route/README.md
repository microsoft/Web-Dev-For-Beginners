### рдХрд╛рд░реНрдп

рдЖрдордЪреНрдпрд╛ рдЕтАНреЕрдкрд╕рд╛рдареА рдкреБрдвреАрд▓ рдкрд╛рдКрд▓ рдореНрд╣рдгрдЬреЗ рдкреГрд╖реНрдард╛рдВрдордзреНрдпреЗ рдиреЗрд╡реНрд╣рд┐рдЧреЗрдЯ рдХрд░рдгреНрдпрд╛рдЪреА рд╢рдХреНрдпрддрд╛ рдЬреЛрдбрдгреЗ, URL рдореЕрдиреНрдпреБрдЕрд▓реА рдмрджрд▓рдгреНрдпрд╛рдЪреА рдЖрд╡рд╢реНрдпрдХрддрд╛ рди рдареЗрд╡рддрд╛. рдпрд╛рдЪрд╛ рдЕрд░реНрде рджреЛрди рдЧреЛрд╖реНрдЯреА:

### рдХрд╛рд░реНрдп

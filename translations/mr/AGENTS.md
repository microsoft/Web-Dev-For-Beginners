# AGENTS.md

## प्रकल्पाचा आढावा

हे सुरुवातीच्या लोकांसाठी वेब विकास मुलभूत तत्त्वे शिकवण्यासाठी शैक्षणिक अभ्यासक्रम संकलन आहे. हा अभ्यासक्रम Microsoft Cloud Advocates यांनी विकसित केलेला 12 आठवड्यांचा सखोल कोर्स आहे, ज्यात JavaScript, CSS आणि HTML यावर 24 व्यावहारिक धडे आहेत.

### मुख्य घटक

- **शैक्षणिक सामग्री**: प्रकल्पाधारित विभागांमध्ये आयोजित 24 संरचित धडे
- **व्यावहारिक प्रकल्प**: Terrarium, Typing Game, Browser Extension, Space Game, Banking App, Code Editor, आणि AI Chat Assistant
- **परस्परसंवादी प्रश्नमंजुषा**: 48 प्रश्नमंजुषा, प्रत्येकात 3 प्रश्न (धड्यांपूर्वी/नंतर मूल्यांकनासाठी)
- **बहुभाषिक समर्थन**: GitHub Actions च्या माध्यमातून 50+ भाषांमध्ये स्वयंचलित अनुवाद
- **तंत्रज्ञान**: HTML, CSS, JavaScript, Vue.js 3, Vite, Node.js, Express, Python (AI प्रकल्पांसाठी)

### आर्किटेक्चर

- शैक्षणिक संचयिका ज्यात धड्यांवर आधारित रचना आहे
- प्रत्येक धडा फोल्डरमध्ये README, कोड उदाहरणे आणि उपाय सोल्यूशन्स असतात
- स्वतंत्र प्रकल्प वेगळ्या निर्देशिकांमध्ये (quiz-app, विविध धड्यांचे प्रकल्प)
- अनुवाद प्रणाली GitHub Actions वापरून (co-op-translator)
- दस्तऐवजीकरण Docsify द्वारे वितरित आणि PDF स्वरूपात उपलब्ध

## सेटअप आदेश

हा संच मुख्यतः शैक्षणिक सामग्री उपभोगासाठी आहे. विशिष्ट प्रकल्पांवर काम करण्यासाठी:

### मुख्य संच सेटअप

```bash
git clone https://github.com/microsoft/Web-Dev-For-Beginners.git
cd Web-Dev-For-Beginners
```

### Quiz App सेटअप (Vue 3 + Vite)

```bash
cd quiz-app
npm install
npm run dev        # विकास सर्व्हर चालू करा
npm run build      # उत्पादनासाठी बिल्ड करा
npm run lint       # ESLint चालवा
```

### Bank प्रकल्प API (Node.js + Express)

```bash
cd 7-bank-project/api
npm install
npm start          # API सर्व्हर सुरू करा
npm run lint       # ESLint चालवा
npm run format     # Prettier सह स्वरूपित करा
```

### Browser Extension प्रकल्प

```bash
cd 5-browser-extension/solution
npm install
# ब्राउझर-विशिष्ट विस्तार लोड करण्याच्या सूचना पाळा
```

### Space Game प्रकल्प

```bash
cd 6-space-game/solution
npm install
# ब्राउझरमध्ये index.html उघडा किंवा Live Server वापरा
```

### Chat प्रकल्प (Python Backend)

```bash
cd 9-chat-project/solution/backend/python
pip install openai
# GITHUB_TOKEN पर्यावरण चल सेट करा
python api.py
```

## विकास कार्यप्रवाह

### सामग्री योगदान देणाऱ्यांसाठी

1. **तुमच्या GitHub खात्यावर रेपॉजिटरी Fork करा**
2. **तुमचा fork स्थानिक क्लोन करा**
3. **तुमच्या बदलांसाठी नवीन शाखा तयार करा**
4. धडा सामग्री किंवा कोड उदाहरणांमध्ये बदल करा
5. संबंधित प्रकल्प निर्देशिकांमध्ये कोड बदलांचे चाचणी करा
6. योगदान मार्गदर्शक सूचनांनुसार pull requests सादर करा

### शिकणाऱ्यांसाठी

1. रेपॉजिटरी fork किंवा clone करा
2. धडा निर्देशिकांमध्ये अनुक्रमे पुढे जा
3. प्रत्येक धड्याच्या README फाइल वाचा
4. https://ff-quizzes.netlify.app/web/ येथे धड्यापूर्वीचे प्रश्नमंजुषा पूर्ण करा
5. धडा फोल्डरमधील कोड उदाहरणांवर काम करा
6. असाइनमेंट्स आणि आव्हाने पूर्ण करा
7. धड्यानंतर प्रश्नमंजुषा द्या

### थेट विकास

- **दस्तऐवजीकरण**: मूळ फोल्डरमध्ये `docsify serve` चालवा (पोर्ट 3000)
- **Quiz App**: quiz-app निर्देशिकेत `npm run dev` चालवा
- **प्रकल्प**: HTML प्रकल्पांसाठी VS Code Live Server विस्तार वापरा
- **API प्रकल्प**: संबंधित API निर्देशिकांमध्ये `npm start` चालवा

**Quiz app सुरू होत नाही**:
- Node.js आवृत्ती तपासा (v14+ शिफारसीय)

**API सर्व्हर सुरू होत नाही**:
- Node.js आवृत्ती पूर्णपणे तपासा (node >=10)
- पोर्ट आधीच वापरात आहे का तपासा

**ब्राउझर एक्सटेंशन लोड होत नाही**:
- manifest.json योग्य प्रकारे स्वरूपित आहे का तपासा
- ब्राउझर कन्सोलमध्ये त्रुटी तपासा

- GITHUB_TOKEN पर्यावरण व्हेरिएबल सेट आहे का तपासा
- GitHub मॉडेल्स प्रवेश परवानग्या तपासा

**Docsify दस्तऐवज सेवा देत नाही**:
- docsify-cli ग्लोबली स्थापित करा: `npm install -g docsify-cli`

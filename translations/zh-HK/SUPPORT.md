# 支援

## 如何提交問題及獲取協助  

此項目使用 GitHub Issues 來追蹤漏洞及功能需求。在提交新問題之前，請先搜尋現有問題以避免重複。若有新問題，請將您的漏洞或功能需求提交為一個新問題。

如需使用此項目的協助及問題解答，請參閱[我們的貢獻指南](CONTRIBUTING.md)。

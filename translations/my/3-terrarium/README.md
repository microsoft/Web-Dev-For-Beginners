## သင့် Terrarium ကို တပ်ဆင်ရန်

သင့် Terrarium ကို **Azure Static Web Apps** ကို အသုံးပြု၍ ဝက်ဘ်ပေါ်တွင် တပ်ဆင် (deploy) သို့မဟုတ် ထုတ်ပြန် (publish) နိုင်သည်။

1. ဤ repo ကို fork ပြုလုပ်ပါ

2. ဒီ ခလုတ်ကိုနှိပ်ပါ 👇  

[![Deploy to Azure](https://aka.ms/deploytoazurebutton)](https://portal.azure.com/#create/Microsoft.StaticApp)

3. သင့် app ကို ဖန်တီးရန် setup wizard ကို လိုက်နာ ဆောင်ရွက်ပါ။  
   - **App root** ကို `/solution` သို့မဟုတ် သင့် codebase ၏ root သို့ သတ်မှတ်ပါ။  
   - ဤ app တွင် API မရှိသောကြောင့် API configuration ကို ကျော်လွှားနိုင်သည်။  
   - `.github` ဖိုဒါသည် Azure Static Web Apps က သင့် app ကို တည်ဆောက်ပြီး ထုတ်ပြန်ရန် အလိုအလျောက် ဖန်တီးပေးမည်ဖြစ်သည်။

## ਸੈਟਅਪ ਕਮਾਂਡਜ਼

ਇਹ ਰੀਪੋਜ਼ਟਰੀ ਮੁੱਖ ਤੌਰ ਤੇ ਸ਼ਿਖਿਆਮੂਲਕ ਸਮੱਗਰੀ ਖਪਤ ਲਈ ਹੈ। ਖ਼ਾਸ ਪ੍ਰੋਜੈਕਟਾਂ ’ਤੇ ਕੰਮ ਕਰਨ ਲਈ:

### ਮੁੱਖ ਰੀਪੋਜ਼ਟਰੀ ਸੈਟਅਪ

```bash
git clone https://github.com/microsoft/Web-Dev-For-Beginners.git
cd Web-Dev-For-Beginners
```

### ਕੁਇਜ਼ ਐਪ ਸੈਟਅਪ (Vue 3 + Vite)

```bash
cd quiz-app
npm install
npm run dev        # ਵਿਕਾਸ ਸਰਵਰ ਸ਼ੁਰੂ ਕਰੋ
npm run build      # ਉਤਪਾਦਨ ਲਈ ਬਣਾਓ
npm run lint       # ESLint ਚਲਾਓ
```

### ਬੈਂਕ ਪ੍ਰੋਜੈਕਟ API (Node.js + Express)

```bash
cd 7-bank-project/api
npm install
npm start          # ਏਪੀਆਈ ਸਰਵਰ ਸ਼ੁਰੂ ਕਰੋ
npm run lint       # ESLint ਚਲਾਓ
npm run format     # Prettier ਨਾਲ ਫਾਰਮੈਟ ਕਰੋ
```

### ਬ੍ਰਾਊਜ਼ਰ ਐਕਸਟੈਂਸ਼ਨ ਪ੍ਰੋਜੈਕਟ

```bash
cd 5-browser-extension/solution
npm install
# ਬਰਾਊਜ਼ਰ-ਖਾਸ ਐਕਸਟੈਂਸ਼ਨ ਲੋਡ ਕਰਨ ਦੇ ਨਿਰਦੇਸ਼ਾਂ ਦਾ ਅਨੁਸਰਣ ਕਰੋ
```

### ਸਪੇਸ ਗੇਮ ਪ੍ਰੋਜੈਕਟ

```bash
cd 6-space-game/solution
npm install
# index.html ਨੂੰ ਬ੍ਰਾਉਜ਼ਰ ਵਿੱਚ ਖੋਲ੍ਹੋ ਜਾਂ Live Server ਦਾ ਪ੍ਰਯੋਗ ਕਰੋ
```

### ਚੈਟ ਪ੍ਰੋਜੈਕਟ (Python ਬੈਕਐਂਡ)

```bash
cd 9-chat-project/solution/backend/python
pip install openai
# GITHUB_TOKEN ਵਾਤਾਵਰਣ ਚਲਾਕ 변수 ਸੈੱਟ ਕਰੋ
python api.py
```

## ਬਿਲਡ ਅਤੇ ਡਿਪਲੌਇਮੈਂਟ

### ਕੁਇਜ਼ ਐਪ ਡਿਪਲੌਇਮੈਂਟ (Azure Static Web Apps)

ਕੁਇਜ਼-ਐਪ Azure Static Web Apps ਡਿਪਲੌਇਮੈਂਟ ਲਈ ਸੰਰਚਿਤ ਹੈ:

```bash
cd quiz-app
npm run build      # dist/ ਫੋਲਡਰ ਬਣਾਉਂਦਾ ਹੈ
# ਮੁੱਖ ਵਿੱਚ push ਕਰਨ 'ਤੇ GitHub Actions ਵਰਕਫ਼ਲੋ ਰਾਹੀਂ ਤੈਨਾਤ ਕਰਦਾ ਹੈ
```

Azure Static Web Apps ਸੰਰਚਨਾ:  
- **ਐਪ ਦਾ ਸਥਾਨ**: `/quiz-app`  
- **ਆਉਟਪੁੱਟ ਸਥਾਨ**: `dist`  
- **ਵਰਕਫਲੋ**: `.github/workflows/azure-static-web-apps-ashy-river-0debb7803.yml`  

### ਡੌਕਯੂਮੇਨਟੇਸ਼ਨ PDF ਜਨਰੇਸ਼ਨ

```bash

```

### Docsify ਡੌਕਯੂਮੇਨਟੇਸ਼ਨ

## ਡਿਬੱਗਿੰਗ ਅਤੇ ਸਮੱਸਿਆ-ਹੱਲ

### ਆਮ ਸਮੱਸਿਆਵਾਂ

**ਕੁਇਜ਼ ਐਪ ਸ਼ੁਰੂ ਨਹੀਂ ਹੁੰਦਾ**:  
- Node.js ਵਰਜਨ ਦੀ ਜਾਂਚ ਕਰੋ (v14+ ਸਿਫਾਰਸ਼ੀ)  
- `node_modules` ਅਤੇ `package-lock.json` ਮਿਟਾ ਕੇ `npm install` ਦੁਬਾਰਾ ਚਲਾਓ  
- ਪੋਰਟ ਟਕਰਾਅ ਦੀ ਜਾਂਚ ਕਰੋ (ਡਿਫੌਲਟ: Vite ਜਨਾਨਾ 5173 ਪਰ ਵਰਤਦਾ ਹੈ)  

**API ਸਰਵਰ ਸ਼ੁਰੂ ਨਹੀਂ ਹੁੰਦਾ**:  
- ਯਕੀਨ ਕਰੋ ਕਿ Node.js ਵਰਜਨ ਘੱਟੋ-ਘੱਟ ਹੈ (node >=10)  
- ਜਾਂਚੋ ਕਿ ਪੋਰਟ ਪਹਿਲਾਂ ਵਰਤੋਂ ਵਿੱਚ ਨਾ ਹੋਵੇ  
- ਸਾਰੇ ਡਿਪੈਂਡੈਂਸੀਜ਼ `npm install` ਨਾਲ ਇੰਸਟਾਲ ਕੀਤੇ ਗਏ ਹੋਣ  

**ਬ੍ਰਾਊਜ਼ਰ ਐਕਸਟੈਂਸ਼ਨ ਲੋਡ ਨਹੀਂ ਹੁੰਦੀ**:  
- manifest.json ਨੂੰ ਠੀਕ ਫਾਰਮੈਟ ਵਿੱਚ ਹੈ ਕਿਵੇਂ ਦਿਖਾਓ  
- ਬ੍ਰਾਊਜ਼ਰ ਕੰਸੋਲ ਵਿੱਚ ਗਲਤੀਆਂ ਦੇਖੋ  
- ਬ੍ਰਾਊਜ਼ਰ-ਵਿਸ਼ੇਸ਼ ਐਕਸਟੈਂਸ਼ਨ ਇੰਸਟਾਲੇਸ਼ਨ ਨਿਰਦੇਸ਼ਾਂ ਦੀ ਪਾਲਣਾ ਕਰੋ  

**Python ਚੈਟ ਪ੍ਰੋਜੈਕਟ ਸਮੱਸਿਆਵਾਂ**:  
- OpenAI ਪੈਕੇਜ ਇੰਸਟਾਲ ਕਰੋ: `pip install openai`  
- GITHUB_TOKEN ਵਾਤਾਵਰਣ ਚਰ (env) ਸੈਟ ਹੈ ਜਾਂ ਨਹੀਂ ਸਹੀ ਚੈੱਕ ਕਰੋ  
- GitHub Models ਦੀ ਐਕਸੈਸ ਅਧਿਕਾਰ ਦੀ ਜਾਂਚ ਕਰੋ  

**Docsify ਡੌਕ ਯੂਮੇਂਟ ਸਰਵ ਨਹੀਂ ਕਰਦਾ**:  
- docsify-cli ਗਲੋਬਲੀ ਇੰਸਟਾਲ ਕਰੋ: `npm install -g docsify-cli`  
- ਰੀਪੋਜ਼ਟਰੀ ਰੂਟ ਡਿਰੈਕਟਰੀ ਤੋਂ ਚਲਾਓ  
- ਯਕੀਨੀ ਬਣਾਓ ਕਿ `docs/_sidebar.md` ਮੌਜੂਦ ਹੈ  

### ਡਿਵੈਲਪਮੈਂਟ ਮਾਹੌਲ ਲਈ ਸੁਝਾਵ

- HTML ਪ੍ਰੋਜੈਕਟ ਲਈ VS ਕੋਡ ਨਾਲ Live Server ਐਕਸਟੈਂਸ਼ਨ ਵਰਤੋਂ  
- ESLint ਅਤੇ Prettier ਐਕਸਟੈਂਸ਼ਨ ਇੰਸਟਾਲ ਕਰੋ ਸਥਿਰ ਫਾਰਮੇਟਿੰਗ ਲਈ  
- JavaScript ਨੂੰ ਡਿਬੱਗ ਕਰਨ ਲਈ ਬ੍ਰਾਊਜ਼ਰ ਡਿਵਟੂਲਜ਼ ਵਰਤੋਂ  
- Vue ਪ੍ਰੋਜੈਕਟ ਲਈ Vue DevTools ਬ੍ਰਾਊਜ਼ਰ ਐਕਸਟੈਂਸ਼ਨ ਇੰਸਟਾਲ ਕਰੋ
